　　黑红丝线交织，好似一个碗倒扣在地面，把方圆数十公里完全笼罩。

<br/>　　黑红牢笼释放的气息，让他如临大敌，即便隔得那么远，压力依旧像是在面对蚁王三护卫，不寒而栗！。
<br/>　　“是那个人！”诺布瞳孔骤然收缩，眼中全是难以置信。
<br/>　　“你知道是谁？”莫老五皱着眉头沉声问道。
<br/>　　“就是刚才闯入包围圈的青年，我见过他动手。虽然只看到残影，但确实是那种黑红色念力丝线。只是，用念线编织这么大一牢笼？！这……这也太恐怖了！”诺布额头见汗，之前的那种致命感再次袭上心头！
<br/>　　“青年？”尼特罗若有所思：“一头黑发，皮肤有些晶莹黑色，眼睛狭长透着邪异的青年？”
<br/>　　“会长您认识？”
<br/>　　“啊，见过。那是一位老朋友的孙子。”尼特罗已经认出了苏重。
<br/>　　他和苏重交过手，见过苏重的念线，熟悉苏重念力气息。刚才就感觉熟悉，诺布一说，他就知道是谁。
<br/>　　感受着空中，黑红牢笼散发的狂暴气息，尼特罗一脸感叹“只是不到一年，竟有如此进步。”这让尼特罗兴奋不已，又一个强力对手。
<br/>　　他可是非常期盼和强者交手啊！
<br/>　　正是因为这旺盛的挑战欲，他才会仅仅带着两人，就进入ngl深处。偌大猎人协会，难道没有其他猎人？
<br/>　　最出名的猎人十二支，每一个可都是强者。为什么尼特罗没带来？因为他没法带。帕里斯通和他身后的五大帝国，或明或暗的压制，让尼特罗根本无法倾尽协会全部力量，来剿灭嵌合蚁。
<br/>　　而他的挑战欲望，让他等不及和这些人周旋。所以他只带着两人，就直奔嵌合蚁老巢而来。
<br/>　　帕里斯通等人，正是利用了尼特罗和强者战斗的欲望，让他心甘情愿的身赴险境！
<br/>　　“没想到，除了蚁王，竟还能遇到糜稽揍敌客这等对手！这一趟，不虚此行！”尼特罗裂开嘴，露出一口雪亮白牙，苍老的面孔容光焕发。
<br/>　　“诺布，这个牢笼对空间能力应该没有克制。你出去，用最快的速度把医疗人员和生物学研究人员带来。蚁后和人类结构不同，需要生物学家提供治疗建议。”尼特罗压下情绪波动，冷静安排，接着转头看向寇鲁多道：“我们接受嵌合蚁的投降，但需要知道嵌合蚁王的信息。”
<br/>　　……
<br/>　　“牢笼合并，嵌合蚁，你们可以去死了。”
<br/>　　苏重勾动食指，静静耸立的线之牢笼嗡的一颤，下一刻猛然开始向中间收缩。
<br/>　　这是苏重第二次发动线之牢笼，范围比第一次大，合拢速度更加快速。念线的切割能力越发强劲，沿途所过之处，所有障碍尽数被斩开。

<br/>　　刚才还兴高采烈，为脱离蚁后统治而兴奋，现在却以比刚才更快的速度，向着巢穴死命奔逃！
<br/>　　天空中，蚁王和三大护卫缓缓飞行，远远就看到了笼罩天地的线之牢笼。
<br/>　　四蚁来到牢笼边缘，降落下来。
<br/>　　砰！
<br/>　　尼飞比多随意一脚，把一块三米高的巨石踢向牢笼。

<br/>　　“我来试试！”尼飞比多身躯半蹲，背部微微弓起，好似一只猎豹。纤细的大腿骤然变粗，青黑血管暴起，狰狞异常。
<br/>　　砰！空气一声爆鸣，尼飞比多瞬间就撞上了线之牢笼。
<br/>　　铛！一声金铁相互碰撞之声响起，火花四溅。嗡嗡嗡！整个牢笼顿时发出一阵令人牙酸的嗡鸣，牢笼竟真的暂停了收缩！
<br/>　　可仅仅维持了三秒，嗤！一阵白烟从地面冒起，牢笼再次猛然收缩起来。

<br/>　　但他们三者不同，他们是真正的战斗嵌合蚁！一个就足以横扫所嵌合兵蚁！
<br/>　　尼飞比多全力攻击，竟只让这个怪异牢笼暂停三秒？！枭亚普夫有种不祥，如果他们被困在这里，很可能会遇到致命危机！
<br/>　　轰轰！
