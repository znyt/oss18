        “爷爷……”
<br/>
<br/>        “我知道你是为了何事而来。”傅炎廷随之打断了童筱颖的话，手中拿着毛笔，继续若无其事地练字。
<br/>
<br/>        童筱颖轻轻地将手里的粥碗，放在了书桌上，然后主动站到了傅炎廷的身旁，替傅炎廷磨墨。
<br/>
<br/>        傅炎廷并未看童筱颖一眼，一边握着毛笔写着自己的字，一边话中有话地说道：“小景的事情，是他咎由自取。而你现在要做的，应该是要独善其身。”
<br/>
<br/>        “昼景是我的丈夫。”童筱颖斩钉截铁道。
<br/>
<br/>        傅炎廷却嗤笑：“所以，你打算助纣为虐？”
<br/>
<br/>        “爷爷，如果是大哥犯了错。爷爷一定会拼尽全力保全大哥。那么为什么昼景就不行？昼景也是爷爷的亲孙子，为什么爷爷不肯保全昼景，重新给他一次机会？昼景不能去坐牢！他如果坐牢，这一辈子，就真的毁了！”童筱颖声音哽咽道。
<br/>
<br/>        傅炎廷不以为然地反驳：“真是妇人之仁！”
<br/>
<br/>        首先，他可以肯定，他的长孙傅夜沉才不会去做这种事情，其次，就算傅夜沉犯了错，他身为爷爷，也会让他的长孙，自己去承担所犯下的错的一切后果。
<br/>
<br/>        傅家确实家大业大势大，但并不代表，傅家的子孙，就可以利用这些便利为所欲为。
<br/>
<br/>        坐得端、走的直、行得正，堂堂正正去做人。
<br/>
<br/>        这是傅炎廷曾给自己儿子冷卫峰的家训。
<br/>
<br/>        只可惜，卫峰殉国，让他这白发人送黑发人。
<br/>
<br/>        但傅炎廷从未为此伤感过，他为自己的儿子悲而光荣。
<br/>
<br/>        “爷爷，大哥和昼景都是您的亲孙子，您真的从未偏心过大哥吗？”童筱颖愤愤不平地质问。
<br/>
<br/>        如果爷爷对大哥和昼景一视同仁，或许，昼景就不会利用自己的职务便利，选择走一条能证明自己实力、能让自己成功的捷径。
<br/>
<br/>        “我偏心又如何？傅家家业，从古至今，只传长孙。更何况，小景还是个私生子，再说得不好听点，如果不是小景的母亲因为单恋，而自私地用那‘下三滥’的手段，又岂会跟我儿子有了小景？又岂会让我儿子和儿媳妇感情心生间隙？我身为爷爷，为何要委屈自己的嫡长孙和儿媳妇？再说了，除了傅氏集团和傅家大庄园，小沉有的，小景都有。小景还有什么不满足？我身为爷爷，何时亏待过他？他大学毕业，我给他送车送房送工作，只希望他过安稳日子，丰衣足食。就连他不经过我首肯，就直接跟你领证结婚，我又何曾反对过？筱颖，你是否真正地了解过你的丈夫到底是一个什么样的男人？”傅炎廷不禁恼火地对童筱颖怒斥道。
<br/>
<br/>        童筱颖顿时沉默了。
<br/>
<br/>        傅炎廷见童筱颖不再吭声，再次下逐客令：“你走吧！这件事情，不要跟我再提！”
<br/>
<br/>        语毕，傅炎廷放下了手中的毛笔，离开了书桌前。
<br/>
<br/>        童筱颖见傅炎廷要离开，“啪”地一声，跪在了傅炎廷的面前，哀求道：“爷爷，我求你，求你保全昼景。”
<br/>
<br/>        “你爱跪，就跪着吧！”傅炎廷直接无视童筱颖的感受，甩手离开了书房。
<br/>
<br/>        傅夜沉随后赶过来的时候，爷爷傅炎廷已经离开了，只留下不知在地上跪了多久的童筱颖。
<br/>
<br/>        “筱颖，你不要这样。爷爷铁了心的事情，是不会改变。”傅夜沉伸手想要拉童筱颖起来，却被童筱颖甩手拒绝。
<br/>
<br/>        “我不管昼景是什么样的人，但是他是我的丈夫，他若是出了事，我不会不管他。大哥，你帮我求求爷爷，好不好？只要爷爷答应保全昼景，让我做任何事情，我都愿意。大哥，你帮我去求求爷爷，好不好？爷爷他最宠你，他一定会为了你而心软。大哥，我求求你！”童筱颖双眸里充斥着眼泪，声音诚恳地哀求道。
<br/>
<br/>        她见傅夜沉不为所动，忽然俯下身去，额头重重地砸在地上，向傅夜沉磕头，以此来表示自己的决心与诚意。
<br/>
<br/>        傅夜沉惊愕失色，连忙抓住童筱颖的手腕，将她的身子从地上拽了起来。
<br/>
<br/>        “筱颖，你够了！”傅夜沉心痛地呵斥道。
<br/>
<br/>        她压根就不了解爷爷的性子，哪怕她拿她自己的性命去威胁，爷爷也不会因此而改变他的主意。
<br/>
<br/>        童筱颖仰起头来，看着傅夜沉的那对犀利的黑眸，同时也看穿了这件事情的严重性。
<br/>
<br/>        她挪开了傅夜沉的手，独自一个人走出了书房。
<br/>
<br/>        但是，她还是倔强地跪在了爷爷傅炎廷的卧室门外。
<br/>
<br/>        傅夜沉明知道她这么做，也不会改变爷爷傅炎廷的心意，但是他还是选择了陪着童筱颖一起跪。
