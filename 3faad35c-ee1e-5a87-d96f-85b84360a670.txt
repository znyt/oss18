　　“小子尔敢！”
<br/>　　云千秋反应过来之后，顿时暴喝一声。
<br/>　　可惜，为时已晚！
<br/>　　“彭！”
<br/>　　没有任何悬念，云天青在叶尘这狂暴一拳之下，瞬间被打成了一团肉渣，甚至连惨叫都没有来得及发出。
<br/>　　堂堂药王宗宗主，蓬莱仙界的巨头级人物，竟然被叶尘一拳打爆了！
<br/>　　所有人看到这一幕，全都目瞪口呆。
<br/>　　从叶尘想要逃跑，到突然实力大爆发，一连斩杀药王宗四大长老，而后又一鼓作气，打爆了云天青。
<br/>　　说时慢，其实也不过一转眼的功夫。
<br/>　　之前还处于劣势，眼看就要不敌的少年，居然一举完成了大逆袭！
<br/>　　若非亲眼所见，他们实在难以想象，如此年轻的一个世俗界少年，竟然具有如此恐怖的实力。
<br/>　　“这家伙到底是什么来头？怎么会这么强！”
<br/>　　“他刚刚似乎动用了某种秘术，实力一下子变强了好多！”
<br/>　　“单单就刚才的表现来看，此人如今的实力，应该足以进入天榜前三了！”
<br/>　　“恐怕连云千秋都要逊色一筹！”
<br/>　　“这下药王宗算是踢到铁板了！”
<br/>　　……
<br/>　　众人议论纷纷，而下方的祝明飞兄弟，同样惊骇不已。
<br/>　　“哥，叶大哥好厉害！”
<br/>　　祝明飞脸上露出一抹苦笑道：
<br/>　　“是呀！亏我一向自认为是潜龙榜首，修炼天赋不输任何人，没想到今日才知道，自己不过是坐井观天！真是人外有人，天外有天啊！”
<br/>　　苏蔓则满脸得意之色，
<br/>　　“我就说过，我老弟不会输！”
<br/>　　祝明飞听到这话，下意识的想要反驳，不过想起之前被连续打脸，话到嘴边就又咽了下去。
<br/>　　不过心中，却依旧有些不太看好叶尘。
<br/>　　毕竟，
<br/>　　他虽然杀了云天青和四大长老，可是药王宗的最强者云千秋还在，而且其他四大宗门的高手，只怕同样不会善罢甘休。
<br/>　　难道他要以一人之力，挑战五大宗门不成？
<br/>　　便是化神天君在此，恐怕也没有这样的底气吧！
<br/>　　“叶狂仙！你竟然——竟然杀我的天青孩儿，我要让你死！”
<br/>　　云千秋亲眼看到云天青，被叶尘一拳打爆，整个人顿时愤怒到了极点，双眸中几乎要喷出火来！
<br/>　　轰！
<br/>　　云千秋声音落下的同时，背后陡然出现了一个硕大的蜘蛛状的赤色虚影，
<br/>　　“天、罗、地、网！”
<br/>　　云千秋再次一声暴喝，同时双掌朝叶尘猛然拍出！
<br/>　　轰轰轰！
<br/>　　他身后的那道庞大的蜘蛛虚影，竟然喷出一张张青色巨网，顿时便将整个药王谷的上空，全都笼罩在了其中，然后又朝中间的叶尘包围而去。
<br/>　　周围众人看到这一幕，顿时纷纷脸色大变，全都开始不自觉的倒退。
<br/>　　“这个就是云千秋的成名绝学‘天罗地网’吗？”
<br/>　　“据说云千秋为了提升功力，吞噬一头赤尾妖蛛的妖丹，能喷射毒网，并且常年以身喂毒，毒网剧毒无比，沾到即死，相当可怕！”
<br/>　　“唉！他们药王宗的功法大多都与毒有关，虽然这云千秋的实力，并不算最强，可是即便是天榜榜首无涯老人，恐怕也不敢轻易和他交手！”
<br/>　　“这小子要倒霉了！”
<br/>　　……
<br/>　　叶尘看到铺天盖地的巨网，朝他笼罩而来，同样眉头微微一皱。
<br/>　　他可以清晰的感受到，这些巨网上全都包含剧毒，即便是他一不小心沾上，恐怕也非死即伤。
<br/>　　“破！”
<br/>　　叶尘不及多想，当即抡起紫琼剑，开始阻挡那些毒网。
<br/>　　噗！
<br/>　　毒网被叶尘轻松破开。
<br/>　　然而，还没等他松了一口气，又有更多的毒网接踵而至！
<br/>　　而且速度越来越快！
<br/>　　嗖嗖嗖！
<br/>　　一眨眼的功夫，云千秋足足射出了数百张毒网！
<br/>　　虽然全部被叶尘用紫琼剑斩破，可是那些毒网的残骸，居然在周围堆积起来，最终形成了一个巨大的、厚厚的球体，将叶尘包裹在了其中。
<br/>　　下方的祝明飞，看到这一幕，顿时失声道：
<br/>　　“不好！这是一个圈套！叶兄弟一旦被毒网封住退路，云千秋便会动用‘痛苦之源’来对付他！”
<br/>　　一旁的祝小白，听到这话，同样脸色大变。
<br/>　　苏蔓则一脸不解道：
<br/>　　“什么是痛苦之源？”
<br/>　　没等祝明飞开口，一旁的祝小白道：
<br/>　　“痛苦之源，据说是药王宗最强大的毒，而且是一种针对灵魂的毒！”
<br/>　　“中了此毒之人，会将他此生经历的痛苦再度经历一遍，而且痛苦会被无限放大至千倍、万倍！”
<br/>　　“最终整个人的心智，都会被这种强大的痛苦所控制，陷入无尽的愤怒之中，最终活活癫狂至死！”
<br/>　　蹬蹬！
<br/>　　听完祝小白的解释，苏蔓顿时娇躯一颤，险些晕厥过去，继而望向一旁的祝明飞，
<br/>　　“那你还等什么？还不赶快帮帮他！”
<br/>　　祝明飞叹了一口气，道：
<br/>　　“只怕药王宗的其他高手，和四大宗门的人，是不会让我相帮的！”
<br/>　　说完这话，
<br/>　　祝明飞猛然一掌拍出，
<br/>　　轰！
<br/>　　一条璀璨的火龙，自他的掌中发出，直奔上方的云千秋攻去。
<br/>　　然而，
<br/>　　那条火龙才刚飞至一半，天空中陡然出现了一道中年男子的身影，竟然只用一掌，便将那条火龙打爆！
<br/>　　“小火神，这是药王宗和世俗界之间的恩怨，你还是不要插手的好，如果你手痒的话，本宗主倒是可以陪你玩玩！”
<br/>　　祝明飞顿时脸色微微一变，冲着苏蔓低声道
<br/>　　“看到了吧！此人便是天雷宗的宗主上官云阙，天雷宗和药王宗向来交好，我就知道他们肯定不会坐视不理的……”
<br/>　　苏蔓听到这话，顿时大为焦急，可是却也毫无办法。
<br/>　　倘若连他们也被人围攻，只怕不但帮不了叶尘，反而会让他担心。
<br/>　　另外一边，
<br/>　　云千秋忽然一声暴喝，
<br/>　　“封！”
<br/>　　轰隆！
<br/>　　天空中那漫天的毒网，顿时便凝聚在了一处，瞬间形成了一个大大的球团，将叶尘和他，全都包裹在了其中。
<br/>　　“桀桀！”
<br/>　　云千秋见一切顺利，顿时狂笑了一声，
<br/>　　“小子！能够逼老夫动用这一招，你也足以自傲了，接下来，老夫会让你明白，什么叫做痛不欲生！桀桀桀！”
<br/>　　一道毛骨悚然的笑声，在九天之上传来，下方众人顿时感觉心头一阵发毛，尤其是苏蔓三人，更是脸色惨白无比。
