        她这个动作已经做得很不经意间了，就像是无意的一样，但是她却心知肚明，她这是在犯贱，故意引诱一个有妇之夫。
<br/>
<br/>        落以柔以自己的第六感告诉自己，傅夜沉和童筱颖之间的感情一定出现了裂痕。
<br/>
<br/>        她现在趁虚而入，是最佳时机。
<br/>
<br/>        但傅夜沉却只是看着车头的前方，抬手抓住了她的手腕，一个力度，就痛得她不得不对他放手。
<br/>
<br/>        他冷冷地说道：“我不喜欢除了筱颖以外的女人碰我，你最好离我……”
<br/>
<br/>        “我不会去美国，死也不去！哪怕你派人把我绑去别的地方，我也会用我的双腿走回来！我落以柔这一辈子，哪里都不去，我就留在傅氏集团，守着你的‘江山’等你回来。”落以柔急忙打断傅夜沉的话。
<br/>
<br/>        “随你便，下车！”傅夜沉不悦地蹙起剑眉。
<br/>
<br/>        落以柔接着深情款款地说道：“大哥，我永远不会对你说出那三个字，因为你不允许，所以，我不会再说。可是，我已经感觉到，你已经知道我对你的心意是什么了。就到这里，我也已经心满意足。”
<br/>
<br/>        “不要对筱颖下手，否则我不会放过你。”傅夜沉眉头紧锁。
<br/>
<br/>        落以柔忍俊不禁地应了声：“大哥，你放心好了，我不会做让你恨我、讨厌我的事情。大哥，你一定要平安归来。”
<br/>
<br/>        她爽快地说完，打开了车门，抱着手提包便跳了下去。
<br/>
<br/>        落以柔刚把车门关上，傅夜沉便开着车扬长而去。
<br/>
<br/>        而她却站在路边，恋恋不舍地目送他的离开。
<br/>
<br/>        连华生带她去私宅见傅夜沉之前，其实，她在路上问过连华生，她和童筱颖的差距到底在哪里。
<br/>
<br/>        她原本会很期待连华生能给她一个满意的答案，但是到头来，他只跟她说：“就算她落以柔是这世上最适合他傅夜沉的女人，但她终究不是童筱颖。”
<br/>
<br/>        傅夜沉这一走，就是了无音讯。
<br/>
<br/>        不过，连华生倒是在第二天傍晚时，才接到上头下达的追捕傅夜沉的全球通缉令。
<br/>
<br/>        看样子，季雅珊是被傅夜沉成功救走了。
<br/>
<br/>        要去国际安防局的暗牢里救季雅珊出来，怎么可能不会暴露自己的身份。
<br/>
<br/>        局里戒备森严，又都是高科技的安防系统，傅夜沉要想把季雅珊从里面带出来，只能利用他自己的那重特殊身份。
<br/>
<br/>        对于傅夜沉救走季雅珊这事，局长夏阳枭，也就是他傅夜沉的干爹，还亲自提审了他连华生。
<br/>
<br/>        连华生自然是说自己毫不知情，而且还迫不得已地添油加醋道：“阿沉私生活本就混乱，前有女明星，后有女秘书的，虽然娶了个老婆在家当摆设，但也总是跟这个女人扯上关系后，又跟那个女人扯上关系。保不齐，他在冒险者集团的古堡里当四爷的时候，与那个季雅珊朝夕相处，日久生情也说不定啊！”
<br/>
<br/>        “那就去把他的‘正宫’带回来！我就不信，我们抓了他老婆，他还不给我乖乖就范，回来认错！”夏阳枭似乎是真的被傅夜沉的行为给气到了。
<br/>
<br/>        要知道，他是多么地重视傅夜沉，甚至还想着将来自己退休后，这位子接着由他傅夜沉来坐。
<br/>
<br/>        现在倒好了，他傅夜沉，倒是伤透了他这个当干爹的心。
<br/>
<br/>        “我说头儿，我帮您把他的‘正宫’带过来也没用啊！您想想，阿沉为了那个季雅珊，连咱们组织都可以背叛。可见，他心里，看中的是那个季雅珊。他若是为了家里那位，那他还叛变做什么？更何况，他家里那位，您都答应过他不抓她了，也不对她进行审讯。所以，他家里那位‘正宫’，无法作为王牌，让您去逼阿沉就范了。”连华生条条是道地说。
<br/>
<br/>        实则，他是在保证童筱颖的安全。
<br/>
<br/>        他答应过阿沉的事情，就一定要做到。
<br/>
<br/>        童筱颖若是被带到这安防局里来，日子铁定是不好过。
<br/>
<br/>        就算不被审讯员轮流审讯，严刑逼供，也会被压去心理实验室，进行一番心理上的暗黑折磨。
<br/>
<br/>        季雅珊就被他们折腾得半死不活了，童筱颖再进去，要是被傅夜沉知道，傅夜沉不炸这了安防局才怪！
<br/>
<br/>        夏阳枭前思后虑了之后，觉得连华生说的并不是没道理。

<br/>        等到连华生从安防局里开完各种会议，面见完各个领导后，赶回私宅的时，是傅夜沉离开的第七天了。
