　　就在楚风眠面前的不远处，七彩祥云突然凝聚而成，随即七彩霞光，缓缓凝聚。
<br/>　　在这七彩霞光的背后，一道古老宫殿的虚影，显现而出，这宫殿虚虚实实，内部另有乾坤。
<br/>　　天地异相！
<br/>　　在这远方数百里，都足以清晰的看到这天地异相。
<br/>　　有大型宝藏出世了！
<br/>　　楚风眠眼神之中露出几分喜色，这大型宝藏出世了，而且就在距离楚风眠不远处的地方。
<br/>　　在这赤红大陆之中，所出现的大型宝藏，许许多多都是曾经天人洞府，或者是一些称霸九域一个时代的势力遗迹。
<br/>　　无论是哪一种，任由楚风眠得到，都是一大机遇。
<br/>　　在这赤红大陆之中，一些古圣药，材料，都不算是什么奇遇了，只是从这些大型宝藏之中流出的冰山一角。
<br/>　　这些大型宝藏，才是真正的机缘。
<br/>　　“有大型宝藏出世了！”
<br/>　　“快去通知宗门！”
<br/>　　“我们先去，碰碰运气，没准这宝藏就是我们的了！”
<br/>　　“大型宝藏，十有八九之中都有着天人之物！我们的济源到了！”
<br/>　　在这赤红大陆之中，不知道是聚集了多少个星域，多少个势力的无数强者。
<br/>　　看到这大型宝藏的出世，纷纷一道道遁光，直接是冲了过去，都打算是抢先一步，占据先机。
<br/>　　每一个大型宝藏开启的方式都不一样，有的是需要信物才可以进入其中，而有人则是人人可以进入，谁先到就归谁的。
<br/>　　所以无数武者，都是将遁光的速度催动到了极致，赶了过去，古帝，真帝，甚至是一些准帝，都想要去碰碰机缘。
<br/>　　“走。”
<br/>　　楚风眠也毫不犹豫，化为了一道遁光直接向着那七彩霞光的方向飞去。
<br/>　　这大型宝藏，距离楚风眠不远，只有十里左右的距离，以至于楚风眠都可以看见，在这七彩霞光背后隐藏的一座宫殿。
<br/>　　楚风眠的遁光，快到了惊人的地步，刹那之间便是超越了无数武者，一瞬之间就来到了这七彩霞光的面前。
<br/>　　只见在这七彩霞光的最下方，突然显现出了一道古老的铜门，如今这古老铜门紧闭。
<br/>　　但是上面的纹路，却是在一点点的被唤醒，当这纹路被唤醒的一刻，应该就是这大型宝藏开启的一刻。
<br/>　　“不知道这是九域哪一位强者所留？”
<br/>　　楚风眠看着七彩霞光，想要从背后那模糊的宫殿之中看出一番究竟来。
<br/>　　但是仅仅是从这七彩霞光背后的虚影，楚风眠看了几眼，也难以认出来。
<br/>　　九域历史无比久远，在许多的古籍之中，数十万年的故事，都有记载。
<br/>　　曾经位列九域古籍之中赫赫有名的强者也众多，楚风眠虽然熟读过九域的古籍。
<br/>　　可单单凭借这一方宫殿的虚影，就想要看出其中究竟来，也太过困难了。
<br/>　　“已经有不少人到来了。”
<br/>　　楚风眠扫视了一眼，在这周围已经是聚集了十几名武者了。
<br/>　　他们的实力，多是古帝，真帝境界，其中最强的两个人，才是八劫古帝的境界。
<br/>　　如今都站着远远的，隐藏在周围的空间之中。
<br/>　　宝藏没有开启，自然是没有争夺的必要，只是占据好位置，一旦是宝藏开启，便是足以先发制热，抢先一步。
<br/>　　三大势力的武者，倒是并没有，这周围方圆百里，三大势力的古帝，都已经是被楚风眠全部轰杀了。
<br/>　　所以在这里的武者，都不是属于三大势力的，许多都是来自于大罗天星域其他一些大陆，或者其他一些势力的。
<br/>　　“燕莫，燕无峰，你们两个人还想往哪里逃？将那渡厄圣叶交出来！不然的话你们逃到天涯海角去，也没有能救得了你们！”
<br/>　　轰隆。
<br/>　　突然之间两道遁光，向着这边飞了过来，这两道遁光，是两名年轻男子，看起来年纪不大，但是一身实力都已经是达到了七劫古帝的境界。
<br/>　　可以说是小天才了，足以在圣龙秘境之中晋升圣子了。
<br/>　　如今这两名年轻男子，却是一路遁逃，有些慌不择路的感觉，飞到了这边来。
<br/>　　而在他们的身后，五名武者，也正在催动遁光，在疯狂的追逐。
<br/>　　这五名武者，实力都是七劫古帝巅峰，身上弥漫着一身血腥味，有着几分魔道功法的气息，不过却并非是属于六道魔门的魔功，而是来自于其他势力。
<br/>　　魔道，不止是在九域存在，在大罗天星域之中的许多大陆，也有魔道宗门的存在。
<br/>　　只是魔道在大罗天星域的势力不算强，据说是在一些其他星域之中，都是以魔为尊，反倒是少有修行武道的武者。
<br/>　　看到这五名武者一路血色遁光，追逐而来，不少人的脸色都是猛然大变。
<br/>　　“是血虎五子！”
<br/>　　“什么人被这几个凶人盯上了？”
<br/>　　一名名武者脸色大变，都是急忙退去，生怕的招惹到这五名武者。
<br/>　　不过有一名武者，却是没有反应过来，还没有来得及离开，正好挡在了这血虎五子的面前。
<br/>　　“挡路者死！”
<br/>　　这血虎五子其中为首一人，冷哼一声，一掌拍了过去，直接将这一名挡在面前的一名五劫真帝，一掌拍死，随即张开嘴，将一团精血直接吞噬殆尽。
<br/>　　随即这遁光，又是继续向着那两名年轻男子追了上去。
<br/>　　“这二人修行的，是燕州大陆的功法？莫非是坦洲大陆的武者？”
<br/>　　楚风眠倒是没有在意那所谓的血虎五子，而是将目光看向了那被追逐的两名年轻男子。
<br/>　　他们二人身上的功法，楚风眠倒是感觉到了几分熟悉，跟燕皇身上的功法，无比接近。
<br/>　　看来多半就是燕州大陆的武者。
<br/>　　燕州大陆只是六级大陆，古帝都不会很多，看着二人身上的功法，多半是跟燕皇关系极为密切，极有可能更是燕皇弟子。
<br/>　　如今这二人，已经是被那血虎五子逼道极限了，他们二人的身上，都有着不少的伤势。
<br/>　　现在是在强行催动遁光，想要逃走，可这样强行催动，终有到极限的一日，没有人出手，这二人必定是死在血虎五子的手中。
<br/>　　眼看着，那二人的遁光，都要被血虎五子给追上了。
