        毕竟，是她把他气出病来的……
<br/>
<br/>        “叮”地一声，电梯抵达顶层，门一开，童筱颖刚走出去，迎面就遇上了正往电梯口这边走过来的落以柔。
<br/>
<br/>        落以柔一袭浅粉色西服裙，扎着马尾，手中也提着饭盒，饭盒随着她的手摆动着一定的弧度，很显然，她的饭盒已经空了。
<br/>
<br/>        “傅总已经吃过晚饭了，童小姐，请回吧！”落以柔将童筱颖打量了一番后，看到她与自己的衣服撞了颜色，顿了顿，接着嗤之以鼻地说道，“你这身打扮是来探望病人的，还是来勾引病人的？”
<br/>
<br/>        “？！”童筱颖错愕地低头看了看自己的着装，觉得没什么不妥的地方，不懂这个女秘书为什么说这种带刺的话。
<br/>
<br/>        “还不回吗？童小姐！”落以柔见童筱颖不吭声，语气极冷地接着说。
<br/>
<br/>        这女秘书真的超凶啊！
<br/>
<br/>        童筱颖轻咬着唇瓣，顿了下，淡定地咧嘴一笑：“我来看我老公，与你何干？”
<br/>
<br/>        “老公？！”落以柔像是听到了一个天大的冷笑话一般，微微扬起下巴，讥笑道，“所以，你才明目张胆地跟小叔子私奔，共度良宵，给傅总戴绿帽，把傅总气得吐血住院后，才觉悟傅总是你的老公？”
<br/>
<br/>        两个漂亮的女人站在电梯口起了争执，因此暗中关注她两的路人很多。
<br/>
<br/>        而落以柔的这句话一出口，周围的人神情都是一愣，随之一个个用一种鄙夷的目光砸向了童筱颖。
<br/>
<br/>        童筱颖垂下眼帘，心里又气恼又无奈，但对落以柔的这番话并不惊讶。
<br/>
<br/>        感受到四周投来的各种鄙视的目光，童筱颖有些不知所措，她用力抿了抿唇，努力挤出一抹微笑，落落大方地反驳：“落秘书，你大庭广众之下，对你上司的太太造谣生非，充满敌意，挑拨我和我老公的关系，你这是图什么？”
<br/>
<br/>        “我可没挑拨你和傅总的关系！你少在这胡说八道！”落以柔顿时又气又忧，瞪着童筱颖咬牙切齿。
<br/>
<br/>        “一直在胡说八道的人不是你吗？”童筱颖冷冷地回敬了落以柔一句，伸出手推开挡在面前的她，走了几步后停了下来，转过身对落以柔说，“落秘书，顺便提醒你一句，你若想继续留在傅氏集团，最好不要来招惹我。”
<br/>
<br/>        童筱颖说完，对落以柔扬起一抹浅浅的微笑，然后头也不回地走了。
<br/>
<br/>        留在原地的落以柔愤愤不满地瞪着童筱颖离去的背影，被她的话气得火冒三丈。
<br/>
<br/>        这个女人，简直就是在恃宠而骄！
<br/>
<br/>        她就不明白，傅夜沉到底喜欢这个童筱颖什么？只不过是个二手货，被其他男人穿过的破鞋。
<br/>
<br/>        殊不知，至于她童筱颖到底是不是二手货，是不是破鞋，傅夜沉一点都不在乎。
<br/>
<br/>        童筱颖走到病房门口时，只见傅夜沉正翘着二郎腿托着笔记本电脑，坐在沙发上，一手搭在扶手上打点滴，一手在笔记本电脑键盘上快速敲击着。
<br/>
<br/>        看样子，他确实已经吃过晚饭，正在加班工作了。
<br/>
<br/>        虽然，他身上穿着蓝白条纹的病号服，但他那张带着几分高冷气质，如刀镌般俊美的脸庞，以及举手投足间自然散发成熟男性的沉稳与内敛，足以让女人为之着迷。
<br/>
<br/>        童筱颖怀里抱着饭盒，臂弯上挂着装有衣服的纸袋，就站在病房门边踟蹰着。
<br/>
<br/>        进去？还是不进去？
<br/>
<br/>        反正他都已经吃过饭了，而且他又正在加班，不如不进去了？
<br/>
<br/>        可是，她都来了，不进去岂不是白跑一趟？
<br/>
<br/>        刚刚，她还可以在落以柔面前，傲气十足地声称自己是来探望老公的，现在，她又心虚地怂了。
<br/>
<br/>        什么时候起，她开始把他当成是自己的老公呢？
<br/>
<br/>        她一直以为，他喜欢的人有季思妍、苏漫雪、那个什么悔婚了的未婚妻、还有那个女秘书落以柔，结果到头来，都是她一个人在这儿自以为是。
<br/>
<br/>        忽然，一阵悦耳的手机铃声响起。
<br/>
<br/>        童筱颖连忙转过身去，一边走开一边在手提包里掏手机。
<br/>
<br/>        此时，傅夜沉被那道铃声所吸引，他抬眸时，只看到一抹粉色的身影从门口掠过。
<br/>
<br/>        筱颖？不对，难道是小落还没走吗？
<br/>
<br/>        傅夜沉皱了下眉头，收回目光，继续看着笔记本电脑屏幕工作，但心里却总想着童筱颖那个小女人。
<br/>
<br/>        童筱颖跑进了安全通道里，才接听了电话。
<br/>
<br/>        “大少奶奶，还有份鲍鱼鸡汤，你忘记带了。”是刘婶打过来的电话。
<br/>
<br/>        童筱颖觉得自己好像有理由不进去了，立即笑着回答道：“那我马上回来接！”
<br/>
<br/>        她挂了电话后，半垂着水眸，暗自窃喜地转身，谁知一头撞到了一个健硕的胸膛上。
