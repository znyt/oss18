　　第443章
<br/>　　飞箭如雨，犬走鹰飞，草原上赫赫一片杀机。
<br/>　　一支百人伍的轻骑兵从荒草中杀出来，对程大雷三人展开追逐战。
<br/>　　“我擦，还有伏兵！”程大雷『牛』不停蹄，握着剑大骂。
<br/>　　在他们逃跑的方向上，出现一伙伏兵，身披重甲，手持铁矛。
<br/>　　此时，伏击程大雷三人的人数已经超过三百人。
<br/>　　“不按江湖规矩来，不地道呐！”徐神机跑得气喘吁吁：“大当家，怎么办，我实在跑不动了。”
<br/>　　果然，情况比自己想得还要严峻。本来程大雷思考的是，能否及时赶到目的地，救下林少羽。现在看来，他自己能否活着走出这片草原都是个问题。
<br/>　　如果是程大雷自己，当不用担心脱身的事，但有徐神机这个大尾巴，却是不得不考虑。
<br/>　　弓兵、轻骑兵、重甲兵三支队伍已展开合围，密不透风，程大雷目光转了一圈，眼神忽然一厉。
<br/>　　“来，斧头给我。”
<br/>　　鬼面斧交在程大雷手中，程大雷已准备大开杀戒。
<br/>　　“保护好军师，跟紧我。”
<br/>　　程大雷骑着黑牛，向轻骑兵展开逆冲锋。身后发生的一幕，程大雷没有看见，否则一定会惊得下巴掉在地上。
<br/>　　徐神机骑在马上，心惊胆战，银眸竟然抓住他的衣领，将他拎到自己马背上。徐神机趴在马背上，口中大喊大叫，四肢乱动。
<br/>　　银眸跟着程大雷冲锋。
<br/>　　墨玉照月兽的野蛮冲撞技能发动，直接撞翻了两匹马。程大雷手中的鬼面斧抡了出去，刑天三斧技能发动，一股狂躁的力量从体内泛起，双目猩红，须发倒竖。
<br/>　　有一人不小心挨着程大雷的斧头，脑袋直接从腔子上被削了下来，喷出一腔热血。
<br/>　　程大雷见血更狂，这三斧本就是偏残暴的功夫，适合乱军混战。刚得到这功夫的时候，程大雷还是废了好大功夫才适应它，今天算是驯服之后，第一次放出来。
<br/>　　一片人头落地，有人肢体分离，鲜血乱飞。真正的战争本就是如此，干净且残忍，容不得半点体面和温情。
<br/>　　好在程大雷、徐神机都是久经沙场的老油条，这一幕还不会令他们畏惧。至于银眸，她似乎生来不知恐惧为何物。
<br/>　　程大雷陷在轻骑兵队伍正中央，而弓兵与重甲兵两支队伍也向这边合拢，将程大雷围在大众。
<br/>　　随着杀人数的增长，程大雷也在不停收获恐惧值，这一战，算是补充了程大雷最近一直缺乏的恐惧值。
<br/>　　当然，他现在的目的是突围，而不是杀人。要想凭程大雷一人屠尽这三百人……虽然因为刑天三斧的缘故，程大雷有些狂，但确实没狂到不知天高地厚的程度。
<br/>　　而如此数目的士兵，的确也拦不住程大雷，程大雷担心的是徐神机。
<br/>　　他身后紧跟的银眸，不知何时夺了两根长矛，一手一个，也没什么章法，在程大雷身后乱挥。
<br/>　　她一身凭空而来的力气，手中长矛也不刺，简单就是一个字：砸。被她砸到的，非死即伤。
<br/>　　程大雷抽空回头看了一眼，吓了一跳。这姑娘的力气，还是要重新估计呐，看着像个用锤的，有时间该给她打一柄大锤，看会有什么效果。
<br/>　　趁着程大雷回头的光景，一柄剑向他袭来，所以说战场之上，容不得半点失误，哪怕程大雷反应及时，这一剑也刺破了他的肩膀。
<br/>　　程大雷看到面前一白袍男人，敢在战场上穿布甲，不是装X就是蠢货。程大雷大概估计到眼前这人的身份。
<br/>　　四大才子之一：剑客。
<br/>　　程大雷眉头拧起，忽然放声大笑：“能刺中我这一剑，你这一辈子死而无憾。”
<br/>　　声音落地，程大雷一斧迎头劈了过去。对方举剑相格，直接被大斧将剑砸成两段，紧跟着劈到书生脑袋上。
<br/>　　对方从头到尾，被劈成两段，身体从马背上分开，跌落在地上。
<br/>　　“我就说嘛，提剑上战场的，都是傻瓜。”
<br/>　　沉浸在这一战中的所有人，都没有注意到，远处山坡上有两个人，正向这边张望。
<br/>　　“那人就是传说中的程大雷么？”一身着青衫的男人道。
<br/>　　“骑牛背斧，身后跟着一个没表情的姑娘，应该是他无疑了。”另外一人穿着花裳，俊美非常，比平常女人都要妩媚几分。
<br/>　　“果然名不虚传呐，书生的正道剑，只来得及刺出一剑，而且还是偷袭，便被他劈开了。举世高手，能做到这一点的，估计不超过十位。”
<br/>　　“但程大雷只能算半个江湖人，据说他在幽州还有一捎人马，平日根本用不着亲自上阵的。”
<br/>　　“青牛山，蛤蟆寨，怎么出了这么个妖孽。”花裳男摇头叹气道。
<br/>　　“我们要不要出手帮一下他们，再不出手，看他们就要冲出去了。”青衫男人道。
<br/>　　“帮什么，怎么帮，我们出手也是去送死而已。”花裳男道：“更何况相府和将军府本就水火不容，我们还是袖手旁观，准备好我们的局吧。”
<br/>　　二人远远观战，所料也是不差，大概也就是一炷香时间，程大雷已经带着银眸冲出了重围。
<br/>　　本来的包围圈，生生被程大雷趟开一条血路，地上堆满了残肢断骸，鲜血流了一地。
<br/>　　原来的包围圈，彻底被击溃，没有事先的计划，想要留住程大雷谈何容易。弓兵与重甲兵几乎已对程大雷形成不了威胁。那残余的轻骑兵队伍，分出十几股人向追来。
<br/>　　程大雷浑身浴血，都是别人的血溅到他身上的，此刻顺着大氅湿漉漉往地上流。
<br/>　　程大雷突然回头，开口爆喝：“滚！”
<br/>　　嘶！
<br/>　　几匹迫得紧的骏马昂头长鸣，竟前蹄发软，噗通噗通跌倒在地上。
<br/>　　人仰马翻一片，马犹如此，何况马上的人，太多人已经被程大雷的血腥吓破了胆。
<br/>　　虽然程大雷也是两个肩膀顶一个脑袋，但怎么看不像个人，却像一只天地凭空而生的恶魔。
<br/>　　人人畏惧，心惊胆战，不敢向前。而这个时候，程大雷已经骑着黑牛，带着徐神机和银眸向远处逃去。
<br/>(本章完)
