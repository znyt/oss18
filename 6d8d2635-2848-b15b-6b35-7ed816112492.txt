　　“少阳，我跟静茹退出了，对你也是好事，但你还是得做出抉择，你只能娶一个人……”
<br/>　　叶少阳知道她说的是什么，点点头，心里在想，自己怎么就陷入了这种境地。与法术界有关的事，天大的事他也不怕，但就是这感情问题，一直剪不断理还乱。
<br/>　　时间一点点流逝，到了登机的时间，谢雨晴站起来，把包背在身上，冲叶少阳大方地笑了笑，道：“天下无不散的宴席，你也不要难过了，抱一下吧，有始有终，我就当是跟你恋爱过一场了。”
<br/>　　叶少阳还在怅惘，谢雨晴主动抱了他一下，在他耳边轻轻说道：“少阳，你保重了。”
<br/>　　松开他，又看了看他，朝检票口走过去了。
<br/>　　“雨晴！！”
<br/>　　叶少阳突然大喊了一声，引得边上几人都在看他，谢雨晴也愣了一下，然后朝他挥了挥手，过了安检。
<br/>　　走上浮桥，谢雨晴又转过身来，双手背在身后，很清脆地叫了他一声：“少阳啊！”
<br/>　　叶少阳几乎要冲过去了，谢雨晴却再次朝他摆了摆手，“你一定要保重啊，加油！”
<br/>　　然后她转身，偷偷抹了一下眼睛，走上了飞机。
<br/>　　留下叶少阳一个人，茫然无措地站在登机口，心如刀绞。
<br/>　　如果相识是为了分别，那么相识的意义是什么呢，只是为了留一段记忆，去怀念某个人？
<br/>　　在最痛苦的时候，叶少阳没来由地想起了昨天跟地藏菩萨讨论的时候，地藏菩萨所谓的‘空’，当时自己不解其意，就在方才一瞬间，他明白了，空，的确不是没有，而是什么都记得。
<br/>　　因为一起过，所以懂得什么是别离，因为拥有过，才会懂得失去。因为爱过，才会懂的被爱。
<br/>　　终归劫数尽，尘世也成空。
<br/>　　这才是真正的空。
<br/>　　他有点悟了，因为他失去了。
<br/>　　飞机启动的一刻，谢雨晴看着窗外的蓝天，从包里拿出了灭灵钉——虽然很锋利，但因为它长得实在不像是凶器，上面还有花纹和图案，被当做是文玩饰品，安检并没问题，谢雨晴不放心托运，因此带在了身上。
<br/>　　将灭灵钉拿在手中，谢雨晴轻轻摩挲着上面的花纹，感觉就像摩挲叶少阳的脸。
<br/>　　就这样了吗？
<br/>　　就这样了吧。
<br/>　　谢雨晴深吸一口气，尽管知道叶少阳看不到，还是微笑着，对着窗户轻轻挥了挥手，像是完成了一个仪式。
<br/>　　然后她把目光落在灭灵钉上，在她眼中，这算是叶少阳留给她的最后的东西，她并不知道，命运中已经注定，这东西在多年以后，会带给她什么，又会在三界之内引起怎样的震动。
<br/>　　这都是多年后的事了。
<br/>　　叶少阳一个人颓然在机场呆到了中午，激活瓜瓜的魂印，半小时后，瓜瓜来了，看叶少阳坐着发呆，四下张望了半天，问道：“雨晴姐呢？”
<br/>　　“她去了云南，不回来了。”
<br/>　　“啥意思。”瓜瓜眼睛瞪得老大。
<br/>　　“不要细问了，你……陈悦呢？”
<br/>　　“去了一个很安全的地方，她开始修炼了。”瓜瓜看了叶少阳一眼，有点不自在。他跟随陈悦这么久，知道她虽然现在实力羸弱，不是天赋不行，相反，她天赋极高，但是多年来修炼出的罡气，都化作戾气了，现在她学会七宝绝经，能炼化戾气，最近几天他在旁边看着，陈悦进步惊人，估计要不了多久，她就能跻身一流强者。
<br/>　　瓜瓜看在眼里，喜忧参半。他当然希望陈悦能厉害点，但又害怕将来有一天会威胁到叶少阳……
<br/>　　叶少阳不知道他想了这么多，又一个人坐着发呆了，瓜瓜看在眼里，猜测八成是跟谢雨晴有关，但也不好问太多，干脆就坐在一边陪着他。
<br/>　　上飞机之前，叶少阳又尝试在微信上跟谢雨晴联系，但她没回。
<br/>　　两个小时的飞行，等到了石城，叶少阳赶紧打开手机，发现谢雨晴在一个小时前回了条信息，只有一张图片，湛蓝的天空，漂浮着一些云彩，应该是她到了云南之后拍的，只有这张图。
<br/>　　云淡风轻的意思吗？
<br/>　　叶少阳本想回她一条什么，想了想，只回了珍重两个字。
<br/>　　还有一条微信，是雪琪发来的，告诉他，自己也去了云南，去陪谢雨晴了，让他保重，如果有需求就找她，她一定会来帮忙。
<br/>　　虽然是自己的鬼仆，但她跟叶少阳更像是朋友的关系，听说她去陪雨晴，叶少阳也是开心的，一来雪琪能保护她，二来也能陪着她，度过这段难熬的日子。
<br/>　　还有一个未接来电，是苏钦章打来的。
<br/>　　他没事不会给自己打电话，叶少阳有点担心，立刻回了电话，苏钦章接的，先问候了几句，便说到了正事：原来是龙虎山那边新任掌门继位，给茅山下了帖子，在十天之后的腊月十五，举办正式的典礼，邀请了诸多门派参加，自然也包括茅山。
<br/>　　叶少阳跟苏钦章商量了下，让他备礼，到时候提前一天过去，在鹰潭碰头。
<br/>　　接着叶少阳问了茅山的近况，苏钦章汇报了一下，最近茅山收了不少弟子，其中也有几个资质不错的，正在培养。
<br/>　　“对了掌门师兄，这快过年了，您今年要不要回山过年，搞点什么活动？”
<br/>　　道教是过春节的，而且一般还有举行一些祈福道场之类的活动，一向都是本门掌教主持，叶少阳想了一下，表示如果没意外的话，自己年底之前回去。
<br/>　　挂上电话，叶少阳心中感慨起来，想到去年春节，自己还跟青云子在山上乐乐呵呵地一起过节，这一年来……却是发生了太多太多的事，物是人非。
<br/>　　叶少阳叹了口气。
<br/>　　“老大，你最近……好像心情不太好？”瓜瓜有些敏感地问道。
<br/>　　“赶紧自己好像一下子老了很多，也许这就是成长吧。”叶少阳勉强笑了笑。
<br/>　　落寞地回到自己家。
<br/>　　包子和碧清都在，正在看电视，见到他都吃了一惊，他们并不知道他今天回来，叶少阳提前没说。碧清问他事情办的怎么样了。
