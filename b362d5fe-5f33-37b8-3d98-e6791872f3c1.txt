　　楚风眠炼化了太初之气，度过了逆天大劫，已经是成为了始祖天龙那般的天地异种。
<br/>　　现在天魔老人只要是可以将楚风眠吞噬，便是将楚风眠取而代之。
<br/>　　从一开始这天魔老人便是都在算计着，要将楚风眠吞噬。
<br/>　　刚刚天魔老人故意是被楚风眠打飞出去落败，实际上只是在隐藏着他，一直是隐藏在暗处，等待着机会。
<br/>　　现在机会终于来历，天魔老人一出手，便是就将楚风眠成功斩杀。
<br/>　　看着近在咫尺就可以成为天地异种的机会，天魔老人大笑一声，右手猛然对着楚风眠抓了过去。
<br/>　　可就在这天魔老人突然出手抓向楚风眠身躯的一刻，感受到了却是一片虚无。
<br/>　　楚风眠的身形，就在他的手心之中，缓缓消失。
<br/>　　“什么？”
<br/>　　天魔老人脸色骤然一变。
<br/>　　那被他抓入到手心之中的楚风眠，其实只是一道虚影。
<br/>　　刚刚那魔刀斩杀的，只是一道楚风眠的虚影。
<br/>　　“天魔老人，你以为我察觉不到你的算计？”
<br/>　　楚风眠的身形，缓缓就从天魔老人的面前显现而出，嘴角露出几分玩味笑容道。
<br/>　　这天魔老人的算计，从一开始楚风眠便是看的一清二楚，他故意露出破绽，也是逼迫这天魔老人出来罢了。
<br/>　　“你！”
<br/>　　天魔老人看到楚风眠的现身，都是浑身战栗起来，他的一切算计，都瞒不过楚风眠的双眼。
<br/>　　他本以为可以趁机夺得渔翁之利，可现在一切的计划，都被打破了。
<br/>　　看着一副玩味笑容的楚风眠，天魔老人也已经是骑虎难下了。
<br/>　　他也明白，以楚风眠的性格，绝不会放过他。
<br/>　　“天魔解体刀！”
<br/>　　轰！
<br/>　　天魔老人高声咆哮一声，他的身躯突然炸裂开来，化为了滚滚的魔意。
<br/>　　这天魔老人也是拼了命了，直接舍弃了自己的肉身，滚滚魔意，就在空中化为了一把魔刀。
<br/>　　这一把魔刀，便是天魔老人舍弃肉身，以自身血气凝聚而成的。
<br/>　　凝聚成的魔刀，向着楚风眠一把斩杀而去。
<br/>　　天魔老人也是心志坚定之辈，自知逃命不成，就只有跟楚风眠血战这一条路了。
<br/>　　毕竟天魔老人可是半步天人境巅峰，无限接近于天人境的人物了，如今以身化刀，爆发出的刀锋，俨然超越了半步天人境的范畴，媲美真正天人打出的仙术。
<br/>　　刀锋所斩之处，星河都纷纷破碎。
<br/>　　在这最后的时刻，天魔老人也施展出了他最强的手段，拼命的手段。
<br/>　　知道这个时候众人才了解到，身为一代魔道宗师的天魔老人实力的强横。
<br/>　　不过任由这刀锋在强，楚风眠的表情依然俨然不动，只是他手中的灵剑之上，剑意也在凝聚，突然之间，楚风眠手中的剑锋动了。
<br/>　　祖龙至尊剑之中，龙力，剑道之力，合二为一。
<br/>　　“吼！”
<br/>　　剑锋斩出的一刻，惊天的龙吟咆哮而出，就跟那一把魔刀，凌空碰撞。
<br/>　　楚风眠的剑锋，再一次的展露出来，在这剑锋之中，隐隐约约都足以感受到始祖九式的力量。
<br/>　　楚风眠已经是将始祖九式，融入到了他的剑锋之中，化为了九域剑术的一部分。
<br/>　　这在无数人看来，简直是梦幻一般的事。
<br/>　　始祖九式，乃是始祖天龙所创，代表着天下间万千龙族武技的本源，称得上是龙族本源之一。
<br/>　　许许多多古籍之中所记载的，古老时代之中的龙族强者，想要将始祖九式全然参悟出，都困难无比。
<br/>　　跟何况现在楚风眠还要将始祖九式化为剑术？
<br/>　　始祖九式论起威力可是要远超九玄仙术，正常来说，只怕是楚风眠一身境界达到仙帝境界，都没有能力将其转化为剑术。
<br/>　　可现在的楚风眠，却是不同了，他得到了属于那始祖天龙之爪的太初之气，已经是参悟了龙之本源。
<br/>　　始祖九式的玄妙，楚风眠已经是全然参悟出了，足以是将这始祖九式，转化为剑术。
<br/>　　楚风眠毕竟是一尊剑修，哪怕他得到龙族血脉，也依然是一尊剑修，剑术，才是楚风眠最强的手段。
<br/>　　“从此之后，这就被称为始祖九剑吧。”
<br/>　　轰！
<br/>　　楚风眠身上的剑意迸发而出，惊天而起，就在楚风眠身躯之中那剑道古籍之中。
<br/>　　崭新的两页，再度凝聚了出来。
<br/>　　这两页之中所记载的，便是始祖九式的前两式，不，现在这应该是被称为始祖九剑了。
<br/>　　崭新的两页凝聚出来，融入到了剑道古籍之中，便是代表着这两找，已经是彻底的化为了属于楚风眠的九域剑术了。
<br/>　　不过始祖九剑，现在楚风眠只是凝聚出了前两剑，至于剩下的七剑，只有等到楚风眠参悟出始祖九式的后七式之后，才可以再度凝聚。
<br/>　　楚风眠的身上，属于龙族血脉，剑道的两股力量，随着始祖九式融入到了楚风眠的九域剑术之中，居然是开始融合了起来，变的紧密了起来。
<br/>　　这两股力量，在剑锋之下，已经是合二为一。
<br/>　　“始祖九剑！第一剑！”
<br/>　　耀眼的剑光冲天而起，剑气之力凝聚，化为了一尊始祖天龙的模样，咆哮而出。
<br/>　　滚滚龙威，震慑天地。
<br/>　　跟那魔刀所碰撞的一刻，那天魔老人舍弃肉身，化为的魔道，如今在这始祖九剑的面前，都是不堪一击，碰撞之下，轰然破碎。
<br/>　　魔刀被破，天魔老人便是又被打回了本来的模样，不过他身上的力量，已经是无比虚弱了。
<br/>　　舍弃了肉身精血，对本来就已经是寿元将尽的天魔老人来说，已经是最后的拼命了。
<br/>　　现在他这最后的手段，都是被楚风眠化解，他已经是没有了任何的手段。
<br/>　　“死吧！”
<br/>　　楚风眠也没有手下留情的想法，又是一道剑光，直指那天魔老人斩杀而去。
<br/>　　“砰！”
<br/>　　天魔老人的身躯就在空中，被楚风眠的一道剑光所贯穿。
<br/>　　天魔老人，陨落。
<br/>　　这一次一共十三位半步天人境强者，前来围攻楚风眠，现在却是无一例外，全部陨落了。
<br/>　　“大罗天星域，变天了。”
<br/>　　一尊九劫古帝，见到这一幕，喃喃自语道。
