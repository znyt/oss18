        希望，这一次，她没有爱错人……
<br/>
<br/>        在大大的木质浴缸里洗完花瓣澡后，童筱颖的新娘红妆是由婆婆冷太太亲手为她梳妆打扮的。
<br/>
<br/>        新娘霞帔，是傅夜沉在跟她领证后，花钱请了国内最优秀的新娘红妆设计团队替她量身定制。
<br/>
<br/>        而梳妆台上那顶纯金的凤冠，则是祖上传下来的。
<br/>
<br/>        “嫁入傅家的女人，都得穿凤冠霞帔，我当年也是这么嫁给小沉的父亲的。”冷太太一边替童筱颖梳头，一边笑容满面地回忆着自己的往事，“那个时候，小沉的父亲是八抬大轿娶我进门，我便是十里红妆相嫁于他。”
<br/>
<br/>        冷太太娘家地位很高，童筱颖有所耳闻，所以冷太太所说的“十里红妆”场面一定非常壮观。
<br/>
<br/>        童筱颖听着冷太太讲述着过去的喜事，心里艳羡，却不憧憬。
<br/>
<br/>        虽然婚礼有些匆忙，但是并不从简。
<br/>
<br/>        除了没有八抬大轿、放鞭炮、奏喜乐以外，其余的繁文缛节，全部按部就班。
<br/>
<br/>        古人结婚，选择黄昏时刻，一拜天地、二拜高堂、夫妻对拜，再是送入洞房。
<br/>
<br/>        洞房花烛夜。
<br/>
<br/>        镶金的红烛，烛火微微摇曳，红鸾帐暖，床上铺满红枣、花生、桂圆、莲子，整个地面铺满玫瑰花瓣，让这个古今结合的婚房充满着温暖又喜气的暧昧。
<br/>
<br/>        童筱颖真觉得自己这是穿越，而且头顶这凤冠感觉是千斤重，压得她脖子都快断了。
<br/>
<br/>        傅夜沉拿杆秤，俐落地掀起了童筱颖凤冠上的红巾。
<br/>
<br/>        或许是因为对他已经太熟了，她此刻没有了那份娇羞，反而随着红巾被挑起，自顾自地撩开头上凤冠前的珠帘。
<br/>
<br/>        待视线无阻后，她明亮的水眸，与傅夜沉四目相对。
<br/>
<br/>        此刻，她可以把他的俊美的容颜，看得一清二楚。
<br/>
<br/>        傅夜沉真的好俊、好好看，好看到让她忍不住抿唇微笑。
<br/>
<br/>        这看着看着，童筱颖忽然间就害羞了。
<br/>
<br/>        她赧然低下头去，屏气凝神，脸蛋上的绯红颜色也更加明艳动人。
<br/>
<br/>        “以后，我给你补办一个西式婚礼。”傅夜沉在床沿边坐下，伸手挑起童筱颖的下巴，再度凝视着她的甜美脸蛋。
<br/>
<br/>        他这一看，闪烁着迷恋与爱慕的黑眸就再也移不开了。
<br/>
<br/>        童筱颖小脸胀红，还咳了起来。
<br/>
<br/>        “怎么呢？”傅夜沉一惊，发觉她呼息极不顺，像在闭气，连忙轻抚她的背，“你要不要紧？”
<br/>
<br/>        “我……我……”她轻喘了起来，她方才竟然不自觉地憋起气，“我感觉自己在做梦，还感觉自己好像是穿越了，又或者我跟你只是在拍古装剧。”
<br/>
<br/>        “你真是个小傻瓜。”他拇指指腹轻抚着她娇美的下颚，深情地凝望着她。
<br/>
<br/>        “大哥，你是不是有事瞒着我？”她的心在他的抚摸下又飞上天，思绪也跟着恍恍惚惚起来，“我总觉得不对劲。”
<br/>
<br/>        “还叫我‘大哥’嗯？”
<br/>
<br/>        “大哥、大哥、大哥！我不管，我就要叫你大哥！”童筱颖俏皮地笑了笑。
<br/>
<br/>        傅夜沉宠溺地刮了刮童筱颖的鼻梁，然后替她卸下了头上的凤冠。
<br/>
<br/>        她那如缎般的黑发散了下来，落在大红的嫁衣上，再配上她那宛若皓雪一般白皙柔嫩的肌肤，沉黑、艳红、雪白交错，就像一幅醉人的新娘图，让他不由地看得入迷出神。
<br/>
<br/>        其实，他觉得自己才是在做梦，因为，她终于成了他的妻……
<br/>
<br/>        “你还没回答我的。”童筱颖接着又复述了一遍，“大哥是不是有事瞒着我？”
<br/>
<br/>        “没有。”
<br/>
<br/>        “你说你有任务在身，爷爷一听就能理解，可我不理解，是什么任务？”
<br/>
<br/>        “等任务完成了，我再告诉你，好不好？”傅夜沉会心一笑。
<br/>
<br/>        童筱颖也不再追问，而是很乖巧地点了点头，接着说道：“我们去吃点东西，我肚子饿了。”
<br/>
<br/>        婚床前方的圆桌上可是摆着六碟下酒菜和一壶酒，而两个小酒杯则用一根红线连在了一起。
<br/>
<br/>        傅夜沉牵着童筱颖的手，走过去一起坐在了圆桌前，主动倒了两杯酒，然后端起其中的一杯，递给了童筱颖。
<br/>
<br/>        “要喝交杯酒，对吧！”童筱颖接过酒杯，抿嘴一笑。
<br/>
<br/>        傅夜沉微微点头，然后跟她绕过臂弯。
<br/>
<br/>        两人一起将杯中的酒一饮而尽。
<br/>
<br/>        这酒很甜，入口丝滑，不辣，味道有点像糯米酒。
<br/>
<br/>        “好甜。”童筱颖觉得这酒味道不错，拿起酒壶还想再给自己倒一杯来喝的时候，身体却被男人的手臂腾空抱起来。
<br/>
<br/>        下一秒，他毫无征兆地攫住了她的红唇。
<br/>
<br/>        “大哥……唔……”她张嘴惊唤的那一瞬间，一丝甜糯的酒，从他的吻中滑入到她的味蕾上。
