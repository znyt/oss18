<br/>    三天之后，陈明看着众多的妖王带着麾下的大妖前来上交灵兵，白狐和黑蟒的属下正在迎接一位位妖王办身份文牒。
<br/>    
<br/>        黑蟒在陈明身旁道：“山主，燕山的炼丹房，乃是燕山的崛起之谜，就这么交给这些妖王，真的没事吗？”
<br/>    
<br/>        黑蟒所担心的，还是这些妖王强大起来之后，燕山会掌控不住局面。
<br/>    
<br/>        陈明笑了笑，还是太嫩了，修行者啊，虽然很聪明，但是却是聪明的过了头，都用到修行上面去了。他们永远不会明白陈明的做法，也想不通陈明的做法。
<br/>    
<br/>        这是一个现代人的智慧，没有任何东西，可以和真正的国家机器抗衡，以前的修行者控制方式，实在是太低劣了。
<br/>    
<br/>        “你要明白最深刻的一点，那就是他们从此以后将没有任何武力，所有的武力全部集中在燕山，没有了武力的保护，他们就是一群养起来待宰的猪。而且，他们想的，太过美好，当大量的丹药出现之后，根本不会有那么高的暴利。”
<br/>    
<br/>        “就算他们真的积累了很多的灵石，燕山还会有更厉害的手段去收割这些灵石，比如说他们现在派来的大妖，最多只能学到一种丹药炼制手段而已，想要第二种，便需要花海量的灵石从燕山学习，而且，在他们拥有一定数量的灵石之后，燕山将会售出炼丹房名额，以拍卖的形式，只要他们互相竞争，那就是一盘散沙。”
<br/>    
<br/>        “想要脱离燕山，简直是痴人说梦，只要我们掌握了他们炼制丹药的种类，燕山一旦发力全力炼制那一种丹药，那一种丹药的价格必定暴跌，恢复到平价的机会。”
<br/>    
<br/>        “而且在有了身份文牒之后，燕山将会彻底掌控每一个修行者的一举一动，所有的一切，都将发生在我们的眼皮子底下。”
<br/>    
<br/>        “更何况看，燕山钱庄根本不收取任何保管费，他们为了自身灵石的安全，为了自己的安全，必定会将大部分的灵石储存在燕山，一旦有任何异动，这些灵石他们还拿的走吗？”
<br/>    
<br/>        “而且，你们的思维太过局限了。首先，你要知道，每个人都上交每年的十分之一上来，那么无论如何变化，燕山都绝对是最强的，而且还是面对一群失去了妖军的家伙，永远都会立于不败之地。”
<br/>    
<br/>        黑蟒听得额头冷汗直冒，山主，当真是神鬼不测之能，论治理天下，竟然也有如此功力！
<br/>    
<br/>        “山主，属下明白了，属下会尽快去做好！”
<br/>    
<br/>        陈明点点头，然后道：“最近让黑虎多做些准备，一旦彻底剥夺他们的妖军，他们麾下的大妖绝对只有一条路可以走，那就是投靠燕山，燕山的妖军，将会有一次巨大的飞升。另外，吞食大妖的事情，控制的严密一点，不能让他们发现，明白了吗？”
<br/>    
<br/>        黑蟒拱手道：“山主，放心吧，控制这件事的人，都是我麾下的儿郎。”
<br/>    
<br/>        陈明微微点头，“将黑虎和火燕叫来，我要安排他们其他的事情。”
<br/>    
<br/>        黑虎和火燕两人在下午来到陈明的身前，拱手问道：“山主，不知叫我们来有何事？”
<br/>    
<br/>        陈明道：“你们成立一个燕山镖局，将那些天赋不好的大妖全部收纳进去。”
<br/>    
<br/>        黑虎道：“属下明白了，属下会让手下的人去做！”
<br/>    
<br/>        陈明继续安排道：“另外，我既然说过会保护他们的安全，那么就在燕山境内建立一座城池，派遣两千血衣剑卫巡逻，其他的血衣剑卫，你们自行安排在燕山边境，严格巡查整个燕山的走动和出行。”
<br/>    
<br/>        “凡是要离开燕山出行的，都要严格审查，所有妖王，在目前不准离开燕山境内。不过想来，他们短时间之内也没想过离开燕山境内，毕竟他们目前的精力，都在建立炼丹房上面。”
<br/>    
<br/>        “所有得到炼丹房资格的，都要他们发下心魔毒誓，不能泄露燕山关于炼丹房的一丝一毫。”
<br/>    
<br/>        这样安排下来，应该不会出什么大事了。
<br/>    
<br/>        收缴上来的灵兵也越来越多，陈明看着众多的灵兵，将这些灵兵重新熔铸，便可以将燕山所有的妖军全部披甲了。而燕山钱庄，在这几日之内，就已经有了三千万的灵石储存进来。
<br/>    
<br/>        这些妖王和大妖们丝毫不担心燕山会侵吞他们的灵石，毕竟在他们的意识之中，他们看到的，只是燕山的冰山一角而已，燕山对于他们这点东西，压根不会动心。而且，有陈明的承诺作保，他们对于陈明，还是相当信任的，陈明在他们眼中，从未失信过，说杀谁就杀谁，绝对的守信！
<br/>    
<br/>        而实际上，陈明对于这些灵石，可是相当的动心，正在想办法怎么把这些灵石塞进自己的腰包。
<br/>    
<br/>        钱庄真是个好主意，还不用抢，直接就有灵石了！
<br/>    
<br/>        燕山的改变正在如火如荼的进行，陈明对于自己的地盘，可谓是彻底安心，应该不会出什么大乱子了，现在，是时候好好想想如何提升自己的实力了。
<br/>    
<br/>        陈明现在是可以直接将很多门功法直接修行到入道境巅峰，但是现在，陈明并没有直接修行，因为他压根不知道自己会得到什么样的大阵，若是修行的功法无法施展大阵，那不是把自己做宝搞？
<br/>    
<br/>        现在燕山收集到的功法，能够凝聚道种的倒是确实有不少，但是能够修行到大君领域的，可就那么几种而已。
<br/>    
<br/>        陈明可以预感到，自己将来，将会很长时间处于道种境，他想要超脱，需要三千道种。
<br/>    
<br/>        所以现在，最好就是找到一门足够复杂的阵法，要求不高，只要这个施展阵法需要的人越多越好！
<br/>    
<br/>        就在这时，黑蟒前来禀报道：“在半个月之后，大妖王金毛吼将会在紫金山举行一场拍卖会！金毛吼，已经给山主发了请帖，请山主前去参加。”
<br/>    
<br/>        陈明的眼中一亮，嗯，拍卖会，这不是主角发家致富的地方吗？
<br/>    
<br/>        对的，除了跳崖之外，主角还有一条发家致富之道，那就是拍卖会！
<br/>    
<br/>        没有宝贝不要紧，拍卖会有着除了主角别人都不认识的仙器！没有功法，不要紧，拍卖会有着除了主角谁都不能修行的仙级功法！没有术法，不要紧，拍卖会有着除了主角谁都看不懂的仙术！没有妹子，不要紧，拍卖会主持拍卖的小姐姐绝对人美歌甜！
