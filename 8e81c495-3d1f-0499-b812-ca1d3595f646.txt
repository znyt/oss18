　　太古蜈蚣，一直是来到了海底的一处海沟之中，才停了下来。
<br/>　　楚风眠一股灵力打出，这周围的海水，都是被隔绝了开来，奇阳师，海公主都是走了下来，走到旁边去打坐休息了。
<br/>　　奇阳师身上的寒毒，已经是被楚风眠以吞天秘术所化解，现在便是要恢复他身上的伤势。
<br/>　　海公主则是打坐开始修炼了起来，之前楚风眠杀了海剑宗的几名弟子，将那些海剑宗弟子的空戒，楚风眠都给了海公主。
<br/>　　这些空戒之中许多宝物，都是对海族血脉的武者妖神，有着极大的用处，海公主也是借助着这些宝物的力量，在提升着自己的实力。
<br/>　　楚风眠则是来到了一处巨石之上，将那境王布袋打开，从其中，将刚刚镇压抓到了海十四放了出来。
<br/>　　海十四的身上，已经是被楚风眠种下了层层禁制，哪怕是他的实力，已经是不亚于寻常的半步天人境强者，可依然是无法催动任何一点点的力量。
<br/>　　在这禁制的压迫之下，海十四被放了出来，也只能够是跪在楚风眠的面前，无妨反抗。
<br/>　　“有本事你就杀了我！”
<br/>　　海十四抬起头看向楚风眠，眼神之中露出仇恨的目光。
<br/>　　“杀了你？我抓你来，不过是想要问题一件事罢了，乖乖听话，我倒是可以饶你一命。”
<br/>　　楚风眠看着海十四，平静的开口道。
<br/>　　“说吧，你们海剑宗之中，是不是有着剑道图腾？”
<br/>　　“你想要剑道图腾？”
<br/>　　听到楚风眠的话，海十四的脸色猛然一变，随即张狂大笑道。
<br/>　　“小子，放弃吧，你不用想要从我的嘴里套出任何一句话来，反正今日我就算是死在你的手里，我的师尊也会为我报仇，倒时候你也要跟我陪葬！”
<br/>　　“不愿意说？”
<br/>　　楚风眠看着张狂大小的海十四，也不动怒，嘴角露出了几分诡异笑容道。
<br/>　　“你不愿意说，我有的是办法让你说！”
<br/>　　“诸天生死！为我掌控！”
<br/>　　楚风眠开口的一刻，突然一团灵力在楚风眠的手心之中凝聚着，这一团灵力，在楚风眠的手心之中，化为了一团漆黑的咒印。
<br/>　　看到这一团漆黑咒印的一刻，海十四的脸色都是猛然变化着，露出了几分惊恐之色。
<br/>　　“你想要做什么？”
<br/>　　“入！”
<br/>　　轰！
<br/>　　大手一挥，这一团漆黑咒印，轰然打入到了海十四的身躯之中，深深的刻入在了海十四的道心之中。
<br/>　　诸天生死印！
<br/>　　随着楚风眠实力的提升，现在他种下的这诸天生死印，也越发的强横，哪怕是半步天人境强者，都无法化解这诸天生死印。
<br/>　　随着诸天生死印打入到了海十四的身躯之中，海十四的脸色都是变的无比狰狞，大声的咆哮道。
<br/>　　“不要，不要控制我，我愿意为你做事，我愿意将一切都告诉你！”
<br/>　　这讨饶之声，楚风眠却是全然无视，他给过这海十四的机会，当时不懂得珍惜，现在也没有第二次的机会了。
<br/>　　很快，海十四挣扎便是逐渐的弱了下去，他的脸色便是变的平静了下来，看向楚风眠的目光之中，无比的恭敬。
<br/>　　“主人。”
<br/>　　“说吧。”
<br/>　　楚风眠大手一挥，将海十四身上的禁制全然去除，种下了这诸天生死印之后，短时间内，海十四根本不可能背叛楚风眠。
<br/>　　“是，我们海剑宗内，的确是有着一座剑道图腾，这剑道图腾，是我们海剑宗最为重要的镇宗之宝，平日里都是由诸位长老看守。”
<br/>　　“只有一少部分的弟子，才有资格前去参悟剑道图腾。”
<br/>　　“剑道图腾，位于海剑宗圣地的核心之中，如果是没有得到资格前去参悟，任何人都无法靠近。”
<br/>　　海十四一五一十的也是将楚风眠想要知道的消息，全部禀告了出来。
<br/>　　“看守的倒是严密。”
<br/>　　楚风眠听完海十四的话，心中喃喃自语道。
<br/>　　剑道图腾，人人都知道是剑道至宝，不过对剑道图腾的价值，每个人的认知却不一样。
<br/>　　对大部分的武者妖神来说，他们最多只是知道剑道图腾是剑道至宝，但是最多却当做是跟无上剑气一般的宝物。
<br/>　　可楚风眠知道，这剑道图腾的价值，比起无上剑气等等宝物，要高得多。
<br/>　　那剑道图腾之中隐藏极深的残篇，记载的可是剑道本源，可以说这十九座剑道图腾，极有可能都是剑道本源所化。
<br/>　　如果海剑宗对剑道图腾的防备不严的话，楚风眠倒是可以尝试直接出手去抢夺。
<br/>　　毕竟纵然海剑宗内有着真正的天人境强者，可这种的强者，平日里应该都在修行天人大道，不会参与繁琐事务。
<br/>　　只有楚风眠真正抢夺剑道图腾的时候，才会惊动到这样的人物。
<br/>　　而起楚风眠也不是要跟天人境强者交手为敌，仅仅只是逃走就行。
<br/>　　楚风眠的实力跟真正的天人境强者相比的确还相差不少，不过若是论起逃命，只怕是没有半步天人境强者可以跟楚风眠媲美。
<br/>　　如果是打定主意要逃，楚风眠还有着几分机会。
<br/>　　不过现在听到海十四的描述，楚风眠便是听的明白，海剑宗对剑道图腾的价值，知道的一清二楚。
<br/>　　防备这样严密的情况之下，楚风眠想要大摇大摆的打入海剑宗之中，夺走剑道图腾，在一走了之，根本是不可能的一件事。
<br/>　　而且要夺取剑道图腾，必须是要一次成功。
<br/>　　若是一次不成，必然是会引起海剑宗的惊觉。
<br/>　　下一次楚风眠再想要动手，难度就会在翻上无数倍，倒时候更是难如登天。
<br/>　　对楚风眠现在来说，除非是可以潜入到海剑宗之中，见到这剑道图腾，不然的话，想要一次成功，机会都不高。
<br/>　　“你有没有资格去参悟剑道图腾？”
<br/>　　楚风眠脑海之中思考这，他看向海十四，突然开口询问道。
<br/>　　“禀告主人，我曾经去参悟过剑道图腾过，不过我，也需要花费大量的贡献点，才可以换取去参悟剑道图腾的机会，这大量的贡献点，我都需要积攒许久才可以积攒到。”
