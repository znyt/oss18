　　三清、十二祖巫、妖族等洪荒生灵都是盘古造化众生所化的生灵。换句话说，他们都是盘古的后代。
<br/>　　但这些后代之中，只有盘古三清与十二祖巫是盘古元神与精血所化，也只有他们是盘古嫡系，盘古正宗可不是说说而已。那可是代表着无量气运与功德，与天地之间有着至善因果，在盘古给他们的气运福泽消耗之前，他们可是有着天道保佑的。
<br/>　　若要说盘古是混沌之中的大道之子，那么他们则为洪荒天地中的天道之子。
<br/>　　是故，欲在洪荒之中找一个身具大气运大功德的身份，就需要与盘古有关系。要是别人可能没有办法，但元释却可以。因为元释是因果魔神，掌控大道因果的因果之主。只要让盘古欠下元释因果，元释自信能依靠因果法则找到这个新身份。
<br/>　　但，欲使盘古欠下元释大因果，又何其之难。大因果意味着福运深厚的身份，小因果则意味着小人物。还有什么比盘古杀了自己欠下的因果大呢？没有！
<br/>　　于是，从最初，元释就打算放弃了混沌魔神真身，将自己的真灵藏于元始印之中，在最后的大战之中，最后用“强结因果术”强行接下这段因果，只要元释没有真正死亡，这段因果就会一直存在，只因元释是因果之主。
<br/>　　因此，在最后的大战之中，元释才会全力以赴，以最强姿态迎战盘古，战到狂，战到死，因为只有如此，元释的混沌魔神真身才会死得有意义。
<br/>　　最后，也证明了元释的选择是正确的。因为在最后的一战之中，元释不仅观看了盘古看天的过程，也在战斗之中领悟了更多，因果大道距离混元十二重天只差一步，只要消化此处战斗后，不仅对于道的领悟提高了，而且还可以参悟出更多神通。
<br/>　　只因开天的过程中大道法则实质化的显现在众人眼前，要是没有领悟才怪。
<br/>　　而现如今，元释果然如预料之中的那般，依靠因果法则在洪荒新天地之中，寻到一具有大气运大功德的身份，只是没想到这身份会是大名鼎鼎的元始天尊。
<br/>　　“元始，元始，一元复始，这可不是一般人能叫的。不过，我喜欢。元释、元始，鸿蒙元始印，是冥冥中自有注定，还是机缘巧合呢？命运，果然是最难预测。”元释适应着自己的新身份，自言自语道。
<br/>　　“管他的，既然我已有了新身份，那么，过去的种种，也就烟消云散了。如今，洪荒之中，吾为——元始！”
<br/>　　说完这句话，元始心中一阵轻松，仿佛身上的枷锁被打开了。果然，具有新身份，就是不同。
<br/>　　“天道么？”元始猜测到，根据这等情况，元始认为这是天道承认了自己的身份。
<br/>　　“既然如今已经苏醒了，那就应该加快速度炼化这道元神，之后修炼化形而出，只是一道元神，还真是不习惯啊！而且，还是如此弱的元神，好久没有体会到此等感觉了。还真是怀念啊！”元始感应到元神的弱小后，感慨道。
<br/>　　不予理会，元始将全部心思投入到灵魂之火炼化元神之上，毕竟，仅仅夺舍，有点不自在的感觉，只有将这些元神力量全部转化为自己的力量，才是最真实的，以后才不会留有后患。
<br/>　　有意识的控制灵魂之火就是不一样，这炼化速度比之前快了不知多少倍，周围灵气流动更快了，让元始炼化速度加快不少，炼化的同时，也使元神变得越来越强。
<br/>　　炼化的同时，元始也开始接受道盘古传承遗泽，盘古传承一共分成三部分，一部分是盘古的修炼心得，这部分三清与十二祖巫估计都有，只是分多与少而已。
<br/>　　第二部分为开天印记。但开天印记也有不同，盘古开天劈地，划破混沌开劈洪荒，大道隐，天道出。天道五十，其用四九，遁去其一，万物一线生机。混沌生无极，无极生太极。太极三才变，两仪四象生。属于元始的第二部分传承是其中的天地人三才之道。
<br/>　　天地人三才，天有三宝：日、月、星；地有三宝：水、火、凤；人有三宝：精、气、神。前世小说之中元始的道也是三才之道，不过，最后他是立教，走功德成圣的路子。
<br/>　　不过，如今，一切都变了，元始才不会去走那最弱的功德成圣之路。不过，圣人也很弱啊，元始作为混沌魔神刚苏醒就是混元大罗金仙了。
<br/>　　这部分开天印记，估计也只有盘古三清与十二祖巫有，依据元始前世所知，十二祖巫得到的应该是空间、时间、金木水火土五行、风、雷、电、雨、天气；老子得到的应该是太极阴阳两仪之道；通天得到的应该是截天之道，取自“万物一线生机”。
<br/>　　这开天传承也不完整，不过，元始观看了盘古开天的过程，与之相互印证，还是能得到很多的。这可不是老子、通天所能比的。
<br/>　　第三部分传承为盘古有力所见及所有的炼器知识。里面有盘古两大混沌至宝的大道禁制。怪不得前世所知中，三清中，老子是炼丹高手，元始是炼器高手，通天是阵道高手。
<br/>　　元始估计老子就是得到盘古丹道传承，通天得到的是阵道传承，截教门人，大多都是阵道高手，封神之中，各种大阵都出自截教。至于十二祖巫，元始估计他们得到的是关于盘古炼体方面的，盘古的神通盘古之身。
<br/>　　不知不觉，在接受传承之中，元释炼化元神的速度更快了，连带着吸收灵气的速度也变快了。顺带着三道清气中的另外两道吸收灵气也变得更快了。
<br/>　　不知过了多久，元始炼化了所有的元神力量，使之全部变为自己的，同时，也接受了全部的传承。元始感受了一番，自己现在的修为正处于第一步的炼虚合道。还是很弱，也就达到前世自己渡劫成仙的地步，不过，这样也好，一步一步的修炼上去，体验一下每一个境界的奥妙，根基更为踏实。
<br/>　　第一步蜕凡证仙，分别为炼精化气，练气化精，炼神返虚，炼虚合道。之后蜕凡成仙，成为第二步强者。
<br/>　　元始现在要做的就是慢慢壮大元神，从分散的元神清气壮大到凝聚成实质，之后渡劫成先天道体。
