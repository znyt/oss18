
<br/>　　<br/>　　    “停，停，停······”东干急了，破口大骂道：“你个疯女人快住手，快停下来”<br/>　　<br/>　　    茅小草莞尔一笑，眯眯着眼睛说道：“你让我停就停，那我多没面子，早干嘛来的啊？”<br/>　　<br/>　　    东干说道：“你疯了是不是？你把人引来怎么办，我好不了你就能好得了么？你手里那盏引魂灯不知道多少人想要呢，你就不怕被人给抢去么？”<br/>　　<br/>　　    “那也得能抢的走算啊”茅小草笑眯眯的神情顿时一收，然后猛地朝着那盏灯火深吸一口气“噗”的一下就吹了过去，顿时灯火瞬间爆涨，蹿起来都能有三四十公分高了，而此时小草的脸上也是布满了汗珠。<br/>　　<br/>　　    但与此同时，五具被封住三魂七魄的僵尸，在一阵猛烈的抖动下，突然就静止不动了，从他们的头顶瞬间飘出一道魂魄出来。<br/>　　<br/>　　    “收！”茅小草随即朝着几道亡魂一招手，那些魂魄顿时全都朝着她这边飞了过来，东干和文华还有秋生愣了愣，一脸颓然的说道：“完了，完了，花费百年时间孕养出来的这些尸体全完了”<br/>　　<br/>　　    他们本来是想着要把尸体赶回湘西的苗寨，但没想到刚刚启程就碰上了追过来的茅小草，要是常人倒也好说了，但偏偏小草手里那盏引魂灯轻而易举的就能将五具僵尸体内的魂魄给引出来收走了。<br/>　　<br/>　　    这真是耗子嘬猫逼，恰好踢到铁板上，点太背了。<br/>　　<br/>　　    五道亡魂被茅小草收入到了引魂灯里，东干三人一脸的颓败和懊恼，紧接着就是呲着牙一脸的憎恨表情。<br/>　　<br/>　　    一百年的时间，才不过养好五具尸体，就这么平白无故的废掉了，这等于是让他们把百年的心血全都给耗费了，本来你的银行账户里存了几百万，可忽然有天你发现被偷了，你说你窝火不的？<br/>　　<br/>　　    东干磨着牙说道：“我们好过不了，你也好不了，特别是你手中的那盏引魂灯·······”<br/>　　<br/>　　    茅小草没有吭声，算是默认了对方的话，有一个词叫做奇货可居，怀壁其罪，她此时手里握着的可能就是个无数人都想要的奇货，关键是这货只此一家别无分号，天下间就只有这一盏引魂灯。<br/>　　<br/>　　    第一次，在渭河边上，茅小草拿出引魂灯是因为她知道王惊蛰不会贪婪她手里的东西，这一次她是逼不得已要为秦翠儿和她夫君合葬，不把尸体里的亡魂逼出来就藏不了，她许下的诺言自然要说到做到。<br/>　　<br/>　　    茅小草弯腰把地上的那双绣花鞋捡了起来放回到身上，手中的引魂灯刚要也揣回去，就听到野地外面有几辆车疾驰而来，然后突兀的刹车声响了一片。<br/>　　<br/>　　    “咣当”<br/>　　<br/>　　    “咣当······”车门关上的动静响了好几下，就有人快速的朝着这边走来。<br/>　　<br/>　　    东干三人幸灾乐祸的看着她，他们都没想到人会被引来的这么快，前后不过十来分钟的时间，就有人赶了过来，越往后人来的肯定也越多。<br/>　　<br/>　　    当先一人来的，也是穿了一身道袍，脑后扎着长发被一根木簪别住了，肩膀上一把木剑的剑鞘露了出来，十分古朴的打扮，正统的道士装，比王惊蛰的还要正统。<br/>　　<br/>　　    如今世上，只有天师教，龙虎山和茅山出来的道士会如此装扮，他们始终延续着千年前的装束从未改变过。<br/>　　<br/>　　    那人过来之后，零零散散的也有几人跟了过来，打扮各有不同，有穿着便衣的也有一身道袍的。<br/>　　<br/>　　    但这些人过来后，眼神无一列外的都落在了茅小草手中那盏引魂灯上，特别是当先过来的那个道士。<br/>　　<br/>　　    “无量寿佛······”对方伸出一只手，声音有些颤抖的说道：“这不是我正一教的镇教引魂灯么？自从我们张道陵祖师仙逝飞升以后，此物就不知去向了，正一教内一直都以为这引魂灯随同天师一起被埋入了地下，现在看来却是不对的，这位女施主我想请问问看，你是从何得到我正一镇教引魂灯的”<br/>　　<br/>　　    这道士一说完，旁边的人都给了他一副不要脸的表情，这人简直了，一出场就来了个先入为主，直接就把这引魂灯给定死在了自己嘴里，说是正一教的，你还让别人这么插嘴？<br/>　　<br/>　　    茅小草幽幽的说道：“我捡的·······”<br/>　　<br/>　　    茅小草知道，不管自己怎么解释都没有用，她是怀壁其罪了，不管她找出一百种借口还是一百零一种，对方最后都会用一句这是我们正一教天师张道陵的话来给她挡回去。<br/>　　<br/>　　    事实也确实如此，世间人都知道这引魂灯是张道陵天师的随身之物，一直跟随他降妖伏魔来着，至于后来怎么到了梵净山天道峰，茅小草也不知道，这是她下梵净山时她师傅交给她的，说是只有天道峰的每一代圣女才有资格持有。<br/>　　<br/>　　    “贫道秋成子，正一教弟子”<br/>　　<br/>　　    茅小草嗯了一声，不瘟不火的问道：“然后呢？”<br/>　　<br/>　　    “女施主手中那引魂灯，为我正一教之物，历代掌门都在为其苦苦而寻，天道可见啊，总算是被我给寻到了，还请女施主能原物奉还，我们正一上下必将感激不尽”<br/>　　<br/>　　    “我捡来的······”茅小草幽幽的说道：“又不是我偷的，我捡来的就是我的了”<br/>　　<br/>　　    秋成子顿时一愣，有些恼怒的说道：“你这是什么道理？”<br/>　　<br/>　　    茅小草拢了下头发，淡淡的说道：“谁和你讲道理了么？别天真了，道理是什么，多少钱一斤啊，你有多少我都要了”<br/>　　<br/>　　    “你······”秋成子的嘴皮子可能不太利索，指着茅小草半天都没把下半段话说出来。<br/>　　<br/>　　    茅小草说道：“有能耐，你就从这再捡回去，那我也算你的”<br/>　　<br/>　　    “好，好，好”秋成子连连点头，说道：“如此顽固不化，如此胡搅蛮缠，说不得我要替你家师长好好教育你一通了”<br/>　　<br/>　　    旁边的人都没有出声，默默的看着两人在那掰扯，茅小草手里那盏引魂灯可不是说你想要就是你的了，这么多人在那看着呢，那得说是谁抢来就是谁的了。
