<br/>    陈明踏过九宫，挥手又是一座丹炉落下，“此处，阴寒最盛！”
<br/>    
<br/>        两座丹炉落下，陈明闭上双眼，而后再睁开双眼，作为一个地级阵法师，一个炼丹大师，这两种职业都站在了这个大陆顶端的存在，如果他做不到，也没有多少人做的到了。
<br/>    
<br/>        一种种药材出现在陈明的眼前，法力涌出，这些药材按照陈明的处理方式，在空中变化着形态，有的切碎，有的磨粉。
<br/>    
<br/>        天人合一的状态下，这一点还是不难的。
<br/>    
<br/>        各种药材准备完毕，陈明看向两个丹炉，喃喃自语道：“不对，这两个丹炉并不适合阴阳和合的炼制之法，还需要重新炼制一番。”
<br/>    
<br/>        紫霞仙子已经看呆了，这到底是个什么怪物，阵法他会，炼丹他会，算卦他也会，连炼器都会不成？
<br/>    
<br/>        等于说，除了打架，他什么都会！
<br/>    
<br/>        然后紫霞仙子就看到陈明直接在一个丹炉上拍了三巴掌，然后这个丹炉就已经浑然一变，变得赤红无比，陈明走到另外一个丹炉旁，又是四巴掌拍下，整个丹炉变成深蓝色，寒气森森。
<br/>    
<br/>        而现在终于到了最后一步，要开始炼丹了。
<br/>    
<br/>        陈明开始将材料一一放进去，按照他对于药性的完全精通把控着。
<br/>    
<br/>        一件件药材被丢入其中，陈明凭借着对于炼丹和阵法，炼器的造诣，几乎一切的过程都是完美的。
<br/>    
<br/>        在将一滴还魂水分成两半，两个丹炉各自半滴，待得两滴还魂水和药材完全融合，陈明x运转法力，两滴炼制完毕的丹液从两个丹炉之中飞出，在空中融合为一处，一阵充斥着整个头骨的丹雾出现，陈明伸手，一颗红蓝交织的丹药落入手中。
<br/>    
<br/>        紫霞仙子看向那一颗珠圆玉润，美不胜收的丹药，还真的被他炼成了！
<br/>    
<br/>        “你快试试看，到底能够提升多少药效！”
<br/>    
<br/>        陈明直接丢入口中，丹药入口，比之前的效果猛了数倍不止，再次睁开眼，仔细开始回味，然后道：“还魂水的效果提升了五倍。”
<br/>    
<br/>        紫霞仙子听到陈明的话，已经是惊得说不出话来了，这就是炼丹师？
<br/>    
<br/>        简直恐怖如斯啊！
<br/>    
<br/>        紫霞仙子承认，她之前确实看不起炼丹师，但是看过了陈明如同艺术般的炼丹手法之后，他对于炼丹师已经彻底改观了，根据群星仙门的猜测，陈明背后的宗门，可是有着数万炼丹师，紫霞仙子小心地问道：“不知道阁下，在你们宗门炼丹的造诣排行第几？”
<br/>    
<br/>        卓青瑶直接替陈明回答了，“第一啊，怎么了？”
<br/>    
<br/>        紫霞仙子这才觉得正常，在几万炼丹师之中排行第一，这般炼丹的造诣，恐怕只有药王谷的谷主才能比得上了。也就是说，整个大陆，唯有两个人。
<br/>    
<br/>        也只有这样的炼丹师，才能炼制出这样的丹药了。
<br/>    
<br/>        药王谷的谷主，虽然修为在大君，但是实力却不如其他大君，但是在大君之中，地位可谓是高崇无比，而眼前这位，居然可以跟药王谷的谷主比肩，这是一位无冕之君！
<br/>    
<br/>        或许，在他的帮助下，自己有机会突破到大君。
<br/>    
<br/>        紫霞仙子道：“既然你现在已经炼制成功了，那你准备怎么做？如果你愿意提供给我这种丹药，我可以倾尽全力听从你的安排。”
<br/>    
<br/>        紫霞仙子试探着问道，她希望可以在陈明这里得到部分的帮助，当然，她可以凭借她高强的实力，保障陈明的安全。
<br/>    
<br/>        毕竟在紫霞仙子的眼中，陈明的实力还是只是入道境巅峰而已，而且，一个精通炼丹，阵法，算卦，炼器的家伙，怎么可能很能打？
<br/>    
<br/>        陈明却是没有半分炼制成功的喜色，而且是眉头紧皱，“还不行。”
<br/>    
<br/>        紫霞仙子疑惑道：“怎么了，你不是已经成功炼制出来了？”
<br/>    
<br/>        陈明道：“太慢了，我炼制一次足足需要三个时辰，就算我每次可以炼制十颗，这也太慢了，根本无法满足所有人的需求，容我再参悟一卷仙级功法，然后再试试看。”
<br/>    
<br/>        紫霞仙子看着陈明，觉得在看一个傻逼，不是三个时辰炼制出来，这还太慢了？你是个什么怪物啊？
<br/>    
<br/>        而且参悟一卷仙级功法，需要的时间很长好吗？紫霞仙子修行的就是仙级功法，如何不知道仙级功法的难度。
<br/>    
<br/>        陈明也是心中一横，闭上双眼，直接将七万师德堆在了《太素灵棺》之上，这太素灵棺生死交融，对于这种相斥性的丹药炼制有着很大的帮助。
<br/>    
<br/>        陈明再睁眼，紫霞仙子只看到空中不断的变化，陈明的炼制速度，足足比之前快了数十倍不止！
<br/>    
<br/>        一刻钟之后，陈明伸手进入丹雾之中，已经是将一颗一模一样的丹药取了出来，“好了，现在就可以了，这个速度应该差不多了！”
<br/>    
<br/>        紫霞仙子的双眼足足盯着陈明炼丹一刻钟，眼睛都没眨一下，刚刚那眼花缭乱的手法，真的是一个入道境可以做出来的吗？
<br/>    
<br/>        而且，他是一直处于天人合一的状态，紫霞仙子深深地看了陈明一眼，确定陈明是一个在修行路上跑偏了的家伙，压根没有任何战斗手段，然后所有的辅助职业全部精通。
<br/>    
<br/>        陈明道：“好了，现在，我们可以将我炼制成功的消息通知整个九幽寒泉之中的人了，定价，十滴还魂水换取四枚丹药，这丹药的名字，嗯，就叫红蓝丸！怎么样，我取名字的水平是不是很高？”
<br/>    
<br/>        卓青瑶扶额。
<br/>    
<br/>        紫霞仙子道：“要不，叫阴阳和合丹？”
<br/>    
<br/>        陈明认真道：“虽然你这个名字一点都不形象，但是看在你是女人，女人优先的份上，我就勉为其难叫做阴阳和合丹了。”
<br/>    
<br/>        紫霞仙子道：“好吧，既然这样，我去召集我们群星仙门进入此地的人王，让他们去通秉这个消息，不过你准备怎么给予我们酬劳？”
<br/>    
<br/>        “叮，你触发了剧情任务，女王的崛起，请协助未来的女王，成为大君之位，奖励：10w（万）灵识，1w声望。”
<br/>    
<br/>        居然还来任务了，看来紫霞仙子在后面的剧情之中，似乎还是个主线人物。
<br/>    
<br/>        陈明道：“十滴还魂水，算上炼制失败的情况，我实际上也赚不了多少，这样吧，每炼制十滴还魂水的丹药，我给你一枚阴阳和合丹，至于你要怎么给你们宗门的人分配，完全在你。”
