<br/>    三个月后。
<br/>    
<br/>        鬼门星议政大殿外城。
<br/>    
<br/>        黑夜降临，天边卷来数朵乌云，狂风掀起无面者的大旗，在风中狂卷，啪嗒啪嗒，雨水落下，秋天的雨水总是有些寒冷。天空之中，有着四人骑着仙兽在大殿之外的外城城门处落下。
<br/>    
<br/>        杨珲，步留痕，段首，吕清玄一一下马，来到地面上，踏着雨水沉积的泥泞路面前进。
<br/>    
<br/>        这里是无面者下达命令的城，这里是三头龙星宿的主城，在这里，除了无面者阁下，没有任何人可以飞行，飞行的，都会被视作僭越。
<br/>    
<br/>        无冕者议会向来血腥黑暗，这里自然也不例外，僭越的下场，便是被压上斩仙台砍上一刀。
<br/>    
<br/>        目前为止，还没有人能够从这一刀下活下来，毕竟动手的是段首。
<br/>    
<br/>        四人披着黑袍，朝着内城快步赶去，无面者阁下，再次召见了他们。他们知道陈明的身份，故而没有任何拖延。
<br/>    
<br/>        数千浑身黑衣的执刀人半跪在大殿之外的广场上，雨水从他们的身躯上划过，只有着雨水落下的声音，除此之外，就没有了其他的声音。
<br/>    
<br/>        四人跨步走过执刀人们，走进大殿之中，四人互相看了一眼，单膝跪地，闭上双眼齐声道：“尊敬的无面者阁下，您的执刀人正在呼唤您.......”
<br/>    
<br/>        陈明的虚影在大殿之中唯一的座位上落下，看向四人，“起来吧。”
<br/>    
<br/>        四人睁开眼起身，陈明淡淡道：“你们应该知道我是谁吧？”
<br/>    
<br/>        杨珲想要开口说，就算是知道了也没什么，我们依旧尊敬您。
<br/>    
<br/>        但是陈明怎么可能会给他们这个机会？现在是主弱臣强的局面啊，陈明虽说杀个步留痕还是很轻松的，但是遇到三位金仙，还是得歇菜啊！
<br/>    
<br/>        拜托，这是金仙，越阶也是要遵循基本法的啊！
<br/>    
<br/>        以后有空了一定要写一本越阶战斗基本法，大概和演绎推理论是同一级别的史诗制作。
<br/>    
<br/>        所以，一定要让一切都是由自己主导，陈明默默地开启了，凡是能够增加自己气势的光环全开，好了，现在是开始吹逼的时候
<br/>    
<br/>        了，“我自无尽轮回之中醒来，看到了这个世界的终结。”
<br/>    
<br/>        四人都是看向陈明，关于陈明是轮回仙的说法，早就已经是沸沸扬扬了，对于陈明的话，他们并没有觉得有什么问题。
<br/>    
<br/>        毕竟，同阶之内，除了最后出现的那个神秘的凡人，没有任何人可堪与陈明一战。
<br/>    
<br/>        除了轮回仙，还能是什么？
<br/>    
<br/>        他们觉得陈明说的没毛病。
<br/>    
<br/>        陈明很是满意现在的效果，继续道：“我，二十三岁醒来，开始修行，三年，斩杀大君，五年，斩杀仙人，十七年，地仙可谓无敌，如今，嘿嘿嘿。”
<br/>    
<br/>        陈明并没有说出来，“你们知道我想要什么吗？”
<br/>    
<br/>        陈明还是没有给他们开口的机会，直接道：“以我的修为，以及我所看到的，我想要的，是整个世界！”
<br/>    
<br/>        陈明站起来，双手如同在环抱天地，狂吼出这一句。
<br/>    
<br/>        主弱臣强，导致臣不满想要弑主的主要原因，就是君主没有野心，而臣子，想要做更多的事。大概就是明明可以得到的东西，君主不想去拿，但是臣子知道他们能拿到，所以干脆把君主一脚踹了，自己来做！
<br/>    
<br/>        就是这么个思路。
<br/>    
<br/>        既然这样的话，现在这个局面，要解决很简单啊，吹牛逼就完了，画一个大饼，让这些强横的属下有些盼头，心中有了盼头，自然是就没心思去弑主了。
<br/>    
<br/>        这个效果大概可以解释为，如果说一群醉汉在酒楼喝酒，说不定就会闹事，但是如同你让一个衣衫半露的火辣女人跳舞的话，那就没这茬了，因为他们正忙着看美女。
<br/>    
<br/>        总之，就是要给麾下的人找点事做，像是什么没事造反的，都是闲着没事做的。
<br/>    
<br/>        就好像陈明，他一般又不需要修行，所以满脑子里都是怎么造反。
<br/>    
<br/>        就是闲的！
<br/>    
<br/>        当然，四大仙国的通缉令也是一部分原因。
<br/>    
<br/>        所以现在，就在给这四个家伙画大饼，反正自己修为提升绝对比他们快，等到他们反应过来，要么已经打不过自己了，要么就是已经绑死在自己的战车上了。
<br/>    
<br/>        陈明继续道
<br/>    
<br/>        ：“将来，你们要做的，不是无面者，不是无免者，是无眠者！”
<br/>    
<br/>        无眠者啊，可是这个世界封王拜相的最高荣耀！至于说让他们当仙国帝主，至高无冕什么的，这有点过了啊，说出去他们自己都不信。
<br/>    
<br/>        陈明认真道：“所以，我们先定一个小目标，成为一位无眠者。”
<br/>    
<br/>        这个时候，这四个应该动心了啊，陈明终于是看到了他们的动作，步留痕道：“无面者阁下，你从天庭的手里救下了我和我女儿的命，我步留痕，这条命就是无面者阁下的！”
<br/>    
<br/>        杨珲道：“无面者阁下将我从瑶池的大牢之中救出来，而且还让我使用天狼弓，我杨珲，任凭阁下驱使！”
<br/>    
<br/>        段首道：“我曾经与无面者阁下为敌，但是无面者阁下却是不计前嫌，愿意传授我大道，我段首和无面者阁下，虽无师徒之名，但是却是有着师徒情分，可惜我的天赋比不上几位宗主，就不厚颜无耻请求拜在无面者阁下的门下了！此生，赴汤蹈火，在所不辞！”
<br/>    
<br/>        陈明看着这三个家伙，等于说这三个家伙实际上都是忠臣，自己刚刚，是在欺负老实人！
<br/>    
<br/>        不是，你们也忒老实了啊！
<br/>    
<br/>        那我刚刚这番话到底是说出来干嘛的？
<br/>    
<br/>        自己一个人瞎折腾？
<br/>    
<br/>        就在这时，吕清玄道：“对！无面者阁下说的对，我等了无面者阁下三百年，就是要做一番大事的！我愿意作为前锋，将一切阻碍无面者阁下成为无眠者的敌人斩杀殆尽！”
<br/>    
<br/>        他的眼中露出一丝期待，显然对陈明所说过的那些话很是向往。
<br/>    
<br/>        陈明这才微微满意，还好还好，还是有用处的！
<br/>    
<br/>        至少还是忽悠到了一个嘛。
<br/>    
<br/>        吕清玄继续道：“事不宜迟，我这就去训练麾下的刺客儿郎，好在将来无面者阁下挑战至高无冕的时候贡献一份力量！”
<br/>    
<br/>        陈明：.......
<br/>    
<br/>        算了，我什么都不想说。
<br/>    
<br/>        “段首，明天来燕山一趟。”
<br/>    
<br/>        段首露出一抹喜色，恭敬道：“属下遵命，属下，多谢无面者阁下传授大道！”
