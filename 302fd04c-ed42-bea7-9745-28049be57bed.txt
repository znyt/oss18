　　眼前这五具尸体，正是虚空五霸，这虚空五霸的名声，李长老自然也知晓，以他的实力，想要对付这五人联手，都不一定可以稳稳斩杀。
<br/>　　这虚空五霸，在虚空海中纵横，欺软怕硬，不去得罪一些生死境高阶的强者。
<br/>　　一般生死境武者，都还不是他们的对手，想要斩杀他们，都有着极大的危险。
<br/>　　可以说是极为难缠的人物，他也想不到，今日居然有人杀了他们。
<br/>　　李长老仔细的打量了楚风眠跟白萱二人一眼，眼神中微微闪过一丝惊讶。
<br/>　　两个神力境九重的武者，居然足以斩杀虚空五霸，足以让他惊讶了。
<br/>　　楚风眠没有理会这李长老的探察，他的实力，还不是这李长老可以探察的出的。
<br/>　　楚风眠平静的开口道。
<br/>　　“不错，正是虚空五霸，虚空五霸的悬赏，是天商会下的吧，这五人的尸体在这里，检查检查吧。”
<br/>　　“好。”
<br/>　　李长老一股灵力，打入到这虚空五霸的身躯去，顿时这五道身躯上，都是显现出了异样的光芒。
<br/>　　一股股灵力涌现出来，这其中的力量，正是来自于虚空五霸的力量。
<br/>　　对像是楚风眠这个级别的武者来说，相貌年纪，都已经做不得数了，随时都可以改变。
<br/>　　唯一不可能改变的，只有自己修行的功法，力量。
<br/>　　“不错，正是虚空五霸。”
<br/>　　李长老收回了灵力，点了点头道，他手心一动，在手心中便是多出了一枚空戒来。
<br/>　　在空戒中，取出了一个玉盒，递了过来。
<br/>　　“这虚空五霸的悬赏，是龙族下的，奖励一共的五枚龙血圣王丹。”
<br/>　　楚风眠接过玉盒打开，其中便是一共五枚龙眼大小的红色丹药，其中每一枚中，都蕴含有着极为精粹的血气，在其中，更是蕴含有着一股恐怖的龙威
<br/>　　这龙威，不是寻常真龙的，而是在真龙一族中，实力极高的龙王龙威。
<br/>　　龙血圣王丹，便是以真龙之血，在加上一滴龙王之血，炼制而成的丹药，足以比起寻常的龙血圣丹，效果要强数十倍。
<br/>　　“这悬赏的龙族，倒是大手笔，五枚龙血圣王丹，足以增强我现在的身躯。”
<br/>　　“我的血脉，跟远古战龙的血脉一样，但是身体却是远远不如，远古战龙的强大，不止是血脉，就算是我拥有远古战龙一成的血脉，真正实力却远远达不到远古战龙一成的实力！”
<br/>　　楚风眠心中暗自想到，他修行的是战龙决，想要强化他的身体，这以龙血炼制而出的龙血圣王丹，的确是很好的选择。
<br/>　　“若是有着龙血圣帝丹就更好了，若是以龙帝的血，炼制的龙血圣帝丹，五枚已经足以将我的身体提升到极限了。”
<br/>　　只是这也只能想想。
<br/>　　楚风眠自嘲的笑了笑。
<br/>　　龙血圣帝丹，可是天级丹药，龙帝，可是统御群龙的存在，想要以他的血，来炼制出龙血圣帝丹，那有那么容易。
<br/>　　不过除了龙血圣帝丹，还有着丹药，可以凝练增强武者的身体，楚风眠倒是可以再去寻找其他丹药来，提升他的身体。
<br/>　　楚风眠一边想着，一边便是将这五枚龙血圣王丹，准备收入空戒之中。
<br/>　　“等等！这龙血圣王丹，我要了。”
<br/>　　一道身影，来到任务殿前，突然看向了楚风眠手中的龙血圣王丹，直接抬脚走到了楚风眠的面前来。
<br/>　　这道身影，是一名看起来二十岁出头的男子，一身灵力，浑厚无比，也是神力境九重的境界。
<br/>　　单单是从身上的气息来看，这男子的实力，应该是足以跟落长恨在伯仲之间。
<br/>　　“就是你们两个杀了虚空五霸，领取了这龙血圣王丹？”
<br/>　　这男子看向楚风眠，眼神中充满着不屑，语气强硬无比，如同命令一般道。
<br/>　　“这龙血圣王丹，我要了，开个价吧。”
<br/>　　“恩？”
<br/>　　楚风眠瞥了一眼这男子，冷哼一声道。
<br/>　　“你买不起。”
<br/>　　这男子要是好好交谈，也许楚风眠还会跟他聊两句，这种上来就以一种命令姿态嚣张的，楚风眠也不会给他什么好脸色。
<br/>　　“买不起？笑话！区区几枚龙血圣王丹，我欧阳烈还不放在眼中。”
<br/>　　这名为欧阳烈的男子，冷笑一声开口道。
<br/>　　这一枚龙血圣王丹，外面也就卖一块王品灵石的价格，给你五块王品灵石，将这龙血圣王丹拿过来吧。”
<br/>　　这欧阳烈冷哼一声，便是从空戒之中，一下子拿出了五块王品灵石来，对着楚风眠直接扔了过来。
<br/>　　“欧阳烈？”
<br/>　　听到这个名字，白萱的脸色也是微微有些变化，似乎是明白过来眼前此人是谁，悄然之间对着楚风眠开口道。
<br/>　　“这欧阳烈，是在虚空海中闯荡的独立者之一，实力极高，一身实力不在落长恨之下，他更是加入了虚空海一大独立势力之一，天盟，在其中地位崇高，不好对付。”
<br/>　　“是吗？不好对付？”
<br/>　　楚风眠冷笑连连，面对这扔过来的五块王品灵石，楚风眠负手而立，直接躲开。

<br/>　　楚风眠冷笑连连道。
<br/>　　“想要买这龙血圣王丹，可以，一枚龙血圣王丹一百枚皇品灵石，一共五百枚，拿出来，我就将这龙血圣王丹卖给你如何？”

<br/>　　一枚龙血圣王丹的价格，也就在一枚王品灵石左右，而一枚皇品灵石，足以换一百块王品灵石。
<br/>　　楚风眠的话，简直便是将这龙血圣王丹的价格，翻了一万倍。
<br/>　　“什么？你说什么？”
<br/>　　欧阳烈听到楚风眠的话，先是一愣，随后他的脸上，布满了杀意。
