　　楚风眠在杀了这四人之后，大手一挥，他们身上的空戒，灵器，全部都被楚风眠收取。
<br/>　　随后楚风眠毫不停留，继续向着山谷深处飞速走去。
<br/>　　如今在身后，可不单单是有着这四人的追逐，无数武者也都在后面，很快都会追赶上来。
<br/>　　这云羽天宫的宝藏，楚风眠可不想要让其他人得到，他必须要第一个通过这峡谷。
<br/>　　“这剑苍穹的身上，倒是有着不少的好东西！”
<br/>　　楚风眠一边催动身法，继续向着山谷深处跑去，一边也是将四人空戒中的所有宝物，都给拿了出来。
<br/>　　这些人如今在归墟天中已经是得到了不少的宝物，如今这些宝物，全部便是落入到了楚风眠的手中。
<br/>　　单单灵器，便是就有着一件天级灵器，还有无数的地级灵器。
<br/>　　楚风眠心神一动，唤出黑玄剑阵，这些灵器全部都是被楚风眠融入到了黑玄剑阵中，至于剩下的东西，全部都被楚风眠收入到了空戒之中去。
<br/>　　而就在楚风眠离开没有一分钟的时间，几道身影便是赶了过来，正是盘纯为首的圣王宗弟子，跟小魔主为首的九魔城弟子。
<br/>　　“剑苍穹死了！”
<br/>　　盘纯打量了周围一眼，闻到其中的血腥味，便是一下子明白过来，到底是发生了什么。
<br/>　　“剑苍穹这个废物，还想要单独杀楚风眠，如今倒是自己死了，不过无妨，他死了，也少一个人跟我们来争抢这云羽天宫的宝藏！”
<br/>　　小魔主冰冷的开口道。
<br/>　　“走，这云羽天宫的宝藏，绝不能落入到楚风眠的手中去！”
<br/>　　他们的遁光，也是丝毫没有停留，便是直接追了上去。
<br/>　　更多的武者，也是都来到了这山谷之中，向着其中急速追逐而去。
<br/>　　云羽天宫开启的消息，已经是传遍了整个归墟天。
<br/>　　无数武者，纷纷前来，如今几乎是归墟天中一大半的武者，都是进入到了这诡异的空间之中来。
<br/>　　“强者越来越多了。”
<br/>　　楚风眠也足以感觉得到，越来越多的强者，都已经是来到了这山谷之中。
<br/>　　他如今虽然杀了剑苍穹，但是眼下，足以跟剑苍穹抗衡的武者，也并非没有。
<br/>　　起码那九魔城的小魔主，以他的实力，只怕更是在剑苍穹之上。
<br/>　　如今楚风眠对着七大宗门弟子中，最为忌惮的人，便是他。
<br/>　　像是刚刚那种情况，楚风眠都没有自信以同样的办法来，对付小魔主。
<br/>　　“吼！”
<br/>　　就在楚风眠的思考之际，就在周围山谷崖壁之上，突然一声兽吼，猛然响彻起来。
<br/>　　这一声兽吼，让楚风眠都是汗毛林立起来。
<br/>　　在这山谷之中，居然还有着生灵的存在？
<br/>　　楚风眠一眼望过去，在这山谷之中，一头巨大无比的身影，突然缓缓的站了起来，乃是一头猛虎。
<br/>　　一头至少达到生死境巅峰的猛虎，这猛虎，一身紫纹，在他的身边，电光闪烁，居然是一头拥有雷霆之力的猛虎。
<br/>　　在九域中，还从来没有过这样的妖兽，眼前这猛虎，显然正是从这山谷之中，独特诞生的一种妖兽。
<br/>　　在楚风眠注意到这猛虎的一刻，这一头猛虎，似乎也是察觉到了楚风眠的存在。
<br/>　　突然之间，这猛虎便是从这山崖上跳了下来。
<br/>　　这猛虎突然跳下来，天际之上，也是一道雷霆突然向着他劈了过去，但是这雷霆，在这头猛虎的周围，却是突然消散，上面的雷霆，似乎都被他所吞噬。
<br/>　　一头足以吞噬雷霆的猛虎，也怪不得他居然是足以从这山谷中，生存下去。
<br/>　　“找死？”
<br/>　　楚风眠冷喝一声。
<br/>　　生死境巅峰的猛虎又如何？生死境巅峰，在楚风眠的面前，也要被轰杀。
<br/>　　人挡杀人佛挡杀佛，这猛虎既然敢拦路，那楚风眠便是就要杀了他。
<br/>　　“剑！”
<br/>　　楚风眠手中祖龙至尊剑转动一刻，便是剑锋一挑，直接向着这猛虎斩了过去。
<br/>　　“砰！”
<br/>　　剑锋斩在这猛虎的身上，居然只是斩出一道血痕。
<br/>　　这令楚风眠都有些惊讶。
<br/>　　他的这一剑，放在外面，足以斩杀任何生死境巅峰的妖兽，可眼下这猛虎的身躯，居然是无比坚韧。
<br/>　　一剑下去，只是勉强足以让他受伤。
<br/>　　“吼！”
<br/>　　这头猛虎，被楚风眠一剑斩伤，丝毫没有胆怯，反而是彻底的愤怒了一般。
<br/>　　突然之间他张开血盆大嘴，从他的口中，一颗紫色玉珠，突然向着楚风眠飞了过来。

<br/>　　楚风眠不由的有些惊讶，这凝炼神雷，控制雷霆，显然是武者才懂得的。
<br/>　　眼前这猛虎，居然也懂得。
<br/>　　“轰！”
<br/>　　神雷一出，直接在楚风眠的身边，猛然爆炸，无数雷霆，席卷这楚风眠的身躯。

<br/>　　楚风眠冷哼一声，直接以身躯，迎了上去。

<br/>　　楚风眠的身躯，在这神雷之下，毫发无伤，甚至是其中的雷霆，都是被楚风眠的身躯，逐渐的吞噬。

<br/>　　眼前这猛虎，正好是可以用来，当做黑玄剑阵的器灵，令黑玄剑阵足以更好的发挥威力。

<br/>　　楚风眠眼神冰冷，突然之间，一剑向着这猛虎斩杀过去，这一剑蕴含剑意，剑锋无双，一剑之下，必要斩杀。
<br/>　　“嘶！”
<br/>　　耀眼剑光，顿时将这猛虎，直接斩杀，黑玄剑阵一出，这猛虎的魂魄，顿时被收入到了其中去，坐镇其中，成为了黑玄剑阵的器灵。
<br/>　　“不错，这猛虎的实力，已经是达到了生死境巅峰的佼佼者，坐镇黑玄剑阵正好，不过若是足以将黑玄剑阵的十万灵剑，全部融入器灵，才算是真正圆满。”
