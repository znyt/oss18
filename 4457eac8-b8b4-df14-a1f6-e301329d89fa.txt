　　“奇叔不用担心，公子不会害我的。”
<br/>　　海公主看着一脸紧张的奇阳师，轻笑了一声，旋即看向楚风眠开口道。
<br/>　　“不错，我是玄鲸血脉，我们蓝家，曾经便是继承了玄鲸血脉的家族，强横一时。”
<br/>　　“那蓝冥，也是蓝家的人？”
<br/>　　楚风眠开口询问道。
<br/>　　听名字，这蓝冥，应该也是跟蓝家，脱不开关系。
<br/>　　“不错，那个叛徒！就是因为他，蓝家才毁于一旦的，不然的话，不至于落到如此下场。”
<br/>　　提到蓝冥的时候，海公主的语气之中，都是无比的愤怒，过了一会，她才平复了心中的愤怒，缓缓的继续开口道。
<br/>　　“蓝冥，曾经是我们蓝家资质极高的天才，在我这一代，蓝家已经没落，不如当年了，不过依然还算是沧海星域的大势力之一。”
<br/>　　“海剑宗，海祭宗等等，虽然也是有着窥探玄鲸血脉的想法，可还是忌惮与蓝家的实力不敢真正动手。”
<br/>　　“蓝冥，早早觉醒了玄鲸血脉，可是他却并不满足于自己的血脉，而是转而将主意打到其他蓝家人的身上，最终我们蓝家的老祖，死在了蓝冥的偷袭之下。”
<br/>　　“之后蓝冥，便是加入到了海祭宗之中去，然后他却一直没有停止夺取其他蓝家弟子玄鲸血脉的想法，疯狂追杀蓝家弟子。”
<br/>　　“被迫无奈，我们这一支，才不得已离开了沧海星域，前往了大罗天星域，不过事到如今，真正拥有玄鲸血脉的蓝家人，只有我一个了。”
<br/>　　“蓝冥一直没有停止对蓝家的追杀，我们这些曾经蓝家的附属，也都被蓝冥追杀，到后来，海剑宗，北海殿等势力，也加入了追杀之中，他们也想要得到玄鲸血脉。”
<br/>　　奇阳师也是恶狠狠的开口道。
<br/>　　在这数十年间，他也不知道是曾经遭遇过了多少的追杀。
<br/>　　那些追杀他的人，其实并不是要杀死他，而是打算利用他，逼出蓝家的人出来，才会一直盯着他，以寒毒消磨他的生机。
<br/>　　如果是要杀他，以海剑宗的实力，杀他一个九劫古帝，根本不用什么手段。
<br/>　　“匹夫无罪怀璧其罪，玄鲸血脉，自然会引人窥探，这一次你回来沧海星域，只怕也会遇到许多麻烦了。”
<br/>　　楚风眠看了一眼海公主，大约也从海公主的口中听懂了以前的事。
<br/>　　曾经古老的时代之中，蕴含先天神兽血脉的武者妖神，还不像是这般的稀少。
<br/>　　可无一例外，每一位拥有先天神兽血脉的武者妖神，都受到了无数次的追杀。
<br/>　　先天神兽血脉，对任何的武者妖神来说，都是有着极大的诱惑力。
<br/>　　若是实力强横还好，还可以保全自身，若是实力不济，死在追杀之下的，也比比皆是。
<br/>　　蓝家一旦没落，这个局面，也是注定的了。
<br/>　　“好在你的玄鲸血脉没有觉醒，隐藏起来，倒是没有那么麻烦，我就来帮你这一次吧。”
<br/>　　楚风眠看着一眼海公主，突然凝聚一股灵力，打入到了海公主的身躯之中。
<br/>　　随着这一股灵力打入，海公主身上的海族气息，全然消失，只有蕴含着一股淡淡的龙威。
<br/>　　同时也有一股灵力，打入到了旁边的奇阳师身上，他的一身灵力，也转化为了一种龙力。
<br/>　　看起来，便是跟真正的龙族妖神无异了，全然没有了海族的气息。
<br/>　　“我以龙族血脉，遮掩你们的气息，只要不动用全力，他人应该是无法察觉出你的真正身份来。”
<br/>　　“多谢公子。”
<br/>　　海公主，奇阳师，感觉到身上的气息，急忙的道谢了起来。
<br/>　　能够让人探察不到他们二人的气息，那他们二人，哪怕是在沧海星域之中，也不会被处处追杀了。
<br/>　　沧海星域距离苍龙星域相近，在沧海星域之中的龙族妖神不在少数。
<br/>　　伪装成龙族的样子，哪怕是海剑宗，海祭宗的人看到，都很难看穿其中的伪装。
<br/>　　以楚风眠的气息遮掩，除非是真正的天人境强者，不然根本探察出其中的玄妙。
<br/>　　“收敛起了气息，那我们也可以去其他的城池了。”
<br/>　　海公主突然喃喃自语一声，转过头对着楚风眠突然说道。
<br/>　　“公子，我打算去其他的一些城池，办一些事情，去寻找曾经蓝家的人。”
<br/>　　“正好，我也打算去海剑宗一趟。”
<br/>　　楚风眠听到这话回答道。
<br/>　　“我答应了你要杀蓝冥，不过在此之前，我还有些事要做。”
<br/>　　在见识了海祭宗的实力之后，楚风眠也是明白，要杀蓝冥，不是那么容易的事情。
<br/>　　海剑宗的实力，都已经是让楚风眠感觉到无比的棘手麻烦了，更何况是比起海剑宗还要强大的多的海祭宗。
<br/>　　一旦是楚风眠杀了蓝冥，海祭宗动怒，只怕是楚风眠要赶快离开沧海星域了。
<br/>　　所以楚风眠的计划，是先夺取剑道图腾，再杀蓝冥。
<br/>　　而且，蓝冥也是蓝家的天才，拥有玄鲸血脉，他的实力只怕是比起楚风眠曾经想象之中的还要强横的多。
<br/>　　楚风眠夺得了剑道图腾，参悟其中的话，楚风眠的实力也可以更进一步，倒时候在杀蓝冥，也有着更多的把握。
<br/>　　“那好，这一枚，是我们蓝家独特的玉符，只要是在沧海星域之中，以灵力注入玉符之中，就可以找到我的位置，等公子事情办完，也可以来寻找我们。”
<br/>　　海公主手心一动，一枚玉符便是出现在了她的手心之中，递给了楚风眠。
<br/>　　“好，如果有危险，你也可以通知我。”
<br/>　　楚风眠心神一动，突然一股灵力在他的手心之中凝聚，化为了几枚丹药，递给了海公主。
<br/>　　“这几枚丹药，是以我的力量所化，让人服下，可以跟你们一样，隐藏身上的气息。”
<br/>　　海公主这一次离开既然是要找蓝家的人，那只怕其他蓝家的人，多半也是跟奇阳师一样，受到各个势力的监视。
<br/>　　这几枚丹药，可以隐藏气息，也可以帮助他们不少。
<br/>　　“多谢公子了。”
<br/>　　海公主脸色露出几分喜色，也不推辞，接过了这几枚丹药。
<br/>　　“那我们就在这分开吧。”
<br/>　　楚风眠身形一闪，直接消失在了这海沟之中。
