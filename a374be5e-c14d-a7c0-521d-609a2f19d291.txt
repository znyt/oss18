　　不管是焚寂，或者是烛龙之鳞，又或者是玉衡，都与长琴太子有着莫大的牵扯。
<br/>　　想要获得这三样东西，都伴随着无与伦比的风险。
<br/>　　当然，慕清霄也可以用武力来解决，但曾经贵为天界的长琴太子，如果说是普通人，打死他都不信。
<br/>　　想到如今，冷酷无情的长琴太子，慕清霄叹息一声，不知何时手中拿出一半玉璧，玉璧与玉玺有些相似。
<br/>　　半块玉璧并不完整，正是他在琴川附近，山谷中的琵琶精身上所获得，而这块一般玉璧，正是玉衡。
<br/>　　玉衡，远古安邑部族用于引出魂魄，操纵灵魂之力进行铸剑的邪物。
<br/>　　把玩着手中半块玉璧，慕清霄能清晰的感觉到，四周灵气都向此处迅速汇聚而来，星眸中满是惊叹。
<br/>　　“如果是完整的玉衡，效果有多恐怖？不愧是仙器......也罢，那就在等上几年吧。”
<br/>　　转眼，两天的时间就过去。
<br/>　　昨日，弟子受封仪式上，陵越毫无疑问，以他的资质与一抹坚韧，成为了剑仙紫胤真人的首徒。
<br/>　　站在受封台上，陵越一袭真传弟子服饰，万众瞩目。
<br/>　　但受封台上，还有一个更为耀眼的存在，以至于令新老弟子，甚至长老都变得黯淡无光。
<br/>　　白衣翩翩，气质高贵，面如冠玉，站在受封台上，吸引住所有的视线。
<br/>　　当天墉城新老弟子，得知如此方年二十的青年是第十一代凝丹长老后，差点惊得连下巴都掉在地上。
<br/>　　紧紧半天的时间，慕清霄的名字就席卷了整个天墉城，且封号‘九霄真人’无人不知，无人不晓。
<br/>　　...
<br/>　　...
<br/>　　随着时间的推移，在名声传遍整座天墉城后，慕清霄并不在意这些虚名，而是了解起天墉城的信息。
<br/>　　经过一周的时间，他终于将天墉城的情况打探的一清二楚。
<br/>　　按照天墉城的戒律，凡是有新任长老接任，都会一起接受掌门的受封，有些是尊称，有些领有实权。
<br/>　　慕清霄自然就是拥有实权的那一类，而且权利大的吓人。
<br/>　　至于普通长老，就是辅佐掌门处理门派的琐事，并无实权。
<br/>　　天墉城以执剑长老最为尊贵，与掌门相同，对门派事务有决策权利，非道术剑术人品资质极佳者不可担任。
<br/>　　简单来说，天墉城除却掌门一脉，尚有五堂，分别是：执剑堂，戒律堂，妙法堂，凝丹堂，威武堂。
<br/>　　执剑堂：负责协助掌门决策门派事务，掌管铸造门派弟子所用兵器、武学修为，百年来一直由紫胤真人担任。
<br/>　　戒律堂：掌管日常清规戒律、吃穿用度，令行禁止是潜心修行之根本，现任戒律堂长老是涵究真人。
<br/>　　妙法堂：负责管理弟子修习符箓道法、静思冥想、阅读经卷，众多弟子便是在这扬清去浊的过程中修为大进，现任妙法长老是凝虚真人。
<br/>　　威武堂：负责门派弟子习武、操练，天墉城上下无不听令于他，现任威武长老是涵晋真人。
<br/>　　凝丹堂：负责炼制内服丹药、外敷伤药，不论静思冥想还是动兵刃净化浊气，内外丹药都是绝佳辅助，现任凝丹长老是还虚真人，九霄真人（慕清霄）。
<br/>　　而作为凝丹长老，慕清霄的任务，便是在每个月月初到月底这段时间内，将天墉城所规定的丹药炼制出来。
<br/>　　只要完成任务后，其余时间，都是他的自由时间，想干嘛就干嘛。
<br/>　　当然，身为凝丹长老，手中的权限可是非常的大，虽然威武堂这些与他无关，但妙法堂却是为他敞开。
<br/>　　意思就是，整个妙法堂中的修炼心法和剑技，他都能随意阅览。
<br/>　　当然，不仅仅是他，任何有实权的长老都是如此。
<br/>　　另外，古剑奇谭的修炼体系，与他的修炼路线没有任何区别。
<br/>　　修炼等级由低至高分别是：练气，筑基，金丹，元婴，化神，练虚，合道，渡劫，剑仙。
<br/>　　前面的体系都相同，但渡劫之后，却有一丝的差别，除了以剑证道的修士可以称之为剑仙，其他都没有这种资格。
<br/>　　而陵越的师尊，紫胤真人便是一名货真价实的剑仙，其余普通长老都是练虚境界。
<br/>　　至于威武长老如此，实力都要强上一些，甚至达到了练虚巅峰，作为掌门的涵素真人，更是渡劫期的强者。
<br/>　　九劫成仙，涵素真人是几劫就不清楚了。
<br/>　　...
<br/>　　...
<br/>　　随着时间的推移，一周很快就过去。
<br/>　　昆仑悬崖处的宫殿，炼丹室内，一袭白衣俊逸青年，闭目静坐在其中，如同老僧入定，身上并无一丝气息波动。
<br/>　　在他的身前，放着一座药鼎，药鼎上刻有狰狞异兽，正是天榜上的万兽鼎。
<br/>　　陡然，慕清霄睁开星眸，其中精芒一闪即逝，紧接着药鼎中就有一股淡淡的药香味弥漫而出。
<br/>　　神火灭，灵丹成。
<br/>　　打开丹炉，其中正堆积着数百颗，龙眼大小，通体碧绿，圆润有光泽的灵丹。
<br/>　　培元丹，一品丹药，丹药上有清晰可见的丹纹，绝对是极品丹药，效果极佳。
<br/>　　天墉城给初入门弟子固体培元所用丹药，可以祛除体内杂质，疏通经络，为弟子的修行打下良好基础。
<br/>　　“第五十炉极品培元丹，总算是炼制完毕了，如此一来，接下来的时间就去凝丹堂和妙法堂逛一逛吧。”
<br/>　　说着，慕清霄取出一个大型玉瓶，手一卷，丹炉中的数百颗培元丹就装进其中。
<br/>　　推开炼丹室，来到宫殿客厅中，小兰早就做好的午膳再次等待，见慕清霄出来，急忙走过来行礼。
<br/>　　“真人，午膳已经准备好。”
<br/>　　微微点头，小兰在倒是帮了他不少忙，取出一个小玉瓶，其中装着十颗培元丹，道：“收下吧，对你筑基有好处。”
<br/>　　“多......多谢真人。”
<br/>　　接过培元丹，小兰喜出望外，一瓶培元丹，可是相当于她半年的资源，居然如此简单就赠予她。
<br/>　　最重要的是，她现在刚迈入筑基，一瓶培元丹对她来说，绝对是雪中送炭。
<br/>　　将培元丹收起来，望着慕清霄的背影，眼中尽是爱慕之色。
<br/>　　“对了，待会将这些送去凝丹堂，我要去妙法堂看一看。”
<br/>　　说着，慕清霄就将装有丹药的玉瓶，放在桌上，示意一声后便开始用膳。
<br/>　　“是，真人。”
