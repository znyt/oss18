　　第701章
<br/>　　“我手下的兵可没做那些见不得人的事。”秦蛮冷哼一声道。
<br/>　　“你这话什么意思，你的兵没有，我便有了么。”
<br/>　　“好啦，好啦。一人都少说一句，都是自家弟兄有什么可吵的。”高飞虎摊开双手往下压了压。
<br/>　　“你有什么资格说话，就你手下人做得多。”高飞豹直接怼了回去。
<br/>　　高飞虎脸上讪讪，秦蛮和高飞豹手下的兵，都是千锤百炼出来的，在这次破城之中都很老实。反而是高飞虎从江南带过来的兵，没少在这次破城之后作恶，只是事情闹得不够大，没有到掉脑袋的地步。
<br/>　　“我劝二位一句，大家都老实一些吧。”秦蛮开口道：“看意思，大当家是要整顿军纪，今夜不过是提前提个醒。若再做得过分，今天能砍孙小毛，改日说不得就能砍二位。”
<br/>　　高飞虎和高飞豹脖后一凉，高飞豹道：“大当家该不会这么狠吧。”
<br/>　　高飞虎摸了摸脑袋：“大当家是越来越狠了。”
<br/>　　“军师，您倒是说句话？”秦蛮开口道，这种动脑袋的事情还得靠山寨的第一军师。
<br/>　　“啊，什么。”徐神机仿佛恍然回过神来：“今天那碗红烧肉还是不错的，大当家说得不错。”
<br/>　　“啊。”
<br/>　　三人一楞，自顾自踱远离开。
<br/>　　三人站在原地是一头雾水，半晌，高飞虎叹道：“瞧瞧，军师才是高风亮节，高某自愧不如。”
<br/>　　“若我们也都能像军师这般，只知有公，不知有私，也就不用担忧了。”秦蛮。
<br/>　　三人几乎同时叹了口气，对徐神机是心悦诚服。
<br/>　　这次破城，即便秦蛮等人能约束住自己，也未必能约束得住手下人。许多不打紧侵吞恶占的事情屡见不鲜。偏偏就是徐神机一个，他也没什么亲信，在这次破城事件中，当真是一文钱都没动，俨然成了蛤蟆寨的道德模范。
<br/>　　……
<br/>　　靠杯酒能压得倒诸人的私欲么？程大雷对此并不抱任何希望。大家聚在一起喝杯酒，一块儿忆苦思甜，然后就你好我好大家好？如果事情真的这么简单，程大雷也就不必如此麻烦了。
<br/>　　兄弟情份是盖不过私心的，程大雷对这一点心知肚明。真正能靠得住的东西，还是摆在明面上的规矩。
<br/>　　他开始着手完成这件事，制定新的军纪，主要目的还是约束手下的兵将。这帮大老粗来自草莽，多是穷凶极恶之徒，程大雷假若没能耐驾驭他们，便会被他们吞噬。
<br/>　　这大概就是水能载舟亦能覆舟的道理。
<br/>　　这件事由吴用领头，军策府的谋士群策群力完成。程大雷这边派出的人手是崔白玉，这姑娘不愧是见过大世面的，程大雷这种草莽根本没办法比。
<br/>　　崔白玉懂得程大雷在做什么事，她现在足不出户，也只有倾心帮助程大雷完成这件事。必须先证明自己的价值，才能维持自己体面的生活，即便想给程大雷使坏，现在也没有这个能力。
<br/>　　崔白玉不得不吃惊程大雷的未卜先知，事实上，蛤蟆寨的山头问题并不严重，人人都往自己口袋里捞东西的事情也并非普遍现象。只不过，如果程大雷不管，这件事会愈演愈烈，但偏偏程大雷出手及时，在事情未发生时便将其扼杀在苗头中。
<br/>　　当然，想要完成这件事也并不容易，军纪好订，谁来执行才是问题。崔白玉一边着手完成这项工作，一边思索着谁能胜任。最后她摇头想了想，觉得蛤蟆寨目前无人能胜任这刽子手的工作。
<br/>　　于此同时，程大雷正在后堂内见一位不速之客。
<br/>　　来人模样普通，衣衫普通，做短打装扮，像是田中的老农，也像是码头上的苦力，但当他报出名号时却吓了程大雷一跳。
<br/>　　鱼龙卫，田七。
<br/>　　长安破城一战，鱼龙卫死伤过半。在事平之后，李乐天又将他们组织起来，没能力扩张人手，主要也是担心有碟子奸细混入。如今李乐天也并非完全无人可用，这鱼龙卫是明帝一手调教出来的，是李氏皇族的死忠。当李乐天决定做一些事情的时候，还得依靠明帝留下的力量。
<br/>　　田七表情麻木，宛若一截木头桩子，说话也一板一眼：“程当家给陛下的信，陛下已经收到，龙颜大悦，特意让我送来回信。”
<br/>　　程大雷撇撇嘴，龙颜大悦？估计是龙颜大怒才对吧，不过这种事就是双方都揣着明白装糊涂，谁先挑明白了，谁便是不懂规矩。
<br/>　　他将当朝天子的信接过来，缓缓读罢之后，便扣在桌上。信中，李乐天是难得的客气，说明帝生前对程大雷就十分看重，常说起程大雷的好处。甚至说了程大雷不服李家，李家定不服程大雷的话。而在信里，天子还封程大雷为凉州王，如此，程大雷的身份便算是名正言顺。
<br/>　　程大雷手敲着桌面，李乐天这是要与自己做盟友，另外，也说明他现在的情况十分糟糕。
<br/>　　眼珠一转，目光落在田七身上。
<br/>　　“田七先生，这一路走来辛苦了吧？”
<br/>　　田七点点头，又摇摇头：“为陛下做事，谈不上辛苦。”
<br/>　　“嗨，坐，坐。”程大雷笑笑：“你不用这么客气，你是不知道我和陛下的关系，我和李行哉情同手足，与明帝也是一见如故，当年明帝还在的时候，非要认我当兄弟，你瞧瞧……”
<br/>　　田七这样严肃的人，眼皮也忍不住挑了挑。这程当家还真是不按常理出牌呐，这句话如果放在以前，简直就是砍头的大罪。
<br/>　　程大雷话锋忽然一转：“陛下差你过来，除了送信，有没有送别的东西，官服军印什么的？”
<br/>　　“来去匆匆，携带也不方便。”田七似乎料到程大雷这么问，回答道：“不过陛下的确想要同程当家借一件东西。”
<br/>　　“借什么，该不会是借我项上人头吧？”程大雷谨慎的盯着田七，这货该不会是来玩刺杀的吧。
<br/>　　田七也被搞得有些懵，半晌他摇摇头，道：“借兵三千。”
<br/>　　程大雷皱起眉头：“当朝天子……已经穷到这个份上了么？”
<br/>(本章完)
