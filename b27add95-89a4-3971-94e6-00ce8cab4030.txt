　　身周缭绕黑雾，苏重静静站在原地。
<br/>　　呼！一股白烟从苏重鼻孔喷出，浓重的硫磺味迅速蔓延。
<br/>　　吼！
<br/>　　凶悍气势直冲天际，一头黑发被吹的猛然竖起。无形力场好似水波，汹涌扩散。一圈圈烟尘，铺天盖地的席卷而去！
<br/>　　苏重抬起头，他的脸上，竟覆盖着一层细密鳞甲！鼻梁消失，只剩下两个鼻孔，不时喷出灼热气息。头顶上甚至长出三根魔龙角！
<br/>　　身上变化更大，衣服造就被冲击波炸碎。漆黑鳞片层层叠叠，组成一副狰狞铠甲。肩肘膝盖部位，全都凸出着锋利角质尖刺。五根手指，直接被五根利爪取代！
<br/>　　此刻的苏重，完全就是一个半龙人！相比之下，蚁王甚至都比苏重更像人！
<br/>　　“哈！”伴随着粗犷的咆哮，苏重完全不理会惊呆的蚁王。身影瞬间消失在原地。
<br/>　　魔龙·极翼！
<br/>　　锵锵锵！
<br/>　　左右双手挥舞起来，好似两把开山刀，蛮横无理的疯狂砍向蚁王。
<br/>　　没有惊天刀芒，没有冲天气势。漆黑色鳞甲覆盖的手臂，拳拳到肉！带着恐怖的力量，狠狠砸向梅路艾姆。
<br/>　　“你不是红说人类脆弱吗，你到时硬给我看啊！怎么不还手，啊？！哈！”
<br/>　　轰！
<br/>　　蒲扇般双手合十，高举过头。好似一柄板斧，轰然劈下！
<br/>　　咔嚓！时间好似静止，在这个激斗时刻，一丝极其细微的断裂声，如惊雷般传入两人耳中。
<br/>　　蚁王脸色狂变，覆盖手臂的漆黑光膜，竟出现丝丝裂纹！
<br/>　　“哈哈！”趁着蚁王愣神，苏重一脚踹过去。
<br/>　　呼！猝不及防，蚁王整个人转着圈倒飞出去。
<br/>　　“这就是你的狗屁圣体？！”苏重不依不挠，身躯微微下蹲，脚下地面轰的炸开一片蛛网，身影瞬间追上蚁王：“我倒要看看你这乌龟壳，能撑到几时！”
<br/>　　魔龙·吐息！
<br/>　　砰砰砰……
<br/>　　一连串拳影，突破空气，带着呜呜低啸，重重砸在蚁王身上。
<br/>　　“可恶！”梅路艾姆狰狞狂吼！半躺在地上的他，只能举起双手护住硬扛苏重攻击。面对苏重雨点般的拳头，他根本就没法站起来还击！
<br/>　　咔嚓咔嚓！
<br/>　　两声细微碎裂声，好似惊雷在耳边炸响，蚁王眼中满是惊骇。竟……竟然真的攻破了圣体！
<br/>　　苏重嘴角一下咧开至耳根，张开一嘴细密尖牙。“哈哈哈！去死吧！”拳头力量再重三分！
<br/>　　咔嚓咔嚓咔嚓……噗噗噗！
<br/>　　紫色血液猛然飞溅而起，脸上沾染血液，苏重竖瞳中充满残暴兴奋。
<br/>　　东拓国皇宫。
<br/>　　百式观音·第九十九式！
<br/>　　嗡！
<br/>　　数十米高大的观音神像，数百手臂同时攻击。各种各样的掌型，带着绝强的力量，凶狠的砸向漂浮半空的蚁王鬼身。
<br/>　　“没用的，我告诉过你。我是鬼身，你的一切攻击，都落不到我身上。”蚁王鬼身突兀出现在半空。
<br/>　　“没用的话，你躲什么呀？”尼特罗戏谑的笑着。身躯却微微弓起，双眼死死盯着蚁王鬼身，不敢有丝毫怠慢。
<br/>　　鬼身沉默不语，尼特罗的百式观音，可不是什么花哨能力。完全是极致的武道。他把所有念，全都用来加成武道。
<br/>　　这种极致的力量攻击，正好是蚁王的克星！
<br/>　　鬼身能一定程度虚无化攻击，但当尼特罗施展九十九连击，狂暴到极点的攻击，已能威胁他的存在。鬼身蚁王不得不躲！
<br/>　　尼特罗呼吸如风箱一般，这一连串的攻击，对他的消耗非常大。
<br/>　　为了能过触碰蚁王鬼身，他不得不加大输出。但事实证明，即便是第九十九击，也只不过让鬼分身忌惮，开始主动闪避而已。
<br/>　　“难道要发动零式？”尼特罗眼睛眯起，心里下了决定。
<br/>　　百式观音零式，是同归于尽的招式。用双手，温柔又霸道的禁锢住对方，然后将全身所有念气，化作无尽的光和热，施展最无情的轰杀！
<br/>　　一旦施展，他全身的念，将会瞬间释放干净。念是生命力，是生物场，是生物生存的动力根本。一旦全部释放，他必死无疑！
<br/>　　不能等了！一旦苏重失败，两个蚁王将会是一场无法想象的灾难，必须遏制！
<br/>　　正当他准备拼命的时候，蚁王鬼身突然一僵。漆黑的身体一震模糊，好似荧屏卡顿雪花一样。如果不是尼特罗始终盯着，根本就无法发觉。
<br/>　　难道，蚁王那边出了变故？是失败，还是胜利，亦或者正在胶着，因此才影响到鬼身。短短瞬间，尼特罗脑海中闪现出数个念头。不等他继续深思，蚁王鬼身一改之前漠然不屑姿态，脸上狰狞毕露！
<br/>　　“拦住他！”伴随一声狂吼，鬼身整个人化作一团漆黑浓烟，拉着一条长长尾焰，迅速向天边飞去。
<br/>　　“果然出变故了！”尼特罗心中一喜。眼角瞥过一前一后堵住他去路的两大护卫，咧嘴一笑。两个小蚂蚁……打不着蚁王，难道还收拾不了你们。
<br/>　　到了他这种程度，任何诡异的能力，都对他无效。是真正的一拳破万法！
<br/>　　瞬间废掉两只蚁王护卫的行动能力，尼特罗身影一闪消失。他不会飞行，只能在地面上狂奔，沿途烟尘滚滚，直追蚁王鬼身而去。
<br/>　　一前一后，两人近乎同时到达蚁王战场。
<br/>　　整片空旷原野，到处是大坑。好似被炸弹雨洗地一般。原本高耸的风化岩石山，早就没了踪影，只剩下一堆碎石。
<br/>　　呜！轰！
<br/>　　战场中心，苏重半蹲在地上，一手按着蚁王，一手握拳，一下又一下砸在梅路艾姆头上。
<br/>　　紫色血浆，不断迸溅而出。蚁王全身甲壳破碎不堪，竟被生生砸烂！他此刻已然奄奄一息，频临死亡。
<br/>　　鬼身和蚁王本是一体，看到梅路艾姆如此凄惨，脸色顿时狰狞起来。身受重伤，毫无反抗的蚁王，看到鬼身到来，眼中立刻亮起一丝喜色。
<br/>　　他和鬼身两者一体。他吞噬了太多人类，鬼身是他无法消化的精神意念汇聚而成。此刻他身受重伤，鬼身对他来说，就是大补丹！
<br/>　　一旦两者融合，蚁王立刻就能恢复，甚至破而后立，融合阴阳两面，变得更加强悍！
<br/>　　呼！蚁王能想到，鬼身怎么会想不到。仗着鬼身特殊，虚不受力。蚁王鬼身狠狠瞪了苏重一眼，竟然根本不理会苏重，直接来到苏重面前，合身扑向蚁王。
<br/>　　“想要合二为一，想要翻盘？在我面前你还这么嚣张，我岂不是很没面子，嗯！”
<br/>　　啪！
<br/>　　苏重右手，竟然直接抓住了鬼身的脖子！
<br/>　　“不可能，！你怎么能抓到我？”鬼身眼中满是惊骇。厉害如尼特罗，都无法攻击到他。苏重竟伸手就能抓到自己！
<br/>　　有道种存在，魔龙法相都能吞进肚子里，抓住鬼身，苏重一点儿都不意外。

<br/>　　“呃！吓死啦？真没趣，我只是说说而已，你怎么还当真了。”苏重毫不负责的道：“看你那一身紫呼呼的血，我就没兴趣。谁知道有没有毒！”
<br/>　　刚刚走到近前的尼特罗，闻言嘴角猛的一抽。蚁王死的可真冤！不过即便活着，估计也会为这语气，羞愤而死吧。这也太不把蚁王当王了。
