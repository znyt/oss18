        童筱颖懵地恍然大悟，完全没料到那些东西竟然是季运恒送的。
<br/>
<br/>        怪不得她一直觉得那些东西来得很虚……
<br/>
<br/>        “你送的那些东西，筱颖一样没碰，都被我们这几个室友给吃了。”严恩恩看得出童筱颖不喜欢这个自称为“本少”的男人，于是好心开口帮童筱颖说话。
<br/>
<br/>        就在这个时候，背后不知道是谁突然用力推了童筱颖一下。
<br/>
<br/>        童筱颖往前扑去的一瞬间，季运恒伸手便将她搂入怀中，坏坏地勾唇一笑：“我还真发现，你原来就是‘夜惑’里的那个小谢啊！”
<br/>
<br/>        “你……”童筱颖下意识地推搡着季运恒。
<br/>
<br/>        季运恒反倒越抱越紧。
<br/>
<br/>        她刚从澡堂里洗完澡出来，头发湿漉漉地黏在了脖子和衣衫上，衬衣上被发梢打湿的地方，紧贴着她的肌肤，呈现一片若隐若现的诱惑色。
<br/>
<br/>        季运恒垂眸扫过童筱颖的胸前，单手扣住她的后背，将她软软的胸脯紧紧地贴着自己的胸膛。
<br/>
<br/>        隐隐的一个“咔擦”声，吸引了严恩恩的注意。
<br/>
<br/>        严恩恩下意识地循声望去，无意间撞见一个男生手里正拿着单反相机对着这边偷拍。
<br/>
<br/>        童筱颖用力推开季运恒，反驳道：“你认错人了。”
<br/>
<br/>        季运恒剑眉微扬，伸手捻住童筱颖的下巴，凑到她的脸前，不怀好意地威胁道：“小谢的声音，本少分得一清二楚。童筱颖，你应该不希望你身边的同学知道你曾经在夜店里卖过唱吧？噢！对了！你何止卖唱，你还卖身吧！那个‘连少’给了你多少钱，才让你肯跟他睡的？”
<br/>
<br/>        “啪”地一声，一个巴掌声突兀地在校道上响了起来。
<br/>
<br/>        童筱颖恼火地挪开季运恒捏住自己下巴的那只手后，抬起另一只手猝不及防地掴了季运恒一巴掌。
<br/>
<br/>        季运恒虽然被童筱颖打得一脸懵，却捉住童筱颖的手腕，咧嘴一笑：“呵，原来是个‘小辣椒’啊！不错，本少喜欢！”
<br/>
<br/>        “可恶！你放手！”童筱颖凝眉怒斥。
<br/>
<br/>        站在一旁的严恩恩，见这男人纠缠着童筱颖不放，心里也没了个主意，想插句话，又不知道自己该说什么，只好像“挂机”了一样，愣在童筱颖的身边束手无策。
<br/>
<br/>        “军训搞完了吧？听说你们学校有三天的休息假，那么就陪本少出去兜兜风！”季运恒痞里痞气地说，拽着童筱颖的手，拉开了车门，将她往自己跑车里塞。
<br/>
<br/>        童筱颖一边挣扎一边向严恩恩呼救道：“恩恩，报警！帮我报警啊！”
<br/>
<br/>        严恩恩连忙上前追出去一步，季运恒转脸瞪着她呵斥了一声：“找死？！”
<br/>
<br/>        这一瞬间，严恩恩便怂了。
<br/>
<br/>        季运恒随之从车内的储物箱里摸出一把手铐，一头铐住童筱颖的手腕，另一头铐在了车椅上头枕的不锈钢固定架上。
<br/>
<br/>        “恩恩！恩恩！”童筱颖见严恩恩发愣，大声叫唤道。
<br/>
<br/>        严恩恩吓得回过神来，连忙掏出手机准备打电话报警，谁知手机却被突然返回来的季运恒给夺走。
<br/>
<br/>        “筱颖是我女朋友，我两的事情，你最好少管！还有，你敢报警，你试试，看我日后会不会找你算账！”季运恒眯起黑眸恐吓道。
<br/>
<br/>        严恩恩一脸无奈地看向童筱颖。
<br/>
<br/>        童筱颖也不再指望严恩恩了，毕竟，季运恒说要找严恩恩算账，他必定说到做到。
<br/>
<br/>        季运恒呵斥完严恩恩，转身上了驾驶座，按下跑车的自动车顶键，盖上车顶后，开车载着童筱颖扬长而去。
<br/>
<br/>        严恩恩心里急得直跺脚。
<br/>
<br/>        此时，童筱颖落下的桶子里传来了一阵手机铃声。
<br/>
<br/>        严恩恩立即走过去，从桶子里的衣服堆里翻出了童筱颖的手机，见来电显示上是“大哥”，于是好心替童筱颖接听了电话。
<br/>
<br/>        “筱颖，你到宿舍了吗？”电话刚一接通，傅夜沉便关问道。
<br/>
<br/>        严恩恩讷讷地回答道：“那个……我不是筱颖……就是筱颖被一个男人带走了！”
<br/>
<br/>        “什么男人？”傅夜沉身心一怔。
<br/>
<br/>        “我不知道是谁，筱颖说她不认识他，她不愿意跟他走，筱颖是被那男人强行带走的。”严恩恩说完后，顿了顿，准确无误地报出了季运恒的车牌号码，“我知道他的车牌号码，他的车牌号码是……”
<br/>
<br/>        “好，我知道了。谢谢你。”傅夜沉挂了电话后，连忙利用自己的关系，定位车牌号码。
<br/>
<br/>        另一边，季运恒载着童筱颖直接开向了海边。
<br/>
<br/>        这片区域，因为还未开发，所以几乎没有人来。
<br/>
<br/>        本来，他是打算帮妹妹季思妍一把，把童筱颖追到手后，就甩了童筱颖。
