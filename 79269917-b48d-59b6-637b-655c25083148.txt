　　北邙学院传承万年，像是创始者，肯定不止是有着夜皇一人。
<br/>　　不过像是这样的创始者，在平日里，都是选择沉睡，很少会出现在学院之中来。
<br/>　　夜皇，也是因为夜众星的呼唤，如今才苏醒过来。
<br/>　　任何一个势力，想要发展起来，都有着自己的底蕴，这北邙学院的创始者，自然都是他们的底蕴。
<br/>　　玄北圣者现在的实力，的确是强大无比。
<br/>　　天命之境。
<br/>　　就算是如今的夜皇，都没有自信，可以在这玄北圣者的面前出手，镇压楚风眠。
<br/>　　可若是真的要让北邙学院的那些创始者，全部苏醒过来的话，可就不止是楚风眠了，就算是玄北圣者，也会被镇压！
<br/>　　“你要是敢，就试试看！”
<br/>　　玄北圣者听到夜皇的话，不屑的开口道。
<br/>　　这语气，简直是让夜皇都有些气的疯狂起来。
<br/>　　他如今威胁玄北圣者，也正是不想要跟玄北圣者起真正的正面冲突，毕竟创始者沉睡，一旦被打扰，都会付出极大的代价才行苏醒过来。
<br/>　　一般来说，只有是北邙学院在生死关头之际，他们这些创始者，才会选择苏醒过来。
<br/>　　如今为了对付楚风眠跟玄北圣者，对付北邙学院的自己人，付出如此之大的代价，的确是有些不值。
<br/>　　夜皇的威胁，也只是在吓唬玄北圣者，想要让玄北圣者主动退避，只要是玄北圣者不插手，一个楚风眠，对他们倒是都算不了什么，想要对付，轻而易举。
<br/>　　可现在，他的威胁却是没有任何一点的作用。
<br/>　　“夜皇，今日你就算是杀了我，我也可以将你们在场所有夜家老祖，全部斩杀，倒时候倒是要看看，你们夜家，在这北邙学院之中，还有没有立足之地！”
<br/>　　楚风眠语气森然，令人听见都是不寒而栗。
<br/>　　尤其是那些夜家老祖，一个个听到楚风眠的话，都是汗毛林立起来，他们可都不敢质疑楚风眠的话。
<br/>　　楚风眠一向都是说到做到，尤其是现在，就连夜众星都是败在了楚风眠的手中。
<br/>　　他们夜家老祖之中大部分人的实力，甚至还都不如那夜众星，楚风眠真的拼上性命出手，很有可能真的像他所说的一样，足以是将夜家的老祖，斩杀大半。
<br/>　　倒时候他们夜家就算是赢了，也是惨胜，将来摔落，便是不可避免的。
<br/>　　“楚风眠，你别扯口舌之勇！老祖宗不杀你，只是给你一个改过自新的机会！你居然还不识好歹，速速跪下认错，才是你唯一的活路！”
<br/>　　夜众星听到楚风眠的话，不服的开口道。
<br/>　　他被楚风眠击败，本来便是让他颜面尽失，现在楚风眠开口，居然是直接拿他们夜家的老祖，作为威胁，令他如何忍受。
<br/>　　“废物一个，也敢开口？”
<br/>　　楚风眠冷哼一声，突然之间，他的身形动了，这身影如同幻影一般，突然之间，整个北邙宫之中便是都被楚风眠的幻影所充斥。
<br/>　　眼花连乱，令不少老祖都分不清，到底哪一个才是真正的楚风眠。
<br/>　　而几乎就是在这瞬间，楚风眠的身形，却是猛然来到了夜众星的面前，突然一掌，直接冲着夜众星猛然抓了过去，这夜众星的脑袋，一下子都被楚风眠抓在了手中。
<br/>　　只要是楚风眠现在稍稍用力，这夜众星，便是真正的会死在楚风眠的手中，死的惨到不能再惨。
<br/>　　“楚风眠，你要做什么！”
<br/>　　在刚刚那刹那之间，夜皇甚至都无法分清楚，到底哪一个虚影，是真正的楚风眠。
<br/>　　魅影身法，虚无缥缈，在整个九域之中都足以称得上是第一的身法，现在楚风眠已经是将魅影身法修炼到了大成的地步。
<br/>　　就算是这夜皇，在刹那之间都无法看清楚楚风眠的身形，等到他反应过来的一刻，这夜众星的头颅，已经是被楚风眠捏在了手中。
<br/>　　就算是他现在出手，都来不及，他现在出手再快，也绝对快不过楚风眠。
<br/>　　本来夜皇还觉得，楚风眠的威胁有些可笑。
<br/>　　可现在，他的脸色却是彻底的僵住了，再也笑不出来。
<br/>　　楚风眠现在展现出如此身法，正是在代表着他所说的话，绝对没有任何一句虚言，只要是楚风眠愿意，现在的他，足以是将在场大部分的夜家老祖，都给屠戮一空。
<br/>　　甚至是可能将外面，整个北邙学院的夜家弟子，全部屠杀。
<br/>　　就算是夜皇可以出手，最终轰杀了楚风眠，可是夜家已经彻底毁灭，他们就算是杀了楚风眠，也无济于事。
<br/>　　这一点，正好是戳到了夜皇的软肋。
<br/>　　他们夜家对付楚风眠，正是为了报当年夜天君的仇怨，可现在，要是杀了楚风眠，要付出夜家大半弟子性命的代价。
<br/>　　就算是夜皇，都不敢承受。
<br/>　　“楚风眠，速速放开众星，众星可是北邙学院的副院长，你居然抓着他的头颅，这是死罪，你知道吗？”
<br/>　　又是一尊夜家老祖，大声的开口道。
<br/>　　“闭嘴，在废话，我下一个就杀了你！”
<br/>　　楚风眠听到那夜家老祖开口，冰冷的开口道。
<br/>　　这一句话，露出森森杀气，令在场无数老祖，都是不寒而栗，太疯狂了。
<br/>　　这种举动。
<br/>　　夜家在北邙学院，已经是纵横了万年，在这万年之中，都从来没有人，敢这样的挑衅他们夜家。
<br/>　　一般强者，就算是有着可以毁灭夜家大半弟子的实力，也无人敢去做，因为这样做，肯定会被夜家不死不休的追杀，能够走到这一步的强者，哪一个不是极为珍惜自己的性命，谁都不会去做如此冒险的举动。
<br/>　　可是楚风眠不同，楚风眠可是疯狂到极点的性格。
<br/>　　前世的他，已经是死过一次了，这一世的命已经是赚来了。
<br/>　　更何况他前世，就算是手无缚鸡之力，都从来不会受人威胁，宁死不屈，这一世，他拥有如此实力，就更不会妥协。
<br/>　　既然夜家要杀，那就杀个鱼死网破。
<br/>　　楚风眠站在空中，他的身上弄弄的杀机，已经是明晃晃的展现了出来。
<br/>　　夜众星的头颅，被楚风眠捏在手中，随着楚风眠逐渐用力，甚至是下一刻，都要被捏碎一般。
