        男人的胸肌纹理清晰，凹凸有致，线条轮廓的触感格外结实。
<br/>
<br/>        童筱颖的小手不安分地落在他左边的胸口上画圈圈，唇上任由他的吻一遍又一遍地碾压。
<br/>
<br/>        突然，一只温热的大手，捉住了她这只不安分的小手。
<br/>
<br/>        童筱颖倏然睁开了双眸，与傅夜沉的黑眸四目相对。
<br/>
<br/>        “想要？”他的声音忽而低沉富有磁性。
<br/>
<br/>        童筱颖怔愣地眨了眨眼睛，“唰”地一下，满脸涨得通红。
<br/>
<br/>        “你想好什么时候爱上我了，我便什么时候要你。”傅夜沉薄唇微扬，语气不冷不热，倒也不像是在说玩笑话。
<br/>
<br/>        刚刚，她这是怎么呢？
<br/>
<br/>        身体竟然会不听大脑使唤地想要去亲近他。
<br/>
<br/>        童筱颖下意识地将手从傅夜沉的大手中抽离，双脚前后推移着身子，挪动着屁股远离傅夜沉的跟前。
<br/>
<br/>        傅夜沉凝眉揉了揉太阳穴，只觉头脑昏沉，接着命令道：“去给我弄碗醒酒汤。”
<br/>
<br/>        “……”童筱颖语塞，顿了顿后，反问，“醒酒汤要怎么弄？”
<br/>
<br/>        见她半晌没动静，他才想起现在的她，已经不会下厨的这件事情来。
<br/>
<br/>        “算了！”傅夜沉又自言自语道，“我还是自己去弄。”
<br/>
<br/>        他妥协了。
<br/>
<br/>        向她妥协。
<br/>
<br/>        明知道她留在自己身边是一颗随时都能威胁到他生命安全的定时炸弹，但他仍旧决定将她留下来。
<br/>
<br/>        死在她的手中，总比死在其他人的手中要强。
<br/>
<br/>        而就在此时，敞开的房门被人“咚咚咚”地叩响。
<br/>
<br/>        落以柔手中端着一个碗，一丝不苟地说道：“傅总，我给您做了一碗醒酒汤，您趁热喝。”
<br/>
<br/>        起初的那道枪声，她听到了。
<br/>
<br/>        当她赶到他的房门口时，看到的却是他正在跟童筱颖接吻。
<br/>
<br/>        每一次看到他俩亲热，她的心依旧会像针扎般痛着，但越是这样，她越是要冷静，于是她转身去了楼下的厨房，给他做了一碗醒酒汤上来。
<br/>
<br/>        “谢谢。”傅夜沉淡淡地应了声。
<br/>
<br/>        落以柔走进来后，将手中的醒酒汤呈给了傅夜沉。
<br/>
<br/>        童筱颖见状，连忙爬到床头的一角坐着，双手抱膝，远离傅夜沉这个阴晴不定的男人。
<br/>
<br/>        “童小姐，我带你去隔壁房间休息吧！这么晚了，傅总也需要休息了。”落以柔站在床边，一副女主人的姿态，居高临下地对童筱颖说道。
<br/>
<br/>        傅夜沉一口气将碗里的醒酒汤喝完后，一边将空碗递给落以柔，一边冷冷地替童筱颖回答道：“她今晚睡我这儿，你下去吧！把门关上。”
<br/>
<br/>        落以柔抿了抿唇，无言以对，接过碗后，转身离去。
<br/>
<br/>        出门后，她便顺手将房门给带关了。
<br/>
<br/>        童筱颖下意识地看了傅夜沉一眼，又将目光挪向了一旁。
<br/>
<br/>        傅夜沉意味深长地凝望着童筱颖，嘴角情不自禁地微微上扬。
<br/>
<br/>        其实，在现在的筱颖身上，仍旧能看到过去的那个筱颖的影子。
<br/>
<br/>        他今晚确实是喝醉了，甚至醉得不省人事，但是当他倏然酒醒的那一刻，看到她想要拔出他的手枪，他的心里是一片慌乱与愤怒。
<br/>
<br/>        可是在他看到她哭着说，不想离开他的时候，他还是心软了。
<br/>
<br/>        他就是无法对她狠心，哪怕她现在已经变了。
<br/>
<br/>        既然她变了，那么他便引诱她再次爱上他，跟她玩欲擒故纵的把戏。
<br/>
<br/>        “你若不想离开我的话，那就要学会乖乖地听我的话，不要做一些惹怒我、背叛我的事情。”傅夜沉朝童筱颖凑近了些，纤长好看的大手轻轻地捏住她的下巴，微微抬起来，让她的这张漂亮的脸蛋面向自己，语气忽然间阴鸷道，“不然，我会杀了你。要知道，你本来就是一个已死之人。”
<br/>
<br/>        童筱颖身心一滞，回眸对上了傅夜沉这双深邃的黑眸。
<br/>
<br/>        这才是他的本性吗？
<br/>
<br/>        浑身充斥着戾气与霸道，那些温柔与体贴，只不过是他的全副伪装。
<br/>
<br/>        可明明他又可以做到为了救她，连自己的性命都可以不顾。
<br/>
<br/>        “我、我去洗澡。”童筱颖缩着肩膀，从傅夜沉的身前逃离，利索地下床，直奔内卫。
<br/>
<br/>        傅夜沉的目光追随童筱颖的背影而去，薄唇勾起一抹意味深长的弧度。
<br/>
<br/>        一如既往，她洗完澡后，他会主动拿吹风机过来替她吹头发，帮她把头发吹干后，他才会去洗澡。
<br/>
<br/>        等到他洗完澡出来时，她已经躺床上窝成一团睡着了，而他却只是睡在墙角的贵妃椅上。
<br/>
<br/>        他跟她，仍旧是同房不同床。
<br/>
<br/>        他倒是把自己的大床，让给了她一个人享受。
<br/>
<br/>        只是令童筱颖心寒的是，自从她昨晚拔枪想要杀他被他撞破后，这天早上起来，那位连老头的儿子连华生，在她后颈的皮下组织里植入了纳米监听定位器。
