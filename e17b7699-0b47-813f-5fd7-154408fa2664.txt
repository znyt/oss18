　　小年之后，离过节也就只有几天了，叶少阳借这个机会，在微信上给认识的人都发了拜年信息，大家也都回了，包括周静茹和谢雨晴，但是当他找他们聊天时，两人又不理了。叶少阳翻开他们的朋友圈，周静茹全是有关旅游的，她好像在欧洲到处游玩，有时候也会发一些别人拍的自己的背影，但都没有正面。
<br/>　　谢雨晴朋友圈发的都是有关新工作的，看上去似乎在新单位呆的很舒服，不过也没自拍照。
<br/>　　叶少阳捧着手机，有点怅然若失。
<br/>　　除了这两位，别人倒是都回复了，叶小萌和张小蕊去了茅山，看什么都新鲜，发了很多照片给他，张小蕊还弄了道袍穿，让人在三清大殿前给她拍照，正襟危坐的，弄得跟道姑似的。
<br/>　　这个蠢货，拍照把后面的神像都拍进去了。于是骂了她几句，让她好生修炼，不要浪费时间在这些没意义的事情上。又跟苏钦章关照，让他不要太照顾那两个妹子，要多加管教。
<br/>　　苏钦章发了个“笑哭”的表情，然后说：管教是不可能的，这辈子都不可能的。山上风景又好，条件也好，她们都超喜欢山上的。
<br/>　　叶少阳无语，他知道张小蕊八成把山上弄得鸡飞狗跳的，不过应该也不会做什么出格的事，打算过了年回山上看看。
<br/>　　年前，叶少阳也算过了几天闲散的日子，让瓜瓜去青冥界，告诉林三生他们，年三十都到这边来，大家聚餐一下，最重要是把芮冷玉和道风也叫来。
<br/>　　在过年前两天，突然微信上有个人发来一条消息：少阳，在石城吗，出来聚聚？
<br/>　　这人网名叫九灯和善，叶少阳一时间没想起来是谁，查看了聊天记录，才知道他是李多，自己的室友，当年调查冯心雨事件时，他也参与了的，因为他是普通人，当时不想让他过多知道法术界的事，对他，还有另外几个室友，叶少阳都没再怎么联系，他们也都忙着实习找工作，平时不联系，只微信上偶尔问候下。
<br/>　　叶少阳很懒，从来没有备注的习惯，这李多改了网名，他一时没认出来。
<br/>　　想到这几个室友，叶少阳顿时有一种沧桑的感觉，冯心雨……那是自己下山之初单独搞定的第一个灵异事件，那时候自己还是个单纯小少年，从那之后，在这条路上就越走越远了。
<br/>　　叶少阳问了李多有哪些人，说是几个同学，都是他认识的，叶少阳想了一下答应了，约了第二天中午。
<br/>　　快睡觉的时候，微信突然响了，叶少阳一下子把手机拿起来，看了一眼不是周静茹或谢雨晴，但一样让他激动，发来信息的是覃小慧，问他在干什么。
<br/>　　叶少阳赶紧回了信息，跟她聊起来，才知道她最近去了广西，这块过年了，就到了城市里，目前在昆明，因此最近有网络。
<br/>　　两人互相都聊了最近的情况，约好过了年见一面，一直聊到半夜，才互道晚安睡觉。
<br/>　　第二天中午，叶少阳去赴约，去吃火锅，到的时候，李多已经在饭店门口等着了，半年不见，他还是老样子，只发型换了，打扮也更成熟了。
<br/>　　“少阳，好久不见。”李多很兴奋，寒暄两句，拉叶少阳进包间去。
<br/>　　“这是我女朋友，你们见过的。”
<br/>　　叶少阳看了一眼，面是熟的，可是忘记她叫什么了，听李多叫她丽丽，才想起来是何丽丽，陈宇也站起来打招呼，陈宇边上还坐着一个人，叶少阳看着也很面熟，一下子却没想起来是谁（在记人方面真的不要太差），不能李多开口，他自己站起来，冲叶少阳露出意味深长的笑容，“少阳哥，你不记得我了？”
<br/>　　“你是……”
<br/>　　“我是胡帅啊。”
<br/>　　胡帅？
<br/>　　叶少阳搜寻记忆，胡帅笑道：“你是贵人多忘事，做了好事就忘了，我们可不敢忘，我家里可供着你的长生牌位呢。”
<br/>　　长生牌位？什么鬼？
<br/>　　胡帅拿出一个小瓷瓶，对着它说道：“你不是一直想见恩人，今天他来了，还不出来打招呼。”
<br/>　　从瓷瓶里飞出来一个身穿长裙的女鬼，站在他边上，冲叶少阳抿嘴笑着：“叶天师，好久不见了。”
<br/>　　是他们！
<br/>　　叶少阳点头道：“我记得了，你是廖青青！”
<br/>　　廖青青，是当初被冯心雨杀掉的那个女学生，后来自己干掉冯心雨，找回了她的魂魄，但她死活不愿意去投胎，非要留在胡帅身边，当时自己差点强行拘她走，还是他们几个苦苦哀求，自己心软才让她留下。
<br/>　　“你们……最近咋样？”
<br/>　　这一人一鬼互相看了一眼，甜蜜一笑，算是最好的回答了。
<br/>　　叶少阳感慨道：“当时我还觉得你们未必能坚持下来，看到你们还这么好，我也就放心了，当初也算做了一件好事。”
<br/>　　叶少阳如今的心态，跟当时又有些不同了，当时自己还是比较恪守传统的，如果换成是今天的自己，肯定想也不想就成全他们。‘
<br/>　　廖青青道：“一生太长，可不敢说以后，不过我们俩都相信对方，一定能克服那些困难，走到最后的，到时候做一对鬼情侣。”
<br/>　　叶少阳听了这话也很感动，当即画了一张符，交给胡帅：“我也就送佛送到西，这个你好生收好，孤魂野鬼是做不得的，假如你死的时候你们还在一起，到时候把这道符烧了，拿着到阴司，交给……”突然想到不能说太多阴司的事，“你们去了阴司随便给个人看就知道了。”
<br/>　　两人千恩万谢。
<br/>　　接着就是吃菜喝酒了，聊起彼此的近况，叶少阳当然不会说什么，随便敷衍，主要是听他们讲述，大半年不见，他们如今都已经找好了工作，等着拿毕业证了。李多考上了公务员，胡帅自由职业，自己在家做平面设计，廖青青也是学这个的，可以帮他一起。
<br/>　　陈宇自己开了一个店，专门做婚庆服务，正好跟胡帅有合作，互相介绍客户，关系很好。
<br/>(本章完)
