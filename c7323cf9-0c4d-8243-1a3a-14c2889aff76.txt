　　从小医仙体内席卷而出的压迫感，白山有种骂娘的冲动。
<br/>　　本以为是最好对付的，没想到是此女隐藏的极深，而且是毒修，以他六星大斗师的修为，根本对付不了。
<br/>　　“小医仙姑娘，你看那萧炎与古薰儿实力都深不可测，琥嘉与吴昊虎视眈眈，不如我们合作，先将他们踢出去？”
<br/>　　小医仙不屑的瞥了眼白山，灰色毒气自体内流转而出，旋即一股死气向四周扩散开来。
<br/>　　脚步微微一抬，便是化作一道残影，携着剧烈的破风声，一掌印向脸色阴沉的白山。
<br/>　　“你……”
<br/>　　感受到扑面而来的死气，白山脸色剧变，下一秒银色斗气狂涌而出将周身笼罩。
<br/>　　手中银枪一震，雷弧布满枪身，右手紧握枪柄，厉喝一声，长枪化作一抹银色光芒，极其狠辣的向小医仙刺去。
<br/>　　“既然如此，就让我看看，斗灵究竟有多强。”
<br/>　　“噗嗤……”
<br/>　　两股恐怖的气息相撞，白山长枪直接刺破毒气屏障，却未伤到小医仙半个发丝。
<br/>　　白山一喜，原本以为自己在她手中一招都走不过，没想到竟能破开防御。
<br/>　　眼底闪过一抹阴狠，若是自己以六星大斗师的修为，将二星斗灵的小医仙击败，进入内院肯定会被重点培养。
<br/>　　想到此处，白山下起手来更是狠辣无比，招招都往要害刺去。
<br/>　　小医仙美眸中闪过一抹愤怒，没想到此人如此狠辣，居然直接朝自己下死手，根本丝毫不留情面。
<br/>　　若非她体内设有封印，大量毒气不能调动，哪里是他一个大斗师能抗衡的。
<br/>　　“仙儿，封印正在快速消弱，给本王住手，否则毒气恐怕会借此一举冲破封印。”
<br/>　　脑海中响起彩鳞急切的声音，小医仙瞥了一眼脸色狰狞的白山，直接调动体内毒气，身如魅影，一掌拍出。
<br/>　　察觉到她力量直接提高数倍，白山脸色狂变，心中一狠。
<br/>　　只见他嘴角挑起一抹阴冷，双手忽然搭在枪腰之上，旋即奋力旋转，顿时一杆长枪几乎化作风轮。
<br/>　　随着长枪疯狂旋转，白山身体表面银光大盛，银色电芒如同小蛇一样吐缩。
<br/>　　此时的白山几乎犹如一个银色光球，并且在光球的表面，还布满着无数的银色触角，其中蕴含着极其狂暴的能量。
<br/>　　霎时间，破风声陡然停滞，白山手握一杆银色雷电枪，双脚一侧，犹如一条雷龙朝小医仙冲去。
<br/>　　在无数道震撼的目光中，长枪化作一条数丈的银色雷龙，奔腾而出。
<br/>　　雷龙速度快的令许多学员肉眼都难以跟上，只见一道银光闪过，地面就出现一条深深的沟壑。
<br/>　　旋即，伴随着一道震耳欲聋的爆炸声，循声望去，只见雷龙与小医仙掌心相撞，恐怖的劲风自两人为中心呼啸而出。
<br/>　　感受到恐怖的力量朝自己挤压而来，白山一喜，因为他感觉后者的力量似乎在变弱。
<br/>　　毒气虽强，但一时间被斗气隔绝在外，难以接触他的身躯，只见白山大喝一声：“斗灵不过如此。”
<br/>　　旋即，便将自己的力量全部灌入长枪，猛的用力一推。
<br/>　　“噗嗤……”
<br/>　　正当众人觉得白山不自量力的时候，只见小医仙粉唇中喷出一口鲜血，娇弱的身躯倒飞而出。
<br/>　　薄纱随风落下，露出一张绝美的容颜，嘴角的血痕令她显得更加憔悴，毒气一震直接在空中稳住身形。
<br/>　　围观学员在这一瞬间，都为她的容貌而惊艳。
<br/>　　让他们没想到的是，那薄纱底下的一张容颜，竟丝毫不输于学院的女神熏儿。
<br/>　　长枪触及地面，白山脸庞上也浮现一抹苍白，但嘴角却挂着阴狠的笑意，这一战是他赢了，后者不过是强弓之末。
<br/>　　他可不会因为后者的美色而手软。
<br/>　　正当他准备再次出手的时候，只见小医仙落在赛场上，身躯摇摇欲坠，绝美的脸上浮现一抹青光。
<br/>　　下一秒，一条小蛇自长袖中钻出，随后化作一道红裙身影将她护住，倾世容颜上布满怒火与冰冷的杀意。
<br/>　　封印要破了！
<br/>　　此时，恐怖威压席卷全场，学员们皆是在威压下颤抖。
<br/>　　比赛终止，无论看台上，或者是赛场中的气氛都压抑的有些可怕。
<br/>　　被后者金色眸子一瞥，白山浑身一震，双腿一弯直接跪在了地上，鲜血自口中喷吐而出，脸色萎靡。
<br/>　　瞬间，白山心中顿时涌上无尽的恐惧，斗宗的恐怖威压，根本不是他一个大斗师能抵抗的。

<br/>　　为首老者正是迦南学院大长老苏千，其余几名老者皆是内院长老。
<br/>　　苏千瞥了一眼小医仙，随后目光定格在彩鳞身上，道：“倒是我们疏忽了，你可知自己坏了学院的规矩？”
<br/>　　这时，其中一名长老上前一步，目光一凝，道：“你的气息给我一种似曾相识的感觉，五年前闯入练气塔的斗宗就是你吧！”
<br/>　　彩鳞完全不理会老者，用斗气不断压制封印，可小医仙体内的封印却削弱的更加厉害。

<br/>　　旋即，二星与三星斗宗的气息自两名老者身上席卷而出，早已经封锁住了彩鳞的去路。
<br/>　　瞥了一眼学院长老，彩鳞柳眉一皱，道：“老东西，该你出手了吧？”
<br/>　　“哎……真是麻烦，就不能让本尊好好修养吗？”
<br/>　　苍老的声音凭空响起，只见戴在小医仙纤细手指上的雪白色纳戒一亮，一股无与伦比的威压倾泻而出。
<br/>　　站在最后面的大长老苏千察觉到凝固的空间，瞳孔一缩，惊呼道：“斗尊！”

<br/>　　此时，周围的学员们早已呆若木鸡，小心肝吓的如敲鼓一样，在斗尊的威压下，仿佛生命都不属于他们自己。
