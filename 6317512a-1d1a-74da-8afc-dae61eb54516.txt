        &lt;h3&gt;第三卷第二百零二章  爱心奶爸郭力&lt;/h3&gt;
<br/>
<br/>        陈子昂和宁雅在机场汇合，看到了憔悴的宁雅，吓了一跳。
<br/>
<br/>        “你怎么回事？去了一趟韩国竟然如此憔悴？”
<br/>
<br/>        “昨晚上和你打电话结束之后，趴桌子上睡了一夜。”宁雅的嗓音有些沙哑。
<br/>
<br/>        “一看就是醉酒的后果，你说你都多大的人了，还不知道照顾好自己？”陈子昂语气里有些无奈。
<br/>
<br/>        宁雅才缓过神不久，这就又要开始醉酒了？
<br/>
<br/>        “陈子昂，你一个人过的时候不孤独吗？”在飞机上宁雅问陈子昂。
<br/>
<br/>        “寂寞与孤独应该是与生俱来的，没有人能躲开他们，每个人活着的目的不尽相同，所感知的情怀也不同，我寂寞、我孤独，我享受寂寞和孤独带给我的宁静，带给我的忧伤，带给我的思考。”陈子昂看着宁雅说。
<br/>
<br/>        曾经在夜深人静的时候她被那种孤独的，空旷的、漫无边际的寂寞所环绕，她不知道身处何地，在无尽的黑暗中挣扎着，伸手是无穷无尽的没有边缘的空洞，她一次又一次的与那种可怕的感觉抗争。终于她战胜了自己内心的恐惧，能勇敢面对自己内心，在孤独来临的时候，放空自己，接纳围绕在自己身边的一切。
<br/>
<br/>        “你要知道你此生的目标，像我的目标就是让我的家人安宁幸福，我要努力的成长，再大一点的目标就是让我的产品能造福一方，让很多的人能用得起，我们有能力来推进我们这个时代的发展。”陈子昂看着外面移动着的白云，眼里是希冀是渴望。
<br/>
<br/>        是啊，每个人活着的目的都不同，她只想活得安宁平静，身边的人都要很幸福，在这样的基础上做一点对社会有意义的事情就好。
<br/>
<br/>        宁雅看着陈子昂，她的眼神里有一丝与她的年龄不相匹配的睿智和沧桑
<br/>
<br/>        “有谁能够像你这样将事情看得那么透彻，能明白自己想要什么呢？”宁雅看着陈子昂的侧脸，她的鼻翼微微的动着，嘴巴紧紧的闭着，下巴倔强的微微上翘。
<br/>
<br/>        他们年龄相仿，也许是经历不同，出生环境不同，陈子昂深深的吸引着她，给了她积极向上的动力。
<br/>
<br/>        “你现在不是已经知道你想要什么了吗？”陈子昂转过脸看住她，眼里是笑意。
<br/>
<br/>        宁雅微微的叹了口气，她现在正在向她想要的生活努力着。
<br/>
<br/>        陈子昂见宁雅不说话，便转变了话题：“你和郭力联系好了吧？”
<br/>
<br/>        “已经联系好了，说实在的，我都没有想到请郭力来我们公司，还是江总提醒我的。”宁雅这几年其实很少和郭力联系。
<br/>
<br/>        “俞轩交际广泛，认识的人都是人才，而且他知人善用，有他推荐一定是错不了的。”
<br/>
<br/>        “郭力在金融界也是奇才，只是最近才添了小孩估计不能来公司任职，爱老婆爱儿子，就是一个奶爸，爱子如命，自己带孩子放弃了很多。”宁雅笑着说。
<br/>
<br/>        “一个对家庭负责的人，必定是个有担当的人，我相信郭力的为人。”陈子昂是用人不疑疑人不用，在她人生信条里就是，诚心待人，人心如何黑暗，总是会有良善的一面。
<br/>
<br/>        “陈子昂，你都没有看见人，你就相信一个人？”
<br/>
<br/>        “其实，人和人在交往中，从平时的谈话中，你就知道一个人的品性如何，郭力倒是和我有一点相同之处，不习惯麻烦别人，也不喜欢被人道德绑架，能聊得来就能聊，聊不来就不勉强。”陈子昂笑着说。
<br/>
<br/>        “哇，你们都已经开始聊天了？”
<br/>
<br/>        “是啊，是俞轩的原因，俞轩给我推过他的qq，所以我就加上了，因为后期工作的原因聊过天，感觉人品不错。也知道他的情况，说真的，他的那个宝贝儿子还真是可爱。”
<br/>
<br/>        “你见过他的小宝贝吗？没想到我们陈大总裁被小宝贝圈粉了。”宁雅笑着，她还是单身，怎么能理解得了陈子昂那泛滥的母爱情怀。
<br/>
<br/>        “我在他朋友圈见过小宝贝的照片，小家伙胖呼呼的小手如藕节葱段，尤其那个笑容，看了你心底都是温暖，我好喜欢，我一定要抱抱小宝贝。”陈子昂微笑着，想着郭力朋友圈那个可爱小宝贝，嘴角就抑制不住的上翘。
<br/>
<br/>        “你这花痴的样子我还是第一次见，陈大总裁，没有想到你是这样的人，啊啊啊啊，我看错你了。”
<br/>
<br/>        “我本来就是这样啊，对美好我总是充满渴望，喜欢光明喜欢温暖，因为小孩子纯真，你看到那个笑容，你会觉得，这个世界会如此美好，美好的让你会忘记一切烦恼。”
<br/>
<br/>        杭州郭力家。
<br/>
<br/>        郭力正在电脑上研究着股市，他给几个客户打理着股票，这几年的股市不是很好，总是给人很多意外，但是因为郭力有着明锐的观察力和分析能力，他手中的股票都是稳赚不赔的，而且股票也就是那么几个小时，在家里带着孩子上着班，对他来说还是比较惬意的，孩子才几个月大，吃了睡，睡饱了逗逗孩子，孩子就开心的笑，孩子哼哼他也哼哼，父子俩人一天闹腾得不亦乐乎，不过当个奶爸唯一一点就是晚上要起来好多次。
<br/>
<br/>        小宝贝睡醒了，哼哼唧唧的，用两只胖手手揉着眼睛，萌态可掬，郭力看着小宝贝：“嘟嘟嘟，小栖桐醒咯。”
<br/>
<br/>        宝贝看着爸爸，吧唧着小嘴，还是哼哼唧唧的，似乎在说：“你家小宝贝睡好了，要抱抱。”
<br/>
<br/>        “宝贝，等会，爸爸给你准备吃的哦。”
<br/>
<br/>        小孩子睡醒了就吃，吃了要你陪着他玩，玩够了就会再接着吃在接着睡，孩子还小，不会说话，只能哼哼唧唧的表示自己想要什么，开始郭力并不清楚，随着孩子越来越大，他已经掌握到了一套方法。
<br/>
<br/>        当陈子昂到达郭力的家里时，郭力正拖着拖鞋，穿着睡衣做着一个合格的奶爸，他并没有想到陈子昂会亲自来。
<br/>
<br/>        陈子昂看着郭力怀里的宝贝：“让我抱抱？”
<br/>
<br/>        郭力：“你还真来了。”
<br/>
<br/>        陈子昂：“我就是因为要抱他才来的。”
<br/>
<br/>        郭力将孩子递给陈子昂，陈子昂抱着就不愿意撒手，看得宁雅的眼睛都惊得快掉下来了。
<br/>
<br/>        陈子昂看着宁雅那夸张的表情对郭力说：“郭力，你是怎么带孩子的？将孩子带得这么好？讲讲经验让宁雅学习学习。”
<br/>
<br/>        郭力：“小孩子没有别的事，哭了就看看是不是拉粑粑了，没有就喂奶，如果不吃奶就是不饿，就抱着哄哄，现在白天睡两次，一次3个小时，晚上睡10个小时，如果吃米粉的小孩就不需要经常换尿不shi了，更方便，但是我不喂米粉，米粉太便宜了，70块的米粉能吃一个礼拜，我感觉不靠谱。”
<br/>
<br/>        陈子昂：“那你现在给宝宝吃什么？”
<br/>
<br/>        郭力：“奶粉，香蕉，蛇果，维生素d，补铁滴剂，每天出去晒太阳，然后就陪宝宝玩，哄睡，喂奶就完了，白天睡玩，晚上睡，白天吃完晚上吃，哈哈哈，注意不要让小孩生病，就挺好带的。”
<br/>
<br/>        宁雅真的是惊的眼珠子都掉下来了：“不是，你们不是来谈工作的吗？怎么就谈起了怎么养小孩了？而且你们很熟吗？”
<br/>
<br/>        陈子昂：“我今天不谈工作，我就是来抱一抱小栖桐的。”
