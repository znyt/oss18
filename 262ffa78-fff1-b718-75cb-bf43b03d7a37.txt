
    凉绣一字一顿的说道“我要八字借命中的那一页生死薄，连山还有归藏”<br/>　　<br/>　　    凉绣狮子大开口，一张嘴就把王惊蛰身上最重要的三件东西都给囊获了进来，她可谓是一下手就往死里整了。<br/>　　<br/>　　    王惊蛰皱眉说道“你疯了是不是八字借命我倒是懂，那一页生死薄我给你偷去啊”<br/>　　<br/>　　    凉绣顿时冷笑着说道“王惊蛰你能瞒得住韩观山，糊弄的了余生婆，但你未必能从我这蒙混过关，当初你从墓崖村去了八拐里坡一呆几天都没出来，要么是人死在了里面，要么就是你在里面有什么遭遇导致你能时隔多天后才能出来，显然我更倾向于后者，自打你跳进澜沧江消失之后，我可始终都一直在惦记着你呢，我就觉得你在八拐里坡中恐怕是要有些故事的，那一页生死薄你未必没得逞，只是给私吞罢了，所以我哪里都没查，就专门查了下你为何能在八拐里坡中一呆多日都没事，这还真是让我给摸出一点线索出来，我觉得十有**是靠谱的”<br/>　　<br/>　　    王惊蛰淡淡的问道“哦你说来我听听看”<br/>　　<br/>　　    凉绣撩了下额前秀发，宛若在世女诸葛一样，一张嘴就分析的让王惊蛰都惊叹不已了。<br/>　　<br/>　　    “我就想啊，这八拐里坡谁进去都出不来，那里留下了一地的白骨，怎么你就那么特殊呢估计是里面有什么在护着你的，你王家人丁稀少到这一辈也没剩下什么人了，你姐姐和你父亲自然不会在里面，但有个叫王风饶的赊刀人却在十几年前消失了很久，无影无踪没有可查的迹象，于是，我就派遣三人进了八百里坡”凉绣幽幽的叹了口气，撩了下额前的长发徐徐的说道“六几年前，七六年后一直都有王风饶的踪影，我遍寻打听过后，差不多将他的画像给描绘出了六七成左右的相似度，我用了三条人命的代价，换回来了一个消息，那就是八拐里坡中有一个人，居然跟画像中的王风饶长得有几分相似”<br/>　　<br/>　　    凉绣一番话，说的王惊蛰心里瞬间拔凉拔凉的，这女人的思路缜密到了严丝合缝的地步，居然考虑的滴水不漏。<br/>　　<br/>　　    王惊蛰面无表情的说道“扯淡，那三个人疯了不成，知道去八拐里坡必死，还一头扎了进去”<br/>　　<br/>　　    凉绣淡淡的说道“每一代的墨家巨子身边，都会云集一些墨家死士，这些死士的首要准则和活着的唯一目的就是听从墨家巨子的命令，哪怕让他们去死，不然巨子屡次遇险最后都能安然脱身，你以为靠的是什么这三个墨家死士进入到八拐里坡后陆续身死，但最后一人临死前见到了王风饶，他死后去了阴曹地府，我们又用通阴的方式又和他接触上了”<br/>　　<br/>　　    凉绣一番话说的王惊蛰一阵头皮发麻，这女人真他么阴损到了几点，为了验证一个可能并不存在的消息，居然眼睛都不眨的就牺牲了三个墨家死士，但你也不得不佩服他，就是凉绣的逻辑推理和判断，简直是到了令人发指的地步。<br/>　　<br/>　　    凉绣眯眯着眼睛，盯着脸色瞬间变换了几次的王惊蛰，说道“想必我是说对了吧王惊蛰，你真是让我挺意外的，多少人都没能从八拐里坡出来，你却偏偏可以，不过这样也好，至少证明你确确实实掌握了那一页生死薄，拿来换吧，把你家的小草换回去还有传闻中你懂得的连山和归藏”<br/>　　<br/>　　    王惊蛰叹了口气，说道“就算你说的都是真的，可你也太贪了吧生死薄，连山，归藏任一一样别人都未必能拿得到，你却想一下子都给受了，我是该夸你想的真好呢，还是说你太天真了”<br/>　　<br/>　　    凉绣咯咯一笑，回身指着小草说道“你是说她不值这个价咯”<br/>　　<br/>　　    王惊蛰略一皱眉，摇头说道“衡量不了，她比这些东西重要的多，根本没个比”<br/>　　<br/>　　    “那不就得了，你把东西交给我，我把茅小草交给你，很简单的事情吗”<br/>　　<br/>　　    “但是我觉得，我如果不交的话，我也可以带她走啊，凉绣你该不会真的天真的以为，我独上唐梁山是白白给你送礼来的吧你爹妈是不是把你宠的太无知了”<br/>　　<br/>　　    凉绣轻轻的跺了下脚，说道“这山上是历代墨家巨子隐居的地方，整座山头在不知多少代的巨子经营下，早就已经成为了一座机关术数的大阵，王惊蛰我料你有几分本事，兴许可以费点力气逃出这唐梁山，但你带在带上一个女人的话，就绝对无法闯出墨家的机关大阵了，不信你就试试看，是一阵要一命呢，还是一阵要两命，你想试，我可以奉陪”<br/>　　<br/>　　    这时候，旁边一直默不作声的小草忽然悄悄的往后退了两步和本来并排站着的凉绣错开了一点距离。<br/>　　<br/>　　    凉绣不以为意，王惊蛰的眼神却随着小草后退瞥了过去，小草张了张嘴唇，吐出几个字，王惊蛰一撇之下，读懂了她的嘴型。<br/>　　<br/>　　    王惊蛰瞬间收回眼神，不漏一点痕迹，他皱眉说道“凉绣，你这个要求我肯定是答应不了的，不过我有个提议，咱俩似乎可以商榷一下”<br/>　　<br/>　　    凉绣顿时一愣，有些不解的说道“你和我还有商榷的可能”
