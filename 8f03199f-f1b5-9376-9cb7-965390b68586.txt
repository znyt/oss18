　　警告布鲁斯韦恩后，苏重就不再关注对方动向。
<br/>　　不知道布鲁斯作何想，但之后确实没再找过苏重。即便到科技应用部打造蝙蝠套装，也表现如常。正常的点头问好，就像两人从来不认识。
<br/>　　苏重也乐得如此，他沉迷科技应用部资料，根本无暇他顾。
<br/>　　“苏，你还在研究你那个超级合金？”福克斯一早来上班，却发现应用部内灯火通明。
<br/>　　他对此见怪不怪，自从苏重迷恋上超级合金制造，已多次彻夜不归。
<br/>　　“嗯，快了，马上就要成功啦。”苏重双眼紧盯着仪器上的数据变化，头也不抬的道。
<br/>　　“年轻人，可要注意身体，不然到了我这个年纪，有你后悔的时候。这是早饭，先吃了在干活吧。”福克斯摇摇头，递给苏重一份简易快餐。
<br/>　　“谢了！”苏重伸手拿过纸包，看也不看，拿出里面的三明治就开吃。
<br/>　　福克斯摇摇头不再劝阻，这种状态，苏重已经持续了快半个月，他知道劝也没用。
<br/>　　半个月前，苏重浏览应用部资料时，发现一份合金研究报告。
<br/>　　这是一份失败的研究报告，本是想研究一种超级合金。硬度和韧性极其出色，而且质量轻盈。一旦研究成功，在航天航空等领域，将发挥极其重要作用。
<br/>　　可惜，由于研究周期太长，整个小组一直没拿出说得过去的产品，集团便取缔该项目。
<br/>　　苏重本只是随便浏览，打算长些见识。可等他看进去之后，却意外发现，这个项目实际上已到成功边缘。
<br/>　　几个失败实验成品，虽有这样那样的缺陷，却一直走在成功的道路上。只要解决最后几个问题，立刻就能完成最终成品。
<br/>　　苏重当即来了兴趣。他看过项目组对该合金的性能预测。其中各项数据，让他大为眼热。
<br/>　　“这可绝好的兵器铸料！”在航天领域到底有多大作用，苏重不知道。但他知道，这种超级合金如果制造兵器，绝对坚不可摧！
<br/>　　他最早修炼的可是夺命剑，对兵刃有着独特的情感。有一柄独属自己的神兵，想想就激动！
<br/>　　苏重当即就决定，他要把超级合金给炼出来！
<br/>　　做出这决定并非莽撞。首先，整个项目已到成功边缘。之前道路已铺好。苏重不需从头开始，不需各种实验去确定方向。
<br/>　　再者，由于猎人世界糜稽揍敌客原因，他本身技术水平足够，有足够能力接手这份实验。
<br/>　　如果他没这个能力，也看不出这份研究的进度。而且他还把该项目，交给福克斯把关并得到确认，超级合金确实已到成功边缘。
<br/>　　正因此，苏重才决定涉足其中。
<br/>　　最后一点，这里是科技应用部。有一大票的先进仪器，完全能支撑苏重研究。下定决心，他立刻动手。
<br/>　　苏重并未盲目开始试验。应用部是韦恩集团边缘部门。这里有很多先进机器，但材料却都是些边角料，它们无法让苏重肆意挥霍。
<br/>　　而苏重性格，也让他不会如此鲁莽行事。他总是多方计算，等有足够把握，才会切实行动。
<br/>　　就像当初他创造功夫，总是借助破界珠模拟无数遍，尽可能找出纰漏后，才会修炼。
<br/>　　现在同样如此。他花了三天时间，仔细研究足有十个大箱子的资料。里面有着详尽的数据。一步步见证超级合金的研究进度。
<br/>　　苏重一点点的推敲，吃透一部分数据后，才会进入下一部分。
<br/>　　他灵魂强大，身体也在锻炼下气血旺盛。大脑不知不觉中强化，且前身本就是天才。这才让他在三天内，完全吃透研究报告，并整理出三分改进方案。这是原项目组，对之后试验的推演。
<br/>　　然后他又花了三天时间，仔细推敲这三分方案。彻底否决一项，并对其中一项进行改进，又结合自身认知，提出两个新方案。
<br/>　　毫不客气的说，经过这六天不眠不休研究。就是当初那个研究小组，也没他对超级合金的理解深刻。
<br/>　　然后他又花费两天时间检查，确认再也找不到任何理论缺陷后，这才开始着手试验。
<br/>　　第一天，苏重进行了三次试验。却都以失败告终。仔细研究整个试验过程，他确定第一、第三项方案不可行。
<br/>　　第二天，苏重做了五次试验，同样失败。他停下来不再继续。反而对八次试验数据，挨个分析。找出其中变化，分析原因，并作多方面论证。
<br/>　　第三天苏重又做了三次试验，再次失败。制造出来的合金，不是太脆，就是太柔软。根本达不到要求。
<br/>　　但苏重却毫不沮丧，反而性质高傲。这三次试验的失败，再次给他指明道路。他已经确定了最终试验方案。
<br/>　　就这样一步步前进，直到今天。这最后一次试验，是他不断论证，得出的最终结果。
<br/>　　盯着屏幕上的数据，苏重饭也不吃了，眼睛一眨都不眨。
<br/>　　叮！
<br/>　　一声脆响，苏重猛然离开座椅。快步走到一个庞大长方形仪器前。透过外壳上透明窗口，一片亮黄色液体在仪器内缓缓流动。
<br/>　　“成啦？”福克斯也凑到旁边，满脸惊讶。
<br/>　　即便超级合金项目，已到成功尽头。但有时候最后一步，甚至会困扰研究数十年。苏重竟在半月内，研究出超级合金！他不得不惊讶。
<br/>　　“性能如何？”福克斯仍有些难以置信。
<br/>　　“不知道，应该不差。不过这金属也有缺陷。”苏重盯着缓慢流动液体，头也不回道。
<br/>　　“什么缺陷？”福克斯好奇道。
<br/>　　“他只能一次成型，难以再塑形。或者说，因为他的超强性能，寻常方法，很难让他重新塑形。”苏重控制仪器，让高温液体流入事先准备的模具内。
<br/>　　“如果性能达到预期，这点儿缺陷根本就不是缺陷！反而是优点，这会让超级合金的使用寿命变得超级长！”
<br/>　　“当然，不然怎么叫超级合金！”苏重回头笑道。打开仪器外壳，一股热浪扑面而来。迫不及待，苏重操控机械臂，将模具拿出来。
<br/>　　打开模具，当啷一声，一把通体漆黑色的短刀出现在眼前。
<br/>　　短刀只有成人前臂长短，刀背刀刃全都是直的，如果不是有个圆弧状刀尖，整个就是一长方形铁片子！
<br/>　　刀背厚度三厘米，刀身宽度在两指左右。短刀尾部，有一段明显比刀身窄，且有两个圆孔，是为方便之后配置刀柄。
<br/>　　“终于成了。”苏重脸上露出笑容。这几天，他可是吃住都在这里。
<br/>　　……
<br/>　　就在苏重潜心玩科技的时候。布鲁斯·韦恩彻底完成蜕变，化身哥谭蝙蝠侠。开始纵横黑夜打击罪恶。
<br/>　　他并不是孤军奋战。多番打探，布鲁斯最后找到戈登探长，这是一位有着自己底线的警察。
<br/>　　一切准备就绪，布鲁斯开始打击本地黑色组织。成瘾的药物排在第一序列。而作为哥谭地头蛇的法尔克尼，成了他的首要目标。
<br/>　　多次调查，布鲁斯果断触动。在港口一举擒获大批药物，并当场将法尔克尼抓住，牢牢绑在现场。
<br/>　　让他和贩卖违禁药品活动，直接绑在一起。铁证如山，即便法尔克尼在地下世界手眼通天，也不得不进局子！
<br/>　　“你必须把我弄出去？我知道你们的全部。不仅如此，我还要分成。这里是我的地盘，没我你们办不成事！”法尔克尼手上绑着手铐，一脸的烦躁嚣张。
<br/>　　他对面坐着一位医生，一位精神病领域的医生乔纳森·克莱尔。可实际上，这位是个超级罪犯。未来蝙蝠侠大敌之一——稻草人。
<br/>　　此刻，他正在为影武者联盟研制恐怖药剂。他需要法尔克尼帮他，把药品送入哥谭。这利用了法尔克尼的毒品线路，神不知鬼不觉。
<br/>　　但偏偏蝙蝠侠出现，猛然打乱了计划。好在药剂已全部运送完毕，并全部倾倒入哥谭水循环系统。
<br/>　　唯一的纰漏，就是眼前法尔克尼。他虽不知道药剂是什么，但却知道药剂去向。
<br/>　　现在已是计划最后阶段，任何纰漏都不允许出现！
<br/>　　“你想让我给你开具精神病鉴定书，还想参与我们的计划？”乔纳森温和笑道。
<br/>　　法尔克尼陡然感觉到一阵寒意，却忍不住愤怒：“你到底在搞什么鬼！我当然是这个意思！你们的所有行动，可都在我眼皮子底下。不想搞砸，就赶快把我弄出去！”
<br/>　　乔纳森丝毫不着恼：“好的，我会给你开具精神病鉴定书。不过在之前，请先看一下我的面具。”
<br/>　　他说着代开皮箱，拿出一个麻布片拼接在一起的丑陋面具。
<br/>　　“你可能不知道，那些我的那些病人称呼它为稻草人。”他饶有兴趣盯着法尔克尼。
<br/>　　法尔克尼浑身汗毛倒数：“你特么到底在搞什么鬼！”
<br/>　　噗！
<br/>　　一阵白色烟雾喷洒，正中法尔克尼。高剂量恐怖药剂，瞬间被法尔克尼吸入肺部。
<br/>　　听觉、视觉瞬间被扭曲，神经好似被人大力挑动，整个人顷刻陷入失控状态。看着眼前恐怖稻草人面具，不受控制的大喊大叫起来。
<br/>　　足足过了十分钟，乔纳森把法尔克尼戏弄的彻底精神崩溃，这才走出审讯室。脸上带着淡然微笑，对等在门外的检方人员道：“他确实有精神问题，至少目前是。”
<br/>　　然后转身潇洒离开。
<br/>　　坐进汽车，乔纳森拿出一个卫星电话，拨通一个毫不起眼的号码：“有一个奇装异服的家伙捣乱，法尔克尼已经栽了。我该怎么办。”
<br/>　　“我会调查。”短短四个字，然后对方果断挂掉电话。如果布鲁斯在这里，一定会听出来。这人不是别人，正式亨利·杜卡！
<br/>　　“布鲁斯，是你吗？肯定是你！”世界某处，一个隐秘基地内。杜卡放下卫星电话，眯着眼睛喃喃自语。只是通过只言片语，他就确定了布鲁斯的身份。
<br/>　　布鲁斯·韦恩可是他一手训练，对其形式风格非常熟悉，第一时间就确定所谓的奇装异服家伙的身份：“你毁了我的基地，就不要怪我烧了你的家！”
<br/>　　“去哥谭，查一查布鲁斯·韦恩，看他到最近在干什么。”
<br/>　　“是！”
