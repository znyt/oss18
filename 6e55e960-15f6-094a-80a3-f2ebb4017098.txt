　　“你叫什么名字？”
<br/>　　在前往魔皇城的路上，一辆宽大的马车之中，叶尘闲来无事，和罗刹城小公主闲聊起来。
<br/>　　“纳兰柔然……”
<br/>　　此时的十七公主，早已经没有了之前的娇蛮，变得乖巧无比。
<br/>　　纳兰百川派了一支十几人的队伍，随叶尘三人一同前行，沿途一切都由他们安排的妥妥当当，倒也十分省事。
<br/>　　随着和叶尘聊得越来越多，纳兰柔然或许也发现，叶尘性格温和，并没有那么恐怖，而且她毕竟年轻尚轻，胆子也开始渐渐大了起来。
<br/>　　“叶前辈，您真的是冲霄境强者吗？”
<br/>　　“算是吧！”
<br/>　　“那您怎么会这么年轻？”
<br/>　　“因为我是天才！”
<br/>　　“且！我猜不信呢，别以为我不知道，很多强者都有驻颜之术，我猜您至少也有一百多岁了吧？”
<br/>　　“嗯，你说是就是吧！”
<br/>　　……
<br/>　　叶尘有一句没一句的，和纳兰柔然随口聊着天，马车被四头灵兽拉着，飞驰前进，速度堪比外界的火车。
<br/>　　三天之后，马车的速度终于缓慢了下来，
<br/>　　“叶前辈，魔皇城马上就要到了！”
<br/>　　戴安娜掀开车帘，观察了片刻之后，冲着叶尘汇报道。
<br/>　　叶尘点了点头，
<br/>　　“进城之后，立即调查阿里津的行踪，还有那万年玄冰的下落！”
<br/>　　“是！”
<br/>　　戴安娜自然不敢违背，连忙点头称是。
<br/>　　不一会，
<br/>　　马车来到城门前，
<br/>　　这是一座通体漆黑色的城门，高度达到了四五十米，宽度更是超过百米，比之前罗刹城还要气派许多。
<br/>　　这便是魔界第一城，魔皇城！
<br/>　　戴安娜向城门守卫，出示了一个类似于通行证之类的东西后，马车便缓缓进入了城中。
<br/>　　如果说罗刹城，是冰冷的寒带，魔皇城应该属于亚热带，温度明显要炎热一些。
<br/>　　叶尘通过车帘，朝外面的街道望去，
<br/>　　只见那数百米宽的街道之上，各种各样的魔族之人，或高大魁梧，或矮胖精悍，形形色色，不一而足。
<br/>　　见识过魔界和妖界之后，叶尘也越发确信，地球和修真界之间，必然存在着某种联系。
<br/>　　地球上的魔族和妖族，应该全都是来自修真界。
<br/>　　只不过，这一切应该都要追溯到数千年前。
<br/>　　至于他们是如何来到地球的，那就不得而知了。
<br/>　　心里正暗暗寻思着，马车忽然停了下来，旁边应该是一处酒馆。
<br/>　　“叶前辈，请您先在酒馆休息片刻，等我查清楚了事情之后，再回来向您汇报！”
<br/>　　“好！”
<br/>　　叶尘点了点头，直接走下马车。
<br/>　　很快，
<br/>　　戴安娜将马车安顿好之后，便带着几名侍卫，前去打探消息去了。
<br/>　　叶尘和纳兰柔然，以及剩下的几名侍卫，则进入了酒馆之中。
<br/>　　“掌柜的，给我们开一间天字号包厢！”
<br/>　　进入酒馆之后，纳兰柔然直接豪气十足道，顿时引得不少人纷纷侧目而视。
<br/>　　一名身穿锦衣的中年男子，连忙迎了上来，陪着笑脸道：
<br/>　　“尊敬的客人，实在不好意思，天字号包厢已经全都满了，您看，换成地字号包厢如何？”
<br/>　　纳兰柔然顿时一滞，脸上浮现出一抹不悦之色。
<br/>　　若是在罗刹城，凭她十七公主的身份，酒馆老板就算是把其他客人赶走，也必定会给她找出一间最好的包厢来。
<br/>　　可是她也清楚，这里毕竟不是罗刹城，而是强者如云的魔皇城，她自然也就没有那样的特权。
<br/>　　“好吧，地字号就地字号吧，那就给我开两间！”
<br/>　　“好嘞！诸位楼上请！”
<br/>　　然而，
<br/>　　就在众人，准备随着那中年男子上楼的时候，又有一拨人从门外走了进来。
<br/>　　“掌柜的，给本公子开一间天字号包厢！”
<br/>　　说话的，是一名模样俊美的魔族青年，身后还跟着十多名身穿金甲的随从，可谓是派头十足。
<br/>　　让纳兰柔然等人不解的是，之前还说天字号包厢已经没有的酒馆老板，竟然将他们直接晾在一旁，然后屁颠屁颠的朝那俊美青年，一路小跑了过去，
<br/>　　“哎呀！是赫连公子到了，快快里面请！”
<br/>　　纳兰柔然一向骄纵，何曾受过如此冷遇，顿时便忍不住了，
<br/>　　“老板，你刚刚不还说，天字号包厢都满了吗？”
<br/>　　“还有，明明是我们先来的，你凭什么先给他们安排包厢？”
<br/>　　那酒馆掌柜顿时眉头一皱，虽然依旧陪着笑脸，不过却透着一抹鄙夷之色，
<br/>　　“小姑娘，您看清楚了，这位可是大名鼎鼎的赫连公子！他在我们酒馆，可是有专人包厢的！”
<br/>　　纳兰柔然自然也听出了，酒馆掌柜的鄙视之意，顿时也一下子来了脾气，
<br/>　　“我管他是什么狗屁公子！明明是本公主先来的，你凭什么先给他安排，让你们老板出来，本公主要找他评评理！”
<br/>　　“公主？”
<br/>　　听到纳兰柔然这话，顿时所有人全都吓了一跳。
<br/>　　唯独那名俊美青年，眉头只微微一皱，便重新恢复了淡然之色，轻轻一笑道：
<br/>　　“我魔皇城一共有十三位公主，本公子全部都认识，不知你是哪一位公主，为何本公主没有见过？”
<br/>　　纳兰柔然顿时一愣，旋即美眸一瞪，
<br/>　　“谁说我是魔皇城的公主了？我是罗刹城的公主，纳兰柔然！”
<br/>　　整个酒馆，经过片刻的寂静之后，顿时哄堂大笑，
<br/>　　“我当是哪位公主，原来是罗刹小城的穷公主啊！”
<br/>　　“罗刹城的实力，还不足魔皇城的五十分之一，也就相当于魔皇城的一个卫城而已，也敢自称公主？”
<br/>　　“罗刹城的穷酸公主，跑来到魔皇城来显威风，也是够可以的！”
<br/>　　……
<br/>　　酒馆众人搞清楚纳兰柔然的身份后，顿时纷纷丝毫不加掩饰的嘲讽起来。
<br/>　　纳兰柔然听到众人的嘲讽声，顿时又羞又恼，俏脸憋的通红，其余众侍卫，也全都一脸羞愧之色。
<br/>　　罗刹城在七大主城之中，实力是最弱的一方，而且又地处偏远，向来被其他六大势力看不起。
<br/>　　尤其是在这高手辈出的魔皇城，纳兰柔然这个罗刹城公主，恐怕连魔皇城一个中等家族的千金，都略有不如。
<br/>　　更不要说，和眼前这位赫连公子相提并论了。
