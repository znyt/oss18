　　女尸一看这样，不由的着急起来，说道：“刘司令是什么人，你们怎么不动手？”
<br/>　　三当家说道：“夫人有所不知，这刘司令，是这一代三山五湖的总瓢把子，就算是大当家的，也得喊他一声大哥，没办法，他的本家亲戚，可是不好得罪。”
<br/>　　女尸一听，面色一冷，说道：“就算如此，刘司令又不在这，你们杀了这几人，谁知道是你们杀的。”
<br/>　　三当家听她这么说，瞳孔一缩，叶少阳早防着这个，说道：“实不相瞒，我这次出门，已经跟刘司令通过信，他知道我要打这走，不然我们怎么敢走你们二龙山的道？这附近只有你们一个绺子，我死在这，刘司令追查起来，你们怎么交代，我可是他本家堂弟？”
<br/>　　三当家一听，立刻打消了毁尸灭迹的主意。
<br/>　　其实叶少阳本想说这女尸是刘司令要的，差点就说出来了，灵光一现，想到刘司令那么大的土匪，应该不会干这事，说了更容易引人怀疑，于是没说。
<br/>　　女尸见三当家犹豫，知道是杀不了了，反应也是极快，说道：“既然如此，你把他们绑了上山，发信给刘司令证明一下不就得了。”
<br/>　　哪想叶少阳反应更是快，想也不想接道：“去就去，呆个十天半个月我也不怕，让刘司令派人来接过更好，只怕到时候刘司令会说二龙山坏了道上规矩。”
<br/>　　女尸还想说什么，三当家一摆手，冲叶少阳三人一抱拳（不过是右手包着左手，一般人抱拳是左手盖着右手，因为左手为善，右手为恶，左手包右手，是以善为先，表示善意，土匪却不一样，做的本来就是恶事，右手包左手，也是一个起手礼，说明了自己的身份，懂的人一眼就知道是土匪）：“兄弟误会了，既然是刘司令的本家，那便是自家人，梁上长草节节高，大水不冲龙王庙，这女子与我家大当家的有缘，我留下了，兄弟们请了。”
<br/>　　“抬举！”叶少阳也不敢再讨要女尸，拱了拱手，飞快地收拾了起行李，唤上腿都吓软了的翠云和陈三，也顾不上下雨，对着那些尸体念了起尸咒，赶着出了庙门。临走之前，偷看了那女尸一眼。女尸阴鸷地看着他，嘴角带着一抹诡笑。
<br/>　　三人走出破庙，也顾不上晴天——幸好下雨也没太阳，对尸体没什么太大影响——直接赶着尸体就往前走，走了有十分钟左右。
<br/>　　叶少阳把尸体赶进了附近的林子里，陈三一脸着急忙慌地说道：“叶天师，咱们还不快点走，在这呆着干什么？”
<br/>　　叶少阳一手拉着翠云的手，安抚着她，望着树林外的小路，冷冷说了两个字：“报仇！”
<br/>　　陈三愣住了，说道：“报什么仇？”
<br/>　　“我娘生我，不是让人骂的！”叶少阳咬牙说道，在这个问题上，他有点极端，主要还是自己从小就没妈的缘故，更加的敏感。
<br/>　　以前跟四宝和小马他们在一起开玩笑，带口头禅，那不一样，被人真正指着鼻子骂，对叶少阳来说，还真是第一次，如果说对方是普通人还罢了，是个杀人如麻的土匪，那真的不能忍。之前因为有翠云和在陈三在，否则他早就动手了。
<br/>　　翠云听他这么说，立刻慌神了，“少阳，你别乱来，他们有枪，你斗不过他们的！”
<br/>　　“姐，你放心！我有办法！”叶少阳拉着她的手安慰道，“我什么时候让你失望过。”
<br/>　　翠云一下子就愣住了。的确，虽然自己认识少阳的时间不长，但是他做的哪一件事情，不是出人意料？
<br/>　　说话之间，小路上又想起了马蹄声。
<br/>　　这在叶少阳的预料之中：那个女鬼怕自己，不敢在破庙里多做逗留，一定会让那几个土匪赶紧送她上山，此地是必经之路。
<br/>　　叶少阳靠着一棵树坐下来，说道：“你们不要出声！”
<br/>　　说完闭上眼睛，灵魂瞬间出窍，飞了出去，到了路上，正好赶上几个土匪骑马过来，只有三个土匪，还有那个三当家，一个土匪喜滋滋地背着女尸。
<br/>　　叶少阳看见这一幕，知道这群土匪肯定是分成了两队，一队继续下山去办事，另一队由三当家亲自带队，把这个女尸送给他们的大当家。
<br/>　　一群傻比，捡了个炸弹，还当真宝贝了！
<br/>　　叶少阳飞了过去，魂魄挺进了三当家的身体，一刻也没有停留，直接用自己的魂力，震碎了三当家的三魂七魄，三当家七孔流血，从马上坠落下去，叶少阳本想把几个杀人越货的土匪都杀了，转头一看，那女尸骑在马背上朝自己阴阴地看过来，显然已经发现自己，只是不想离开肉身来对付自己，毕竟想要克制人形煞的煞气，也很不容易，上一次身也很费事。
<br/>　　叶少阳现在也是魂魄状态，杀不了她，只好对她说道：“你也别得意，我不知道你是哪来的鬼东西，三庭之内，我必杀你！”
<br/>　　“我等你来！”女尸恨恨说道。
<br/>　　两人都是用神识说话，别人也听不到。本来三当家就是骑着马走在最后面，前面几个土匪只顾赶路，都没看到三当家的已经挂了。
<br/>　　叶少阳魂魄飞到树林里的时候，才听见那边大呼小叫起来，魂魄急忙回归肉身。翠云和陈三之前见他突然不动了，正不知道怎么办，见他突然醒了，刚要问他怎么了。叶少阳一把抓住他们，说道：“别问了，走！”
<br/>　　当下赶着几具尸体穿过树林，身后传来呵斥声，叶少阳知道是那女尸指使几个土匪来抓自己了，不过他们骑马进不了树林，于是放心下山，也不停留，让陈三带路，一口气绕过二龙山，又疾行了半个钟头，这时候雨也停了，叶少阳提议找了个隐蔽的树林休息。
<br/>　　“叶天师，那个三当家……”一路上三人都跟逃命似的赶路，根本没来得及说一句话，现在总算停顿下来，陈三立刻略带惊悚的问道。
<br/>　　“杀了。”
<br/>　　陈三一滞，“灵魂出窍？”
