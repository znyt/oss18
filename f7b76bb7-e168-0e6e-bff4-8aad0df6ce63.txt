　　七日时间。
<br/>　　楚风眠进入到众圣堂中，已经是过去了七日时间。
<br/>　　在这七日时间之中，楚风眠不断的凝练着他的九域剑术，其中大部分的时间，全部都是用来参悟这剑道古籍中记载的剑术。
<br/>　　这九域剑术的框架，如今楚风眠已经是凝练完成，只要是楚风眠在见识更多的剑术，全部融入其中，便是足以令九域剑术再度进步。
<br/>　　这九域剑术，是由楚风眠创造而出的，随着楚风眠境界实力的提升，也是会变的更加强大。
<br/>　　将来甚至是超越九域历史之中的任何剑术，都有可能。
<br/>　　楚风眠的境界，力量虽然没有任何变化，但是他现在练成了九域剑术，更是令身上的剑意蜕变。
<br/>　　现在的楚风眠，只怕是比起还没有进入到众圣堂前的他，要强大的多。
<br/>　　身上的气质都发生了极大的变化，化为了一种尊贵之际的气息。
<br/>　　现在楚风眠的实力，他自己都不清楚是达到了哪一步，但是面对那太上长老，之前楚风眠心中的想法是只能够全身而退。
<br/>　　而现在，楚风眠的心中所想，在面对着太上长老的话，楚风眠也就未必是没有一战之力。
<br/>　　这剑意提升的实力，可不单单是得到力量那么简单，等到楚风眠的实力越来越强，他的剑意也发挥的越来越强大。
<br/>　　可以说这得到的好处，远远不止眼前所见的如此简单。
<br/>　　现在的楚风眠，在遇到像是不死魔尊这样的人物，只怕是挥手之间，都可以灭杀。
<br/>　　不死魔尊也是一方强者巨头了，可如今在楚风眠的面前，只能算是蝼蚁。
<br/>　　楚风眠的实力就算是还不如那些超越生死境极限，踏入生死阶梯的老古董，但是却也超越了寻常生死境了。
<br/>　　“这种实力，就算是不贸然度过生死大劫，在北邙圣域中，也足以立足了，起码夜天君想要对付我，我也足以跟他对抗！”
<br/>　　楚风眠眼神中露出几分杀意，这夜天君，他迟早要将他踩在脚下。
<br/>　　“走。”
<br/>　　楚风眠大手一挥，这周围的空间，便是在变化这，逐渐的恢复了曾经的样子。
<br/>　　楚风眠也是从这诡异空间之中，从新回到了众圣堂之中，他的身形一动，直接大步离开。
<br/>　　“恩？”
<br/>　　就在楚风眠从诡异空间离开，回到众圣堂的一刻，突然在他的空戒之中，一枚玉符突然闪烁起来，从中得到了一个消息。
<br/>　　“白萱被人抓走了？”
<br/>　　楚风眠的脸色，瞬间阴沉下来，这消息，正是白萱告诉他的，说是他被几名种子弟子合力擒下带走的。
<br/>　　若是想要救下白萱，就必须要让楚风眠前往生死台。
<br/>　　这生死台，便是北邙圣域中种子弟子解决恩怨的地方，上了生死台，生死由天，就算是斩杀对方，学院都不会追究。
<br/>　　这些人抓走白萱，便是为了逼楚风眠，上生死台。
<br/>　　“找死！”
<br/>　　楚风眠眼神寒冷，身上的怒意，真正的动用了出来。
<br/>　　有人对付他，楚风眠倒是不会理会，兵来将挡水来土掩，来一个楚风眠杀一个便是。
<br/>　　但是居然擒下白萱，来威胁他，这简直是触犯了楚风眠心中的规则。
<br/>　　这些人真是不知死活，敢来威胁楚风眠，楚风眠一生之中，最为痛恨的便是他人的威胁。
<br/>　　“很好，我倒是要看看，到底是谁在找死！”
<br/>　　轰隆！
<br/>　　楚风眠的身形猛然一动，直接是化为一道遁光，从众圣堂中，冲了过去。
<br/>　　“好快的遁光！”
<br/>　　“是谁？居然会有这么快的遁光！”
<br/>　　一些众圣堂中的弟子，看到楚风眠的遁光，居然都是一时间无法看清楚楚风眠的身形，一个个惊呼起来。
<br/>　　一些不知死活，想要阻拦楚风眠的人，更是被直接撞飞出去，口中吐血连连。
<br/>　　这道遁光，直接便是离开了众圣堂，向着北邙圣域之中的生死台，飞速飞了过去。
<br/>　　生死台上。
<br/>　　如今便是站着七八个人，这七八个人，都是北邙学院的种子弟子。
<br/>　　其中一尊女子，站在旁边，身上被设下无数禁制，被直接镇压，正是白萱。
<br/>　　而这一群人为首的，却是一尊长相英俊的男子，这男子的一举一动之间，都有着一种极为尊贵的气息。
<br/>　　在他的身边，一股股灵力，凝聚出来的，居然也是像是楚风眠一般的龙鳞。
<br/>　　“龙未央师兄，你说这楚风眠，会来吗？”
<br/>　　一尊种子弟子，对着为首的那男子开口道。
<br/>　　“我们虽然抓了白萱，但是生死台这种地方，可不是谁都敢踏足的，他肯定知道其中危险，这么久还没来，是不是逃了？”
<br/>　　另一尊种子弟子也是开口道。
<br/>　　“楚风眠这小子，毕竟是一个新人，他要是不来，我们怎么办？”
<br/>　　“不来？”
<br/>　　为首的那男子，龙未央冷笑连连道。
<br/>　　“以这小子的性格，他肯定会来的，他要是不来，也不枉我白费力气的去对付他了。”
<br/>　　“龙未央师兄，这小子已经是半个死人了，他得罪了圣党的夜天君，就连王党陆画的弟弟，都被这小子打伤了，我们不动手，这夜天君，陆画也不会放过他的，为何要我们动手？”
<br/>　　一尊种子弟子，有些奇怪的询问道。
<br/>　　“我们龙党，并不需要参与其中吧。”
<br/>　　“哼，这小子身上动用的，显然是龙族武技，他得到过龙族奇遇，其中动用的武技就连我都没有见过！”
<br/>　　龙未央的眼神中露出几分贪婪之色道。
<br/>　　“这武技，只要我可以得到，我就可以击败夜天君，陆画，成为北邙圣域的第一人，倒时候我们龙党，也就会变成北邙圣域中最大的势力！”
<br/>　　“你们看着，这楚风眠这小子来了，我定然会斩杀了他，将他的一切奇遇都得到！”
<br/>　　“是吗？斩杀楚某？楚某现在来了，看你如何能够斩杀了我！”
<br/>　　一道极为冰冷的声音，突然响彻而起，只见一道遁光，飞速落入到了生死台上。
<br/>　　这道人影，正是楚风眠，楚风眠的脸色，冷漠之际。
<br/>　　他的声音之中，虽然冰冷，但是其中蕴含的杀意，却是毫不掩饰，令人听见，就不寒而栗。
