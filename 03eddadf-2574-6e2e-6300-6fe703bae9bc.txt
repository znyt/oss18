　　第15章
<br/>　　韩玄之跌在地上，看着惶惶从自己身边逃走的士兵，他终于开始觉得害怕。
<br/>　　“公子，逃吧！”黄三元扶起他：“大势已去，现在逃还来得及。”
<br/>　　耳边轰隆巨响，乱石纷飞，死去的人就倒在身边，鲜血冲击着视线，令肾上腺素飙升。
<br/>　　“对，逃。”韩玄之站起来：“古来名将，谁不是从一场场失败中成长起来，这磨难打不到我，黄管家，你将见证我的名将之路，这是你的荣光。”
<br/>　　黄三元眨了眨眼睛，目瞪口呆的看着韩玄之。爆炸的火光照亮他的脸，呈现出一张坚毅的面庞。
<br/>　　“我去你大爷的吧！”
<br/>　　黄三元一脚踹在他身上，掉头向外逃，自己怎么就跟着这么个蠢货。你自生自灭去吧，老黄要自己跑路。
<br/>　　所谓兵败如山倒，便是如此，战斗意志一旦被摧毁，每个人想的都是自己的生死，而不是一只队伍的胜负。
<br/>　　峡谷上，程大雷一股脑把炸药箱都丢了下去，最后累得气喘吁吁倒在地上。
<br/>　　“大当家，我们，我们赢了！”徐神机激动的大喊。
<br/>　　“赢了么？”程大雷还有些不敢相信，五天内，每天休息时间不超过四个小时，呕心沥血就是为这一幕，可当这一幕真的发生后，一切显得那么难以置信。
<br/>　　“大当家，快起来。”徐神机喊道。
<br/>　　“不是已经赢了么？”程大雷有气无力道：“能不能让我歇会儿？”
<br/>　　“还能干嘛……抢东西啊……”徐神机大吼。
<br/>　　程大雷还真无法体味徐神机穷日子过惯后的想法，现在山下可都是财富呐。大刀，长枪，士兵身上的盔甲，随身的干粮……还有马匹，没看到秦蛮知道程大雷肯给他一匹马后，口水差点流出来。
<br/>　　程大雷提着大斧头上马，口中高喊一声：“灵儿，擂鼓！”
<br/>　　徐神机套好马车，他驾车，林少羽负责护卫，徐灵儿一身红衣，将战鼓擂得冲天而响。
<br/>　　鼓声像是敲在程大雷心脏上，咚咚咚，胸腔内豪气顿生，连手中的斧头也觉轻了不少，手中大喊一声：“抢啊！”
<br/>　　几个人沿着山坡冲下来，山坡上秦蛮的战斗也快分出胜负。
<br/>　　他虽然是以一敌百，可这些骑兵已经赶路一天一夜，人困马乏，而骑兵头领的死对他们也是重重一击，所以秦蛮在战斗意志和战斗力上都远强过他们。而秦蛮本身，也是罕见的优秀级山贼。
<br/>　　“冲啊，抢啊！”
<br/>　　伴随着咚咚的鼓声，程大雷轮着大斧头冲下来，看到秦蛮以一敌百，竟然不落下风，心中暗暗赞了一声：“好一个百人敌！”
<br/>　　这成了压死骆驼的最后一根稻草，骑兵的战斗意志被彻底摧毁，人人落荒而逃。
<br/>　　“不想死的速速下马，山寨爷爷只抢马，不抢人。”程大雷口中大喊，他是相中这些人的马，要是都能抢过来，可是一笔不小的财富。
<br/>　　徐神机眼前一亮，大当家真是行家呐，他也马上跟着大喊：“下马免死，下马免死。”
<br/>　　这帮骑兵已经没有丝毫反抗的心理，虽然他们的人数远远超过程大雷，只见一下马果然秦蛮就不在动手，人人纷纷落马，向四周逃去。
<br/>　　“军师、秦大哥把马圈好，少羽，随我去看看，帮我抢柄剑……”
<br/>　　人人都有理想，林少羽想做大侠，韩玄之想做名将，程大雷也想白袍仗剑，匹马江湖呐……
<br/>　　……
<br/>　　众山贼隐在山林中，人人忐忑难安，黑石城出兵千员，这是青牛山任何山寨都挡不住的，至于各家联合，更是想也不要想。
<br/>　　正这个时候，轰隆声响起，群贼吓得面面相觑，都不知发生什么事。
<br/>　　还没过多久，就见黑石城的兵丁疯一般的从峡谷内逃出。群贼更加困惑，谁也不知什么情况。
<br/>　　“大哥，你说是怎么会事，凭蛤蟆寨能挡住一千人？”高飞豹。
<br/>　　“我也不懂呐。”高飞虎并不比弟弟明白多少。
<br/>　　高飞豹摇摇头，长叹一声：“世事，还真是叵测得紧呐。”
<br/>　　啪！
<br/>　　高飞虎一巴掌拍在他脑袋上：“一个山贼吓发什么感慨，忘了我们的本行了，抢他娘的！”
<br/>　　“对，对，弟兄们，和我抢他娘的！”
<br/>　　山林中隐的各家山寨都恍然醒悟，不打家劫舍还他娘的叫山贼么。手中兵器，身上干粮，还有身上的盔甲……这就完了么，脚上的靴子，束发的发箍，身上的衣服……你知道山贼过得多穷么，一家人只有一条裤子的事情并不是没有。
<br/>　　飞虎寨，杏花岭，狗头岭，秃头坡……今天大家带的人都不多，一般也就是二三十人左右，可四五十个山寨的人汇聚在一起，那人数就变得恐怖了。
<br/>　　本来黑石城的人已经兵败如山倒，现在这帮山贼又如苍蝇见血一般扑过来。
<br/>　　大刀，铁枪，狼牙棒，甚至锄头，镰刀……反正各种武器都有，这群人就抽风一般冲了过来。
<br/>　　“把武器交出来！”
<br/>　　“老子好几年没换新鞋了，要你的靴子……”
<br/>　　“趴下，脱裤子！”
<br/>　　兵丁欲哭无泪：“大哥，我不好这一口，您要什么我都给，能不能把我的贞操留下。”
<br/>　　“去你大爷的，谁要抢你的贞操，我要你的裤子！”
<br/>　　韩玄之抱头鼠窜，更可怖的是，身后还有一个轮着大斧头的山贼，死死的追赶着自己。
<br/>　　不仅追赶，口中还高喊着什么：“我的剑……我剑……我剑……”
<br/>　　韩玄之欲哭无泪，我知道你很贱，能不能别喊了，这么多人，你特么只追我干什么。
<br/>　　前方马蹄声声，一百匹高头大马载着一百名精壮大汉，人人表情肃穆如同石像。
<br/>　　只有为首一人穿着布衣，腰间戴着一柄短刀。气质真是一个很玄妙的东西，上千人能被程大雷一个人追着跑，可看到这一百人，程大雷立刻翻身下马，躲在草丛中。
<br/>　　“收枪，换刀。”布衣男人冷声道。
<br/>　　唰！
<br/>　　齐刷刷的抽刀声，一百把铁刀握在手中，没有山贼敢靠前。大家都有一种感觉，凭这一百把刀能扫平青牛山任何一股山贼。
<br/>　　韩玄之一见，便如逢救星，他不顾一切的冲过去，口中大呼小叫。
<br/>　　“薛军师救我，薛军师救我，他们欺负我，他们都欺负我。”
<br/>　　铁刀微微举起，在空中斩过一道寒芒，从韩玄之的脖子划过。
<br/>　　他的身体晃了晃，不可思议的看着布袍男人，最后还是一个字没说出口，噗通一声跌倒。
<br/>　　一代名将之路，就此陨落。·
