　　“道友，难道你不想成圣吗？你难道还想看着三清一个个在你之前成圣，将你压在身下？”
<br/>　　轰！
<br/>　　听到此话，女娲立刻怒了，她最不想见到的事情就是，看着三清将自己超越，她总是对自己说，有一天若是自己成圣了，一定要让三清好好看看我的厉害。此时这道人拿三清来与她对比，尽管她知道，这是眼前这人故意这么说，要来刺激她的，但她还是停了下来，要看看他怎么说。
<br/>　　“道友你就不想在老子、通天之前成圣，然后打他们的脸？”那黑衣道人再次诱惑道。
<br/>　　想！我当然想了，我做梦都想？可是，但那又有什么办法呢？女娲心中暗暗道。
<br/>　　“说吧，你想要什么？”女娲面无表情道。
<br/>　　“呵呵。”那道人盯着女娲上下打量了一番，只看得女娲心中发毛，他才一本正经缓缓道：“道友只需答应本尊，在成圣后，为本尊做三件事即可。”
<br/>　　“不可能，若是你想要本宫死，难道本宫还要听你的吗？”木瓦毫不犹豫的便拒绝了。
<br/>　　“呵呵，放心，本尊不会让你去死的。”黑衣道人毫不在意道。
<br/>　　“那也不行，若是你提出违背本人意愿之事，本宫也不愿去做。”女娲犹豫了片刻，说道。
<br/>　　“这道友可以放心，本尊绝不会提出违背道德之事。”那道人如此说道。
<br/>　　“可以。”
<br/>　　女娲犹豫了片刻，肯定的说道。主要是对于成圣这件事，已经快成为她的执念了，再加上这道人刚才的刺激，她才会答应。若是换做平时，她绝对不会如此。这只能说明，女人疯狂起来，还是很可怕的。特别是一个强大的女人。
<br/>　　“好，爽快！道友既然如此爽快，那么本尊也就告诉道友，你如今正缺少的东西。道友欲以造化之道成圣，而这造化之道，又在于创造。盘古开天，道生一，一生二，二生三，三生万物。”
<br/>　　“此处的一，即是道。二为阴阳，至于三嘛，那自然是天地人三才。如今，天地已分，人道未出。道友创造的，自认是能够秉承人道而出的一个种族。”
<br/>　　轰！
<br/>　　听到此处，女娲只感觉自己元神之中的鸿蒙紫气绽放出前所未有的光彩，道韵涌动，天机显现。显然，眼前这道人说对了。只见那道人不停，继续说道：
<br/>　　“道友之前塑造之物，用的是凡土。这一个承载着人道的种族，区区凡土和凡水又怎能创造出来？故而，道友若是想要成道，还需要找洪荒顶级神土与神水。而这些顶级的神土之中，九天息壤最为合适。”
<br/>　　“所以，道友想要成道，缺少的是九天息壤与它同等级神水。”黑衣道人不急不慢，缓缓说道。
<br/>　　“这样么？我早该想到了。”女娲低声喃喃道。
<br/>　　“哈哈，是啊，道友早该想到了，只是之前道友陷入了魔障之中，一时没有反应过来而已。”那道人哈哈一笑道。
<br/>　　“所以，你就乘虚而入？”女娲翻了翻白眼，风情万种般说道。
<br/>　　“呃？好吧，本尊的确是有些占了便宜，道友既然已经答应了，但也不能反悔，须知，如今执掌天下因果天罚之力的，可是玉清圣人啊。”
<br/>　　“放心，本宫既然已经答应了，自然不会反悔。”女娲骄傲地说道。
<br/>　　“嗯，既然道友如此爽快，那本尊也不占道友便宜，便再免费送道友一则消息。”黑衣道人道。
<br/>　　“什么？”
<br/>　　“本尊有三光神水与天一真水，这都是顶级的神水，可以送些给道友，而九天息壤，本尊虽无，但却知道何处有。”
<br/>　　“何处？”女娲问道。
<br/>　　“巫族后土祖巫手中。”
<br/>　　“是她，巫族啊！我为妖，她为巫，我又该如何向她讨要九天息壤。”女娲轻声低语。
<br/>　　“那是你的事，与本尊无关。本尊走也。哈哈哈！”
<br/>　　黑衣道人大笑一声，留下两团神水，正是三光神水与天一真水。他的身影却渐渐化作一道黑雾，顿时消散于空中。只留下一道嚣张的大笑声：
<br/>　　“万劫不灭邪圣仙，万道森罗魔至尊！哈哈！”
<br/>　　女娲：“……”
<br/>　　不再理会其他，女娲收起两团神水，向着后土部落飞去，却是想要与后土换取九天息壤。
<br/>　　……
<br/>　　后土部落中。
<br/>　　“女娲道友，你怎会有空来我巫族部落中游玩？”
<br/>　　一身黄色衣裙，满是端庄大气，漂亮得不像话的祖巫后土看着眼前同样绝色无双的女娲好奇地问道。
<br/>　　巫、妖两族一直都是敌人，相互争斗，但在巫、妖两族没有开战之前，后土对女娲还是蛮有好感的。大家同为洪荒之中为数不多的绝色美女，修为又差不多，她对女娲有点惺惺相惜的感觉。
<br/>　　之后，巫妖大战，两人之间也就没有什么来往了。但此刻女娲竟会来到她的部落，这是怎么回事？对此，后土十分好奇。
<br/>　　“听说后土道友手中有九天息壤，能否将之让与小妹，有什么条件道友尽管提。”女娲也不废话，直接说明来意。
<br/>　　“咦！道友是从何处听到的消息，我手中有九天息壤，知道的人并不多啊。”后土惊讶的问道。
<br/>　　“是从一位道友那里听到的。这九天息壤关系到小妹的圣道，后土道友能否相让？”女娲也不知道那人的名号，所以她只说是一位道友。
<br/>　　但后土显然也没有在此事上太过纠结，而是对着女娲说道：
<br/>　　“这竟然关乎到道友的圣道，按理来说，我也应该让与道友。但，你也知道，我们巫、妖两族是宿敌，若是你成圣了，那对于我巫族来说，可是一个天大的灾难。”
<br/>　　“这……”
<br/>　　女娲显然也知道此事并不会那么容易，但成圣已经是她的执念了，为了成圣，她可以放弃其他的一些东西。犹豫了片刻，女娲对后土说道：
<br/>　　“这样吧，我可以答应道友，只要巫族不来攻打妖族，我也不会出手对付巫族。”
<br/>　　“道友觉得这可能吗？巫、妖两族可能就此罢休吗？”
<br/>　　后土反问道。女娲的意思她十分明白，就是只要巫族不去打妖族，她就不插手。但若是巫族攻打妖族，那么，她绝对会出手。巫族可能不攻打妖族吗？这绝对不可能。那么，也就是说，她一定会出手了。既然这样，我为何还要给你息壤，让你成道？
<br/>　　“这……”
<br/>　　女娲显然也是知道这是不可能的，巫、妖两族就好像是天生的宿敌，不是你死，就是我亡，就算她自己不想参与到两族中的战斗中，但有的时候，天地大势可不是她自己就可以决定的。
<br/>　　“其实，这也不是没有办法？”后土眼珠一转，幽幽道。
<br/>　　“什么办法？”女娲道。
<br/>　　“道友只需答应，成圣之后，不再插手巫妖两族之事。还有，用一件同等级的宝物来换，我就可以将九天息壤让与你。”
<br/>　　后土慢慢说道。她这句话的意思与女娲之前那句话的意思可是完全不同的。她这话中潜藏的意思是，不管以后巫族打不打妖族，女娲都不能插手。
<br/>　　有所求的是她，又不是自己。若是不答应，可以，反正自己又没有损失。这是后土的心思。
<br/>　　“这……”
<br/>　　正如后土心中所想，有所求的是自己，她也想不答应啊，但奈何这关系到自己的圣道，不是那么容易就能放弃的。女娲有些迟疑。后一个条件还好说，但是让她不插手两族之间的战斗，这事就让她有些犹豫了。虽然她也不喜欢争斗，但她也不能眼睁睁看着自己的妖族被巫族灭亡。
<br/>　　后土不急，反正又不是自己成圣，该急的人又不是自己。她静静地看着眼前的女娲，让她慢慢思索，再做决定。
