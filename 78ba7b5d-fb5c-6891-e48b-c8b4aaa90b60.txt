　　第165章
<br/>　　杨龙停，包括杨龙停身后的齐德强、齐德龙、李善言都用异样的眼神看着程大雷。
<br/>　　程大雷怎么了？戎族破国，一日破幽州兵，半日破黑石城，结果是薛半川死，幽州王逃。而这仅仅是个序幕，再之后的山河破碎，苍生如狗并不是很难猜到。
<br/>　　这种时候，许多人都有资格站出来。管制幽州的杨龙停，幽州王座下大将齐德龙，齐德强……他们吃帝国军饷，享受军衔带来的荣耀，于情于理都该站出来，力挽狂澜，拒敌于国门之外。
<br/>　　可他们没有站出来，程大雷站出来了。无怪大家用异样的眼神看着程大雷，你只是个山贼，怎么算也轮不到你。你有什么资格站出来……
<br/>　　除了病没好以外，无法解释。
<br/>　　杨龙停的眼神令程大雷觉得有些心寒，他再次道：“阁下的往事我也听说过，三十年前戎族大敌来犯，是阁下在绝境力挽狂澜。今日的事和当年的事又有什么区别，为何阁下三十年前能做的事，今天就不能做了？”
<br/>　　程大雷也并非为国为民，他没有这个觉悟，蛤蟆寨的山贼更没有。但他知道，逃，并不是生路，反而是条死路。而以蛤蟆寨的坚固，再加上幽州兵的协助，守，并不是不能守住，甚至不是很难守住。
<br/>　　可这种简单的事情，为什么杨龙停却不愿意去做呢。
<br/>　　“你和戎族交过手么，你知道戎族有多强大么，靠你这不足万的乌合之众怎么守。”杨龙停已经有几分气急败坏：“我和他们打过，我知道戎族的骑兵有多强。三十年前我不怕死，现在我已经怕死了好不好。”
<br/>　　程大雷再次无语。
<br/>　　“你让不让开路，再不让开我便让我的士兵杀过去！”杨龙停马上大吼。
<br/>　　今日他不坐轿了，今日他骑马，骑马可以跑得更快些。可当吼出这句话时，那份幽州王的威严又出现了。他还是那个半日破伏虎山，挥师五万，要替古人报仇的幽州王。
<br/>　　可如此强悍一个人，一个军中的老油条，为何面对戎族却望风而逃，连握紧兵器的勇气都没有。
<br/>　　程大雷拦路不动，深吸一口气道：“你是明帝亲封的幽州王，一人之下万人之上，你没有资格逃！”
<br/>　　“你……我真是怕了你了……”幽州王忽然将帅印接下，丢在程大雷面前：“现在你是幽州王了，你想守你守吧，我不守了！”
<br/>　　印上蹲着麒麟，而不是程大雷那枚将军印的蛤蟆。平时，有此印就可以指挥幽州千军万马，杨龙停得到这枚帅印，也定是废了许多心血。
<br/>　　可如今这枚帅印就丢在地上，粘着雨水和污泥。
<br/>　　程大雷弯腰把它从地上捡起，忽然高举在手中，亮给所有人看。
<br/>　　“幽州王军印在此，可有好男儿愿与我放手一战，杀敌建功！”
<br/>　　目光扫过齐德强，对方低下头去。一声喝无人应，二声喝无人应，三声喝无人应……
<br/>　　“喂，走啦，走啦，不要理他。”
<br/>　　杨龙停拍马，马不停蹄从程大雷身边掠过，程大雷的声音在空谷回荡，却无回音。
<br/>　　“大当家，咱们回去吧，也商量商量逃走的事情。”高飞虎道。
<br/>　　“回去，回去，我还有力气没使完呢！”程大雷忽然道：“随我去落叶城。”
<br/>　　过了青牛山，便是落叶城。如果青牛山守不住，落叶城也必定像黑石城那样遭殃，杨龙停不愿意守，程大雷不相信方伯山也不愿意守，毕竟，关乎他的身家性命。
<br/>　　骑得马上，程大雷望了一眼路上的逃兵，忽然喝道：“下他们的兵器和盔甲，想做逃兵可以，把马匹、兵器、盔甲留下，我不许寸铁过我青牛山。”
<br/>　　这帮逃兵毫无战意，就算程大雷下了他们的装备，也没有一人反抗，一心只想着如何逃命。
<br/>　　程大雷引一队头目，快马奔向落叶城。
<br/>　　一路上到处都可以见到惶惶逃命的人，有兵丁，也有平民。消息已经散开了，戎族大兵入侵，血屠黑石城，大家快逃命去吧。拖家带口，富户赶着马车，上面装着粮食、金银珠宝。平民则带着仅有的财物，带着家人逃命。
<br/>　　谁也不想去用自己的脑袋硬碰戎族的钢刀，这样的逃兵，自然不要想谈什么秩序。那些富户平时作威作福，今天我还不抢你。生得稍微有几分姿色的姑娘，美貌便是原罪。
<br/>　　程大雷在马上唏嘘，道：“若我们逃了，也便是这样，连我们自己的命都守不住。码头定已经人满为患，就算逃到大城，有那个城主会收留这些难民，缺衣少粮，不过是饿死一条路而已，而且死得毫无尊严。”
<br/>　　高飞虎等人看着这一幕，何尝不是如此想法。
<br/>　　“大当家，但咱们守不住！”
<br/>　　“只要方伯山借我一万兵，仗着地利之便，我不是守不住！”
<br/>　　快马加鞭，众人已奔向落叶城。落叶城更无秩序，连城门也无人看守，城内借机生事的人数不胜数，程大雷就算想管也管不过来。当然，他也根本不想管。
<br/>　　带着一队人，程大雷直扑城主府。
<br/>　　“快，把我的玉佛装到车上！”
<br/>　　“你们干什么呢，不想活了么，装车装车，那些衣服还要它做什么！”
<br/>　　“你们再哭哭啼啼，我就留下你们，让戎族去糟蹋！”
<br/>　　城主府，方伯山大声呼喝，谁也没想到，这个爱财如命的慵懒胖子竟有如此的劲头。
<br/>　　程大雷闯进城主府，就看到这一幕。
<br/>　　“方城主，你在做什么？”
<br/>　　“程将军，你怎么来了？”方伯山奇怪道：“是想要向我借船么，我可没有多余的船借给你！”
<br/>　　“不是借船，而是借兵。”程大雷把幽州王的帅印举起来，道：“奉幽州王令，见此印如见幽州王，命你方伯山与我共守青牛山。”
<br/>　　方伯山愣了愣：“杨龙停现在干嘛呢？”
<br/>　　程大雷道：“这军印有假么，他就在青牛山坐镇，命我来传令。方大人，只要能守住青牛山，你可是大功一件啊。”
<br/>　　
<br/>(本章完)
