　　十天之后，于阗主城外的一处荒僻沙洲，一队二十余人的队伍带着一身的征尘驻足于一处沙丘之上。
<br/>　　“白卫，我们已经到了。”一个身披斗篷的精壮汉子，远处影影绰绰的于阗主城说道。
<br/>　　“找到‘六组’的标记没有？”白月宁宛如黄莺出谷的声音在这片荒芜的沙海之中，带出一股别样的生机。
<br/>　　精壮汉子听到白月宁的声音之后，明显打了一个哆嗦，然后才沉声说道：“已经安排人手去找了，一会儿就会有回报。”
<br/>　　“安排五个人警戒，其他人休息吧。”白月宁眺望了一眼远处的孤城，骑着马转下了沙丘。
<br/>　　“喏！”汉子答应一声，便按照白月宁的指示去安排手下的兄弟。
<br/>　　自从在莎车城被白月宁狠狠的教训一顿之后，这些在大漠上混迹了好几年的‘獠牙’精锐，终于认识到什么是天外有天，人外有人，对于白月宁也有了一种发自骨子里的恐惧。
<br/>　　一个能在两个‘獠牙’战斗小组的围攻下游刃有余，同时拿刀子捅自己人眼都不眨的女人，绝不是他们这些‘獠牙’精锐能够惹得起的。
<br/>　　如果真的动起手来，这次出来的二十余人怕是不够这女人一个人杀的。
<br/>　　所以虽然一群三年不知肉味的汉子对小白十分感兴趣，但考虑到小白下手的狠辣，最终每一个人都放弃了对这个女人的非份之想。
<br/>　　“队长，找到了。”就在精壮汉子陷入回忆中的时候，远处一骑如飞赶来，在接近到他身边的时候，马上的骑士自马背上飞身而下，微微蹲身缓去冲力之后，沉声说道。
<br/>　　“收拾行装，出发。”精壮汉子从地上跳起来，看着比他起身还快的白月宁，咂了咂嘴命令道。
<br/>　　“喏！”骑士答应一声，牵过已经自己掉头走回来的战马，偏身骑了上去，等看到其余战友都上了马之后，一马当先冲了出去，向着西北方向冲了去出。
<br/>　　“跟上！”精装汉子一声大喝，追着当先离开的骑士而去，而白月宁等人则是跟在了他的身后。
<br/>　　大概有一刻钟左右，前面的骑士在一块风化岩的下面停了下来，而在他的身边则是站了一个将全身都包在斗篷里的黑衣人。
<br/>　　“警戒！”精壮汉子对后面打出一个手势之后，骑着马缓缓向着黑衣人靠了过去。
<br/>　　“你是谁？”到了黑衣人身边，精壮汉子沉声问道。
<br/>　　“你不认识我了？”黑衣人抬起头来问道。
<br/>　　“吴辰？怎么会是你？你不是在……算了，当我没问过。”精壮汉子摇摇头，随后对远住驻足而立的队伍招了招手。
<br/>　　“文轩，你啥时候退役，我的‘六组’可是一直等着你呢。”趁着远处的队伍没有过来的时候，一身黑衣的吴辰拉着精壮汉子话起了家常。
<br/>　　“想都别想，老子在军中干的好好的，退什么役。”被称作文轩的汉子撇撇嘴，有些不屑的说着，看样子是对加入‘第六小组’有些抵触。
<br/>　　“军中有什么好混的，瞅瞅你现在的鸟样，当初如果跟着老子，现在怎么也混个副组长干干了。”吴辰有些不死心的说道。
<br/>　　“老子玩不来你们那一套，还是当我的兵吧。”随着白月宁等人到了地头纷纷下马，被吴辰叫做文轩的汉子摆手止住了话题。
<br/>　　文轩姓伍，名登，文轩只是他的字，擅使一杆长枪，当年曾经帮助过刘黑闼，甚至与皂袍大将尉迟恭力战百余回合。
<br/>　　只是后来老刘实在是运气不好，加上胡乱折腾，恼了伍登，这家伙一气之下反了老刘，单枪匹马远遁他乡。
<br/>　　再后来隐姓埋名的伍登机缘巧合之下认识了吴辰，然后被吴辰拉着投了李承乾，在他的太子右卫率里混了什长。
<br/>　　仗着官职不高，也不怎么引人注意，后来右卫率改了‘獠牙’，紧接着就是与突厥大战，然后又是出兵西域，所以伍登一直就没有在长安待过多长时间，自然也没有让程老货等人认出来。
<br/>　　否则以伍登的本事，如果被老程或者尉迟老货认出，在军中混个十六卫将军还是不难的。
<br/>　　吴辰知道伍登这家伙够义气、有本事，不过却因为当年种种事情，这货有些没有面皮与尉迟恭等人见面，心中颇有为其惋惜，所以一直变着法的想把他给弄出来，让他能被上面看到，有所提拔。
<br/>　　但是这种事情又是好说不好办，吴辰的想法是好的，但是对李承乾来说，多少有些吃里扒外的嫌疑，所以思来想去，吴辰觉得把这个老友弄到自己身边，给他一个高位这样才够意思。
<br/>　　“吴组长，情况怎么样？什么进候能安排我们进城？”白月宁此时已经将马交给了其他人，走到了吴辰与伍登的跟前。
<br/>　　“明天一早。”吴辰无奈的看了一眼伍登，然后转头对白月宁说道：“只是你们需要换一身行头，像你们现在这个打扮肯定进不去。”
<br/>　　“没问题，但是武器怎么办？你能帮我们运进去么？”白月宁没有一丝犹豫，相比于其他人，刺客出身的白月宁一切都以任务为主，只要能够完成任务，其它事情都是小问题。
<br/>　　“可以帮你们运进去，不过，完全任务之后怎么出来就要靠你们自己了，我们‘六组’在城里的人手不多，帮不上什么大忙。”吴辰大致上把白月宁等人进城之后会遇到什么情况简单的说了一下，然后说道：“这次的任务我们六组帮不上什么大忙，你们要自己小心。”
<br/>　　“嗯！”白月宁嗯了一声，再没有多问什么，一个人转到旁边的岩石之下去休息。
<br/>　　‘第六小组’是属于情报组织，战斗力比‘七组’要差上许多，劫走李元昌的任务如果靠‘六组’，其结果并不比让‘六组’全员自杀要好，所以在接到任务的那一刻起，小白就没有依靠‘六组’的打算。
<br/>　　“在城里给我们准备几匹马，能做到么？”看着靠在一边休息的小白，伍登对吴辰问道。
<br/>　　“没问题，不过可没有什么好马！”吴辰点点头，答应了伍登的要求。
<br/>　　反正这帮家伙应该是需要马来出城，只要出了城立刻就会换乘自己的马，所以准备什么样的马已经无所谓了，只要能骑问题就不大。
