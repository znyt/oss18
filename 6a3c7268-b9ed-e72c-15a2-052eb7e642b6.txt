　　第43章
<br/>　　那一天，所有山贼都看见，蛤蟆寨将一车又一车金银珠宝拉回山。
<br/>　　整整十车，这是大家数了好几遍后确认的数字。虽然看不清楚马车上装的是什么，但想也知道定是连城的金银，无价的珍宝。
<br/>　　可是看着蛤蟆寨将这金银财宝拉回山，却没人动抢劫的心思。通过这一战，蛤蟆寨已经让青牛山群贼看到自己的实力，让那些想打蛤蟆寨主意的人偃旗息鼓。而现在程大雷究竟会不会五雷法，已经不重要了。
<br/>　　他们只能看着蛤蟆寨将一车车财宝拉回山，能做的，也不过是在蛤蟆寨打扫战场后，从尸体上扒一些盔甲，拣一些散落在丛林中的兵器。
<br/>　　回到蛤蟆寨后，程大雷第一时间回到房间，查看自己的信息。
<br/>　　姓名：程大雷（小有名气的普通山贼）
<br/>　　年龄：18
<br/>　　技能：三板斧
<br/>　　隐藏属性：一个好人
<br/>　　恐惧值：3000
<br/>　　相比从前等级并没有变化，只是属性多了一条『恐惧值』。查看刚才的系统提示，当自己系统达到10000后，将获得点将台建设权限。
<br/>　　程大雷暗自琢磨着，以他多年游戏宅的经验分析，这恐惧值类似于系统货币，当获得一定数目后，就可以向系统兑换物品。而这点将台……难道是武将商店？
<br/>　　当然，这只是程大雷自己的分析，只有建设点将台后才能确认。
<br/>　　目前自己的恐惧值是3000，距离目标还有7000，那么这恐惧值该如何获得？
<br/>　　正这个时候，徐神机走进屋子：“大当家，咱们抢来的寿礼已经点清数目了，您要不要去看看？”
<br/>　　程大雷跟着徐神机往外走，心里还在琢磨关于恐惧值的事情。这系统明显是专业版的，连一个新手指南都不给，凡事都要程大雷自己琢磨。
<br/>　　“呔！”看着走在前面的徐神机，程大雷忽然大吼一声，一巴掌拍在他肩膀上。
<br/>　　徐神机扭过头来，一脸嫌弃道：“大当家，你干嘛？”
<br/>　　嘟，获得恐惧值1
<br/>　　“没事，没事。”
<br/>　　程大雷挥挥手，心里还在琢磨着，看来这恐惧值，只要让人害怕就行，倒也不算困难。不过徐神机对自己已经很熟悉了，当然是不会害怕自己。自己想要获得恐惧值，还得从其他方面想办法。
<br/>　　和徐神机一起走到校场，黄三元正拿笔记录着数目。他做这些事情是专业的，如果没有黄三元，蛤蟆寨几个人做这些事情都不擅长。
<br/>　　“大当家，已经点清楚了。”黄三元道：“一共十车，兽皮三车，铜器两车，草药两车，绸缎三车。其他咱们还获得了几匹马和一些兵器。”
<br/>　　这场战斗下来，蛤蟆寨收获颇丰。兽皮，草药，棉布都是在这里生活的必须平。程大雷看着面前堆积成一座小山的物资，道：“那些是咱们能用的？”
<br/>　　“大都用得着，但用不了这么多。”黄三元道：“目前还有个问题，山寨没多少粮食了，眼看天气也越来越冷，咱们得储备一些过冬的粮食。”
<br/>　　其实目前的蛤蟆寨也不过六个人，六个人是吃不了多少东西的，关键是山寨还养了几匹马，今天打仗又抓了几匹，消耗更大，凭目前山寨的储备是过不了这个冬天的。
<br/>　　“那是该把这些东西换成粮食？”程大雷抬头道：“咱们好像没有没有门路，对吧？”
<br/>　　徐神机道：“其实以前咱们有销账的门路，只是蛤蟆寨的实力越来越小，这条路也废了。”
<br/>　　“那看来咱们还得再开一条门路，不然以后抢来东西，根本没办法销账。”程大雷说着，忽然站起来：“苏樱那边呢，家里那点破事还没解决，答应给我们的赎金也没送来，真当从我们这里白走了！”
<br/>　　“那边会不会遇到麻烦了？”黄三元问道。
<br/>　　以目前的情况看来，蛤蟆寨和苏樱建立合作关系是最好的，只是苏樱回去后一直没有消息传来，也不知道辣椒在落石城反响怎样。究竟苏樱那边怎么样了，难道她那个后妈很难处理……
<br/>　　程大雷先让大家把这次抢劫来的东西搬进仓库，反正这些东西都坏不了，搁一段时间也没有问题。
<br/>　　在做这些的时候，程大雷不断收到系统提示。
<br/>　　嘟，收到恐惧值8
<br/>　　嘟，收到恐惧值10
<br/>　　嘟，收到恐惧值3
<br/>　　……
<br/>　　程大雷估计，这是自己打劫的事情传回黑石城后，引起的反响，说不得传言会将自己传成怎样。不过，想要靠此收集10000点恐惧值，显然是不够的。
<br/>　　次日天明，程大雷还没睡醒，徐神机就跑进他房间。
<br/>　　“大当家，李行哉跑了。”
<br/>　　“跑了就跑了吧，你以为真还能留下他们呐。”程大雷打了个哈欠坐起来：“估计他们也该走了。”
<br/>　　“不过在他屋子里留下了一把剑。”
<br/>　　咦！
<br/>　　程大雷腾地从炕上跳下来，伸手将徐神机手中的剑抓过来，锵啷一声拔出剑鞘，剑刃寒如水。
<br/>　　程大雷刚来到这个时代，就心心念念想要一柄剑，他也幻想着能白袍仗剑，马踏江湖。不过时间过去这么久，他都没办法抢到一柄剑，原因他也差不多搞明白了。
<br/>　　在这个时代，剑与其说是兵器，倒不如说是礼器。以这个时代的锻造技术，能幻想双面开刃的铁剑有多大杀伤力么，砍不了几下人就卷刃了。所以在如今，武将上阵，大都选择长枪，大刀，铜锤……锋利不锋利不要紧，够重能砸人就可以；绿林豪侠都选择用大刀，哨棒，随地取材，造价廉价，实在是杀人放火，居家旅行必备。
<br/>　　而剑，简直就是奢侈品了，首先是造价昂贵，还得细心保养，否则生锈了拔不出剑鞘多尴尬。所以在这个时代，书生佩剑，名士佩剑，贵族佩剑……王族也佩剑。
<br/>　　李行哉究竟属于哪一种呢？
<br/>　　握着手中剑，程大雷心中缓缓思索着。
<br/>　　从青牛山去往落石城的官道上，李行哉走在前方，李婉儿跟在他身后，面露不解。
<br/>　　“哥哥，你怎么想的，要把剑留给他？”
<br/>　　“无非一柄剑而已，给了他又如何。”李行哉随意道。
<br/>　　“可那剑……”李婉儿快走几步，到他身边：“那可是匹夫剑……”
