　　轰！
<br/>　　在叶尘放下书本的那一刹那，杨国强的大脑就一片空白，瞬间短路了。
<br/>　　“噗通！”
<br/>　　经过片刻呆滞之后，杨国强双腿一软，直接便跪倒在了地上，
<br/>　　“叶、叶仙人！您、您怎么在这儿？”
<br/>　　杨国强这话一出口，其余众人也全都惊呆了。
<br/>　　“叶仙人？”
<br/>　　“眼前这个学生打扮的少年，竟然就是大名鼎鼎、震撼全球的云州叶仙人？”
<br/>　　杨家众人懵了，唐威、李虎、陈东三人也懵了，杨之凡更是眼珠子都要瞪出来了，眼眸深处充满这浓浓的惊恐之色，继而拼命的摇头，
<br/>　　“不！不可能！老爸，你一定是搞错了！他是我们生物学院的学生，怎么可能会是叶仙人，你一定是搞错了！一定……”
<br/>　　“啪！”
<br/>　　杨之凡的话语还没有说完，便直接被杨国强一巴掌，给扇回了肚子里面。
<br/>　　“你这个孽障！还不给老子闭嘴！”
<br/>　　杨国强这一巴掌，是一点也没有留情，杨之凡的脸颊顿时高高鼓起，瞬间就肿了起来。
<br/>　　打完杨之凡之后，杨国强跪着爬到叶尘跟前，开始拼命的磕头起来，
<br/>　　“叶仙人！我错了！我不知道是您，否则……”
<br/>　　叶尘冷冷一笑，直接将他话打断，
<br/>　　“不知道是我？杨国强，就算今天打伤你儿子的人不是我，又是谁给了你这么大的胆子，竟然敢打我的旗号行事？”
<br/>　　轰！
<br/>　　叶尘这话落下，身上陡然升起一股寒意。
<br/>　　杨国强顿时吓得浑身瑟瑟发抖，额头、背后更是冷汗狂冒，
<br/>　　“我、我一时口不择言，恳请叶仙人……看在杨家也是为您办事的份上，就饶了我这一次吧！”
<br/>　　杨国强说完这话，还不忘踹了一脚身后，还正捂着脸，处于懵逼状态的杨之凡，
<br/>　　“孽障！还不快向叶仙人磕头道歉！”
<br/>　　杨之凡丝毫已经完全疯了，依旧在不停的摇着头，
<br/>　　“不可能！这不可能！”
<br/>　　这也难怪，恐怕任谁也想不到，堂堂世界第一人，神仙级别的大人物，居然会隐身在天海大学，当一个普通的大学生。
<br/>　　“道歉就不必了！他的命，我收了！”
<br/>　　说完这话，叶尘直接屈指一弹，
<br/>　　嗖！
<br/>　　一道无形剑气，瞬间便击中杨之凡的眉心。
<br/>　　杨之凡甚至连惨叫都没有来得及发出，就软绵绵的倒了下去，彻底没有了气息。
<br/>　　“小凡！”
<br/>　　杨国强不由得低喝了一声，整个人呆立当场，脸色灰白无比。
<br/>　　“杨国强，念在你们杨家也曾为苏叶集团出过力，我今日便饶你一命，从今往后，杨家不再隶属于苏叶集团，带上你儿子的尸体，滚吧！”
<br/>　　叶尘瞥了一眼，犹跪在地上，如石雕般的杨国强，再次淡漠开口。
<br/>　　杨国强顿时身体一震，眼眸中透着一抹不甘，不过最终还是重重叹了一口气，低头道：
<br/>　　“是，叶仙人！”
<br/>　　面对叶尘的命令，即便杨国强再心有不甘，也只能选择服从。
<br/>　　更何况，以这位叶仙人昔日的强势作风，能够饶过他们杨家，已经算是不幸中的万幸。
<br/>　　……
<br/>　　教学楼外，之前被杨家保镖驱逐的众学生，此时正热切的议论着。
<br/>　　“你们听说了吗？这个叶尘竟然是一个武道高手，据说就在今天上午，他一个人挑了整个龙盟武道社！”
<br/>　　“什么！这不可能吧？龙盟武道社不是有一位化劲宗师坐镇吗？”
<br/>　　“那又如何？听说那位化劲宗师，被叶尘一掌拍死！连杨之凡也被他废掉了双腿！”
<br/>　　“没想到这叶尘，平时一声不吭的，竟然是一个绝世大高手！”
<br/>　　“嘿！高手又如何？东江杨家可不是能轻易招惹的，就算他具备化劲宗师的实力，恐怕今天也在劫难逃！”
<br/>　　……
<br/>　　众人正议论声，杨国强带着杨家众人，从教学楼里缓缓走了出来。
<br/>　　“快看！杨家的人出来了！”
<br/>　　“咦？那不是杨之凡吗？怎么被人抬出来了？”
<br/>　　“似乎……已经死了？”
<br/>　　……
<br/>　　众人看到两名黑衣壮汉，抬着杨之凡的尸体，顿时纷纷惊呼了起来，一些眼尖之人，似乎已经看出杨之凡死了。
<br/>　　“杨叔叔，之凡他怎么了？他这是受伤了吗？”
<br/>　　杨之凡的女朋友陶婉清，忽然走上前来，一脸急切的问道。
<br/>　　杨国强深吸了一口气，抬头望了一眼周围，生物学院以及其他学院的围观者，重重叹了一口气，朗声道：
<br/>　　“犬子刚刚突然心脏病，不幸离世了，惊扰了各位同学，杨某实在抱歉！”
<br/>　　众人听到这话，顿时相顾愕然，
<br/>　　“突发心脏病？之凡的身体一向很好，怎么会是突发心脏病？这不可能！”
<br/>　　陶婉清得知杨之凡死了，不由得大声质问起来。
<br/>　　正在说间，叶尘一行四人也从后面走了出来。
<br/>　　看到叶尘安然无恙的出现，众人顿时一片哗然。
<br/>　　虽然众人并不知道里面发生了什么，可是如今结果却显而易见，杨之凡死了，而叶尘却丝毫无恙。
<br/>　　难道作为东江霸主的杨家，竟然奈何不了此人？
<br/>　　一时间，人群中顿时躁动起来。
<br/>　　陶婉清看到叶尘，顿时美眸一瞪，指着他怒吼道：
<br/>　　“是你！一定是你害死了之凡对不对？你这个杀人凶手，我要报警，让警察把你抓起来！”
<br/>　　没等叶尘开口，反倒是一旁的杨国强脸色大变，当即一声暴喝，
<br/>　　“住口！我说了，我儿子是病死的！此事与叶仙……叶先生无关！你不可得再胡说八道！”
<br/>　　撂下这一句话，杨国强冲着手下一挥手，便急忙忙的离开了。
<br/>　　也难怪杨国强怕的不行，倘若再继续任由陶婉清闹下去，万一惹恼这位祖宗，就算将他们这些人全杀了，也不是没有这种可能。
<br/>　　看到杨国强如此紧张的模样，就算是傻子也能看出，这件事绝对没有那么简单。
<br/>　　一时间，众人望向叶尘的目光，顿时充满着浓浓的敬畏和疑惑。
<br/>　　叶尘却一脸淡然，仿佛这一切都与他无关，继续缓步朝宿舍的方向走去。
<br/>　　陶婉清反应过来之后，快步冲到叶尘跟前，双眸隐隐发红，怒吼道：
<br/>　　“叶尘！我不知道你用了什么手段，但是这件事，绝对不会就这么算了的！我一定会为之凡报仇的！”
<br/>　　听到这话，叶尘不由得眉头微微一皱，
<br/>　　很显然，这个女人和杨之凡的感情倒是不浅。
<br/>　　若是在修真界，有人胆敢如此威胁他，他只怕早就一巴掌将对方拍死了。
<br/>　　可如今在校园之中，他不想杀人，而且陶婉清又是手无缚鸡之力的弱女子，无论如何也不可能威胁到他，所以也就没有放在心上。
