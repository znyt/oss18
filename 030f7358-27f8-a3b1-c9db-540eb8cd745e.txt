　　第287章
<br/>　　战斗过后，打扫战场。
<br/>　　敌方三千人马死伤过半，逃跑时在岛上留下一具具尸体，同时也缴获了许多兵器与船只。
<br/>　　而己方损失也不小，蛤蟆寨原先的一百多号弟兄倒是毫发无损，但程大雷这次召唤来的一千余人，却只活下来一半。
<br/>　　这也是程大雷为何在最后时刻才愿意把这些人召唤出来的原因。死人的事情是免不了的，不管是轻于鸿毛，还是重于泰山，程大雷都不愿意看到。所以程大雷不想认识他们，也不想和他们有交情，只有这样才能将他们看作是一堆数据。
<br/>　　“大当家，这些弟兄都是来投奔咱们的。”徐神机。
<br/>　　“喔，给他们登记造册，有伤的治伤，先让他们歇几天，回头再训练他们。”
<br/>　　如今山寨约有六百人，人手够了，程大雷下一步练兵的计划就可以展开。不能被组织纪律约束起来的人，只能是一伙散兵游勇。
<br/>　　“大当家，这次咱们虽然打败了他们，接下来莫鸣米会不会派大军攻来？”徐神机有些担忧。
<br/>　　“是呐，不得不防，派兄弟去盯着点，一有风吹草动，随时准备战斗。”
<br/>　　这件事造成的余波，要过好久才能平息，十八家势力折了很多人手，有些势力的头领死在这次战斗中，自然也有新的人冒出来代替他们，只不过权力的更换，总会伴随血的味道。
<br/>　　他们现在没有这个力气，也不愿来找程大雷麻烦。如今，程大雷的恶名已经传开，甚至到谈程色变的地步。
<br/>　　通过这一战，程大雷也算是在琴川一地叫响了腕儿，大家都知道蛤蟆岛上有个程大雷，而且这个人不太好惹。
<br/>　　琴川关。
<br/>　　莫鸣米一身黑衣踱步在城墙上，黑色皮靴踩过墙砖，发出沉闷的响声。他每天都要在城墙上走一走，吹一吹寒风，清一清脑子里的思绪。
<br/>　　琴川关虽然只有一关，可莫鸣米实际控制的区域却能辐射方圆五百里。在这个地方，他是实实在在的土皇帝，一道条子递过去，各城城主都不能不给面子。
<br/>　　这是我的地盘，这里所有人都得听我的……每一次走过城墙，心里都会浮现这个感觉。这，或许就是权力带来的快感。
<br/>　　一言定人生，一言定人死，祸福命运皆在一人唇齿之间，这，就是权力。
<br/>　　可今天，莫鸣米的心情不太痛快，因为他控制的区域，多了一股人，就像一根钉子扎进他的眼里。
<br/>　　“不是说，他只有一百几号人么？”莫鸣米终于问出声来。
<br/>　　严笛一直跟在他身后，心情早就忐忑不安，此刻忙道：“这个情况的确没料到，当时突然杀出来一拨人，而程大雷几个真的很强。”
<br/>　　“能不强么，草原上多少人要杀他，他都能全身而退。”莫鸣米叹口气：“可这样一个强人，为何偏偏要来琴川关这个小池子。”
<br/>　　“将军，龙游浅水，虎落平阳，他现在是龙是虎，在将军面前都得低头。”
<br/>　　“嗳，拍马屁的话就不要说了，他如果不肯低头，就轮到我低头了。”
<br/>　　“那么将军这个程大雷该如何处置，总不能听之任之吧？”
<br/>　　“听之任之，只能是养虎为患，可现在摸不清他的虚实，却也不好动手。”
<br/>　　“不如，我们派个人混进去，摸摸他的情况，最好是能找个机会杀掉他……”
<br/>　　莫鸣米轻轻点头：“这件事，我们想到一起去了。”
<br/>　　……
<br/>　　蛤蟆寨，程大雷开始了治兵计划。
<br/>　　每十人一什，设一什长，十什一队，设一队长，十队……当然，蛤蟆寨总共才六百余人，还凑不够十队，所以暂且不去管它。
<br/>　　赵子龙、关鱼、张肥、秦蛮、高飞豹各领一队，其他文士，铁匠，厨子等组成的后勤队，由刘悲负责。
<br/>　　一方面，程大雷参考前世的经验，另一方面他毕竟也打过许多仗，只要他不是足够傻，多少也能摸索出来一些门道。
<br/>　　每十人为一个战斗小组，设枪兵，刀兵，盾兵，弓兵，当然，也可以每队成一个整体，参考罗马方阵……至于蛤蟆寨该朝那个方向发展，程大雷还没有想好，倒也可以交给各队队长负责，发挥他们的主观能动性。
<br/>　　赵子龙，关鱼，张肥几个，个人战斗力都是不弱的，但他们目前统兵打仗的能力几乎为零，程大雷也要慢慢培养他们这方面的能力。
<br/>　　当然，这些目前都不重要，重要的是如何洗去他们身上的顽劣之气，用组织纪律将他们约束起来。
<br/>　　程大雷与刘悲、关鱼一起定了几条军律，自然大家是一伙山贼，也不需要把山寨搞得像个军事堡垒，但也必须让这些人意识到纪律的存在。
<br/>　　惩罚措施不过是出劳力，打板子，关禁闭……程大雷发现，这三个惩罚手段几乎没用，有些人被关了禁闭，反而有几分不以为耻，反以为荣的意识。一出来就是：兄弟我如何如何……
<br/>　　程大雷被搞得分外无语，最好倒是被他想到一个法子，一人犯错，全队受罚，这些人荣誉感纪律性委实没有多少，但兄弟义气还是有的。靠连坐法，程大雷渐渐确定了军律的威信。
<br/>　　之后，程大雷又训练这些人出操，跑步，程大雷还亲自上场，陪着大家一起训练，言外之意是：大家都看到了，我也没偷懒。
<br/>　　当然，程大雷只训练了三天，就说自己有更重要的事情，不再参加训练。又是连坐法发挥了作用，让这些无组织无纪律的山贼能忍受这枯燥乏味的训练。而当十天过去后，枯燥乏味已经成了他们的习惯。
<br/>　　进攻一段时间的训练，程大雷看到这些人身上的顽劣劲渐渐消失，当然凶悍还在，就像锈铁磨成刀，变得更加锋利。
<br/>　　程大雷心里松了口气，他还正害怕计划执行不下去。在他忙碌制定训练计划的时候，李行哉和福德勒一直陪着他，充当了助理的工作。
<br/>　　“老程啊，近有件事不知当讲不当讲……”李行哉。
<br/>　　“什么？”
<br/>　　“山寨没粮了。”
<br/>
