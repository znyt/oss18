　　巨大的比武台上，一场场战斗激烈地进行着。
<br/>　　为了取得前三百的名次，所有人都实力全开，不敢大意。
<br/>　　进入裂天剑派这个武道圣地，是北山郡所有武者的梦想。
<br/>　　十个区域，有九个都进行得如火如荼。
<br/>　　唯独萧逸的区域，鬼影都没一个。
<br/>　　一开始，萧逸还满脸战意，负手而立，一副傲然模样，等待着挑战。
<br/>　　但久而久之，一直都没人鸟他，他也只能无奈地叹了口气，干脆原地坐下，观看其他区域的战斗。
<br/>　　“那家伙。”林劲撇撇嘴，看了眼萧逸，道，“打败了雷战后，谁都不敢去挑战他。”
<br/>　　秦飞扬也满脸不爽，道，“其他区域的人，一个个都严阵以待，担心被挑战。他倒好，一副百无聊赖的欠揍样子。”
<br/>　　“呵呵。”柳嫣然掩嘴一笑，“没想到萧逸这么强，想来这一轮考核，他又会是第一。”
<br/>　　铁牛瓮声道，“全场就萧逸兄弟能这么闲了，我们也该去挑战其他站桩者了。”
<br/>　　“嗯。”柳嫣然道，“其他九个区域的站桩者，似乎都很弱，我们上吧。”
<br/>　　“不急。”林劲忽然笑笑，道，“我们先等着。”
<br/>　　“等着？”众人面露疑惑。
<br/>　　“嘻嘻。”林劲贱笑两声，低声说了些什么。
<br/>　　当林劲说罢，众人的脸色变得有些古怪。
<br/>　　秦飞扬笑道，“林劲，我忽然发现，你真是个贱人。”
<br/>　　柳嫣然也偷笑道，“林劲，没想到你这么坏。”
<br/>　　铁牛瓮声笑道，“呵呵，好像很有趣的样子。”
<br/>　　时间一分一秒过去，另外九个区域的战斗，少说经历了数十场。
<br/>　　随着一个个站桩者落败、更替，如今，站在比武台上的，都非泛泛之辈。
<br/>　　像花剑、何苍等人，赫然都成了站桩者。
<br/>　　不多时，人群中，顾长风得意地自语道，“那些蝼蚁都败了，剩下的，都是些实力不错的家伙。”
<br/>　　“这才值得本公子出手，前十，非我莫属。”
<br/>　　说罢，顾长风一个跳跃，跃上了比武台，故作优雅模样的整了整衣袖。
<br/>　　而他挑战的区域，站桩者乃是王虎。
<br/>　　王虎，颇有名气的猎妖师，原是先天七重武者，寒冰池修炼七天，突破到先天八重。
<br/>　　至于顾长风，如今还是先天九重，并没有突破。
<br/>　　武者修炼，越到后面，就越难突破，需要的灵气也越来越多。
<br/>　　先天九重，要突破到半步洞玄，所需的灵气极其庞大。
<br/>　　雷战能突破，是因为他本身就天赋过人，而且迈入先天九重已经很久了，所以在寒冰池修炼七天，才一举突破到半步洞玄。
<br/>　　而顾长风，修为远不如雷战扎实，天赋也不及雷战，并没有突破。
<br/>　　言归正传。
<br/>　　比武台上，顾长风不屑地看了王虎一眼，道，“自己滚下去吧，本公子不屑对你出手。”
<br/>　　虽然他仍旧只是先天九重，没有突破到半步洞玄。
<br/>　　但在他看来，收拾王虎还是轻而易举的。
<br/>　　王虎拱了拱手，道，“长风公子，在下自认为不如你，但还是想领教几分。”
<br/>　　“不自量力。”顾长风不屑地说了一声。
<br/>　　战斗瞬间打响。
<br/>　　但结果，却毫无悬念，手持风吟剑的顾长风，一招便将王虎打成重伤，轰下比武台。
<br/>　　“你...”王虎猛地吐出几口重血，显然受了重伤。
<br/>　　“混账。”花剑一把接住王虎，冷冷地看向顾长风，“顾长风，何必下手如此重。王虎重伤，如何能面对之后的比武。”
<br/>　　“哼。”顾长风冷哼一声，“一只蝼蚁罢了，与本公子何干。”
<br/>　　“你...”花剑脸色一怒。
<br/>　　顾长风没有理会，不屑地站在比武台上。
<br/>　　按他看来，不会有人敢挑战他的，前十名次，他稳拿。
<br/>　　然而，下一秒，一道人影猛地跃上比武台。
<br/>　　“顾长风，老子来会一会你。”
<br/>　　人影冷喝一声，竟是林劲。
<br/>　　“你？哼。”顾长风不屑道，“区区先天八重，妄图挑战我，找死罢了。”
<br/>　　“林劲，小心点。”王虎高声提醒道，“顾长风心狠手辣，要是你受重伤了，后面就难以战斗了。”
<br/>　　林劲高声道，“谢你的提醒了，不过，我和顾长风可还有两笔账要算。”
<br/>　　“算账？”顾长风冷笑一声，“就凭你，也敢在本公子面前说这两字？”
<br/>　　林劲冷声道，“顾长风，接招吧。”
<br/>　　两人瞬间战了起来。
<br/>　　另一边，萧逸眉头皱了皱，林劲只是先天八重，能否打得过顾长风，还很难说。
<br/>　　同时，他从怀中拿出一丹药，手指一弹，弹到王虎身边。
<br/>　　“这是三品巅峰丹药，生灵丹。”萧逸淡淡道。
<br/>　　王虎接过，顿时一惊，“生灵丹？号称三品中的疗伤圣药，价格可不低。”
<br/>　　“萧逸兄弟，王虎暂时无以为报，日后但有所需，在下绝不推辞。”王虎面露感激之色，对远处的萧逸拱了拱手。
<br/>　　萧逸点点头，没有说话，目光看向了林劲与顾长风的战斗。
<br/>　　数分钟后，他的脸色忽然变得怪异起来。
<br/>　　因为，两人的战斗，顾长风竟完全进入下风，不，这已经不能算下风了，根本就是被完全碾压。
<br/>　　“顾长风，大半个月前，你伤我之事，还记得吧？”林劲冷声道，“今日，又算计萧逸，新仇旧恨，老子一并与你算了。”
<br/>　　林劲手中，是一把威势过人的重刀，也是一柄灵器。一刀劈下，地面开裂，空气扭曲。
<br/>　　顾长风也是手持风吟剑，奋力一档，却直接被一刀劈飞。
<br/>　　“噗。”顾长风吐出一口腥血，而后衣衫碎裂，狼狈至极。
<br/>　　“好强，你明明只有先天八重，怎么有如此战力？”顾长风满脸不可置信之色。
<br/>　　“哼，井底之蛙。”林劲不屑道，“别以为你们顾家算什么东西，你不知道，不理解的东西，多了去了。”
<br/>　　另一边，萧逸认真看着，脸色变得恍然。
<br/>　　他自然看得出来怎么回事。
<br/>　　林劲之所以完全碾压顾长风，完全是因为武魂的关系。
<br/>　　玄铁重刀，是一种可以不断通过加入玄铁，不断增加重力和威力的特殊武器。
<br/>　　而作为武魂，林劲只要不断注入真气，就等同于变相地注入玄铁。
<br/>　　如今他手持灵器，借助灵器内的力量，真气源源不断，自然注入了极大的真气在玄铁重刀上，也让玄铁重刀的重量和威力达到骇人的地步。
<br/>　　反观顾长风，他是剑武魂，但只是最普通的绿色武魂，没有任何特点，就算手持风吟剑，增幅也不大。
<br/>　　简单来说，同样是手持灵器，但灵器对于林劲实力的增幅异常巨大；而对于顾长风，增幅则非常微小。
<br/>　　言归正传，就在萧逸思索了一会儿的时间内。
<br/>　　场面上的战斗，已经进入最后阶段。
<br/>　　“顾长风，给老子滚下去吧。”林劲冷喝一声，“杀神一刀。”
<br/>　　顾长风奋力一挡，却毫无反抗之力地被击飞下比武台。
<br/>　　“噗。”顾长风站稳身子后，再次吐了几口鲜血，显然受了不轻的伤势。
<br/>　　“哼。”林劲不屑地哼了一声。
<br/>　　“活该。”花剑看着顾长风凄惨的模样，冷冷地说了一声。
<br/>　　“你们这群混蛋。”顾长风咬了咬牙，愤怒至极，脸色狰狞道，“本公子不会放过你们的。”
<br/>　　另一边，萧逸暗暗点了点头，“厉害，林劲属于那种修为越高，实力增幅越大的人。假以时日，必将是名震北山郡的天才之一。”
<br/>　　“拥有灵器，武魂不凡，又在东荒十八城那种险恶地方历练过，林劲，你的来历不简单阿。”
<br/>　　萧逸淡淡一笑。
<br/>　　另一边，战斗、挑战，从未有停歇过。
<br/>　　林劲那个区域，自从顾长风被打败，同样没人再敢去挑战。
<br/>　　不多时，顾长风靠着风吟剑赋予的庞大灵气，逐渐稳下了伤势。恨恨地看了眼林劲，兀自往其他站桩者而去。
<br/>　　嗖，一个跳跃，来到另一个站桩者面前。
<br/>　　“给本公子滚下去。”顾长风愤怒地说道，显然，他被林劲打败后，处于极度的怒火中。
<br/>　　不等那站桩者说话，顾长风已经一剑将其轰飞了下去。
<br/>　　“哼。”顾长风冷哼一声，暗道，这一次，能稳稳的得到前十了吧。
<br/>　　但，下一刻，又是一道身影跃到他的区域上。
<br/>　　而且，人影竟说着和之前的林劲一模一样的话。
<br/>　　“顾长风，大半个月前，你伤俺之事，还记得吧？今日，又算计我萧逸兄弟，新仇旧恨，俺要把你打得稀巴烂。”
<br/>　　上场说话的，正是铁牛。
<br/>　　只见他手上蓦然出现了一双手环，气势顿时激增，显然那手环也是灵器。
<br/>　　下一秒，一头高如山岳的巨兽被凝聚在他身前，正是六级妖兽，撼地神牛。
<br/>　　大半个月前，铁牛对战顾长风时，也曾凝聚过自己的武魂，不过那时只是一道数米大小的虚影。
<br/>　　而现在，却是高达百米的真正巨兽。
<br/>　　“给俺滚下去。”铁牛瓮声瓮气地喝了一声，身上的巨兽伸出了一张巨掌，狠狠一拍。
<br/>　　“噗。”顾长风再次被吐血击飞，在比武台下翻了十数圈才稳住身影。
<br/>　　“该死，你是怪物吗？”顾长风体内一阵气血翻涌，感觉五脏六腑快要被撞碎了。
<br/>　　“厉害。”远处观战的萧逸暗暗点头，“借助灵器，将武魂凝聚出来。”
<br/>　　“可惜，如今铁牛修为尚且，凝聚出来的撼地神牛实力不及真正的万分之一。”
<br/>　　“他日修为提上来，通过武魂凝聚出来的撼地神牛，将不比真正的六级妖兽撼地神牛差，轻松就可拥有地元境以上的实力。”
<br/>　　萧逸淡淡一笑，忽然明白了林劲等人的用意。
<br/>　　果然，不多时，待得顾长风借助风吟剑内的力量堪堪压下伤势，再次挑战其他站桩者，并成为其他区域的站桩者时....
<br/>　　柳嫣然跃上武台，来到他面前，说出了一模一样的话。
<br/>　　“顾长风，准备好被我打下去了吗？”柳嫣然说着，手中出现了一条红色长鞭。
<br/>(本章完)
